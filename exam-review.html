三、十大必考高频知识点

1. 有效数字规则 必考

核心规则：

  • 普通数据：从第一个非零数字起计数
  • pH、pKa 等对数值：有效数字只看小数部分！
  • pH = 11.05 → 只有2位有效数字（小数点后05）
  • pH = 3.458 → 3位有效数字
  • 反对数：$[\text{H}^+] = 10^{-11.01}$，小数部分2位 → 结果保留2位有效数字

修约规则：四舍六入五成双

  • 5后有非零数字：进位（如 2.355001 → 2.36）
  • 5后全零或无数字：看5前面的数，奇进偶不进（如 2.345 → 2.34，2.335 → 2.34）

2. pH计算公式选择（决策树） 必考

一元弱酸 HA（浓度 $c$，解离常数 $K_a$）：

  • 若 $c/K_a \geq 500$：$[\text{H}^+] = \sqrt{cK_a}$（最简式）
  • 若 $20 \leq c/K_a < 500$：$[\text{H}^+] = \frac{-K_a + \sqrt{K_a^2 + 4cK_a}}{2}$（一元二次）
  • 若 $c/K_a < 20$：需用精确式（含 $K_w$）

两性物质（如 $\text{NaH}_2\text{PO}_4$）：

  • 若 $c \geq 10K_{a1}$ 且 $cK_{a2} \geq 10K_w$：$[\text{H}^+] = \sqrt{K_{a1} \cdot K_{a2}}$
  • 注意：最简式不含浓度 $c$，但使用前必须验证条件

3. 缓冲溶液pH与缓冲能力 必考

Henderson-Hasselbalch 方程：

$$\text{pH} = \text{p}K_a + \lg\frac{c(\text{A}^-)}{c(\text{HA})}$$
  • 选择缓冲对：$\text{p}K_a$ 尽量接近目标 pH
  • 有效缓冲范围：$\text{p}K_a \pm 1$
  • 缓冲能力最强：缓冲比 = 1，即 $c(\text{HA}) = c(\text{A}^-)$
  • 例：配 pH = 9.00 的缓冲液 → 选 NH$_3$·H$_2$O/NH$_4$Cl（$\text{p}K_a = 9.26$）

4. 质子条件式（PBE）书写 必考

步骤：

  1. 确定零水准（溶液中大量存在的物种 + H$_2$O）
  2. 得质子产物放左边，失质子产物放右边
  3. 注意系数：失去/得到 $n$ 个质子，系数为 $n$
例：NaH$_2$PO$_4$ 溶液
零水准：H$_2$PO$_4^-$ 和 H$_2$O
PBE：$[\text{H}_3\text{PO}_4] + [\text{H}^+] = [\text{HPO}_4^{2-}] + 2[\text{PO}_4^{3-}] + [\text{OH}^-]$
注意 PO$_4^{3-}$ 前的系数 2（H$_2$PO$_4^-$ 失去 2 个质子变成 PO$_4^{3-}$）

5. 四大滴定可行性判据 必考

滴定类型判据含义
酸碱（直接滴定）$cK_a \geq 10^{-8}$酸性够强、浓度够大
酸碱（分步滴定）$K_{a1}/K_{a2} > 10^5$两级解离常数相差足够大
配位滴定$\lg c_{\text{sp}}K'_{MY} \geq 6$条件稳定常数足够大
氧化还原滴定$\Delta\varphi^{0'} \geq 0.4\text{V}$电位差足够大（$\approx \lg K' \geq 6$）
记忆技巧：所有判据的本质都是要求平衡常数够大（$K' \geq 10^6$），对应反应完成度 $\geq 99.9\%$。

6. 终点误差计算 必考

强酸-强碱滴定（林邦公式）：

$$E_t = \frac{[\text{H}^+]_{ep} - [\text{OH}^-]_{ep}}{c_{\text{sp}}} \times 100\%$$

弱酸-强碱滴定：

$$E_t = \frac{10^{-\Delta\text{pH}} - 10^{\Delta\text{pH}}}{\sqrt{K_t} \cdot c_{\text{sp}}} \times 100\%$$

其中 $\Delta\text{pH} = \text{pH}_{ep} - \text{pH}_{sp}$，$K_t = K_a/K_w$

正负号含义：$E_t > 0$ 为过滴定（多加了滴定剂），$E_t < 0$ 为欠滴定（少加了滴定剂）。

7. 统计检验三步法 必考

严格按顺序执行：

  1. Q 检验 → 剔除离群值（先清洗数据） $$Q = \frac{|x_{\text{可疑}} - x_{\text{邻近}}|}{x_{\max} - x_{\min}}$$ 若 $Q_{\text{计}} > Q_{\text{表}}$，剔除该值
  2. F 检验 → 比较两组数据的精密度 $$F = \frac{s_1^2}{s_2^2} \quad (s_1 > s_2)$$ 若 $F_{\text{计}} > F_{\text{表}}$，精密度有显著差异
  3. t 检验 → 比较准确度（均值与标准值/两组均值） $$t = \frac{|\bar{x} - \mu_0|}{s} \sqrt{n}$$ 若 $t_{\text{计}} > t_{\text{表}}$，有显著性差异
顺序不能反！先 Q（清洗数据）→ 再 F（比精密度）→ 最后 t（比准确度）。

8. 置信区间 高频

$$\mu = \bar{x} \pm \frac{t_{(\alpha, f)} \cdot s}{\sqrt{n}}$$
  • $f = n - 1$（自由度）
  • $t_{(\alpha, f)}$ 查表，常用：$t_{0.05, 5} = 2.571$
  • 含义：有 $(1-\alpha)\times 100\%$ 的把握，真值落在此区间内

9. 指示剂选择原则 高频

  • 核心原则：指示剂变色点应尽量接近化学计量点 pH
  • 强酸-强碱：计量点 pH = 7.0，突跃大（4.3~9.7），甲基橙/甲基红/酚酞均可，但越接近7终点误差越小
  • 弱酸-强碱：计量点偏碱（如 pH 8~9），选酚酞
  • 强酸-弱碱：计量点偏酸（如 pH 5~6），选甲基橙/甲基红
  • 配位滴定：金属指示剂应在 $\text{p}M_{\text{sp}}$ 附近变色

10. 实验方案设计思路 高频

经典题型：混合物中各组分的分别测定

设计思路：

  1. 分析各组分的性质差异（强酸/弱酸/极弱酸）
  2. 利用性质差异分步滴定或特殊处理
  3. 选择合适的指示剂标记各步终点
  4. 写出计算公式
经典例题：HCl + H$_3$BO$_3$ 混合液的测定
关键：H$_3$BO$_3$ 极弱（$\text{p}K_a = 9.24$），$cK_a < 10^{-8}$，无法直接滴定
技巧：加入甘油（丙三醇），与 H$_3$BO$_3$ 络合使酸性增强至可滴定
步骤：先甲基橙终点滴 HCl → 加甘油 → 酚酞终点滴 H$_3$BO$_3$

四、十大易错陷阱

陷阱 1：pH有效数字只看小数部分

pH = 11.05 只有 2位 有效数字，不是4位！整数部分（11）只反映数量级（$10^{-11}$），不算有效数字。

同理：pKa = 4.74 是 2 位有效数字，pOH = 2.150 是 3 位有效数字。

陷阱 2：四舍六入五成双

5 后无非零数字时，看 5 前面的数字：

  • 2.345 → 2.34（4是偶数，不进位）
  • 2.335 → 2.34（3是奇数，进位）
  • 2.3450 → 2.34（5后面是0，仍看前一位，4偶不进）
  • 2.3451 → 2.35（5后有非零数字1，直接进位）

陷阱 3：增加平行实验 ≠ 消除系统误差

增加平行实验次数只能减小随机误差（提高精密度），不能发现或消除系统误差。

消除系统误差的方法：对照实验、空白实验、仪器校正、更换方法。

陷阱 4：NaOH吸收CO$_2$后用酚酞终点滴HCl → 偏高

NaOH + CO$_2$ → Na$_2$CO$_3$。酚酞终点下 Na$_2$CO$_3$ 只反应到 NaHCO$_3$（消耗1mol HCl），而原来2mol NaOH可中和2mol HCl。

结果：同体积NaOH中和能力下降 → 需要更多NaOH才能到终点 → 计算出的HCl浓度偏高。

陷阱 5：弱酸稀释10倍，突跃下限不变，上限降1个pH

  • 突跃下限 ≈ 半当量点附近 pH（由 pKa 决定），与浓度无关
  • 突跃上限由计量点后过量碱浓度决定，稀释10倍 → 上限降约1个pH
  • 例：0.1 mol/L 时突跃 7.74~9.7 → 0.01 mol/L 时约 7.74~8.7

陷阱 6：两性物质最简式不含浓度——但要验证条件

$[\text{H}^+] = \sqrt{K_{a1} \cdot K_{a2}}$ 看似与浓度无关，但使用条件是：

  • $c \geq 10K_{a1}$（浓度远大于第一级解离常数）
  • $cK_{a2} \geq 10K_w$（第二级解离足够强）

浓度太低时条件不满足，不能使用此最简式！

陷阱 7：统计检验顺序 Q → F → t

绝对不能反！

  1. Q检验先做：剔除离群值，清洗数据
  2. F检验次做：比较两组精密度是否有显著差异
  3. t检验最后：比较准确度（均值是否一致）

原因：含离群值的数据做 t 检验会得到错误结论；F 检验不通过时 t 检验的公式不同。

陷阱 8：终点误差 $E_t$ 的正负号

  • $E_t > 0$：过滴定（终点时多加了滴定剂）
  • $E_t < 0$：欠滴定（终点时少加了滴定剂）

别搞反！用强碱滴弱酸时，若 pH$_{ep}$ > pH$_{sp}$（终点偏碱），$\Delta$pH > 0，代入公式 $E_t$ 为负值，表示过滴定方向...但注意公式中分子是 $10^{-\Delta\text{pH}} - 10^{\Delta\text{pH}}$ 恒为负。需根据滴定类型仔细判断。

陷阱 9：配位滴定中 $K'_{MY}$ 随pH变化

$\lg K'_{MY} = \lg K_{MY} - \lg\alpha_{Y(H)}$

不同 pH 下，$\alpha_{Y(H)}$ 不同 → $K'_{MY}$ 不同 → 滴定可行性也不同。

pH 越高，$\alpha_{Y(H)}$ 越小，$K'_{MY}$ 越大，滴定越有利——但要防止金属离子水解！

陷阱 10：碘量法间接法中淀粉指示剂要近终点时才加

间接碘量法中，淀粉指示剂不能一开始就加入。

  • 原因：高浓度 I$_2$ 与淀粉形成的深蓝色络合物不易分解，导致终点拖后
  • 正确做法：滴定至溶液呈浅黄色时再加入淀粉，继续滴至蓝色刚好褪去

五、四大滴定判据对比表

滴定方法判据物理意义等价说法
酸碱（直接） $cK_a \geq 10^{-8}$ $K_a$ 和浓度 $c$ 都足够大 突跃 $\geq$ 0.3 pH
酸碱（分步） $K_{a1}/K_{a2} > 10^5$ 两级解离常数差距足够大 两个突跃不重叠
配位滴定 $\lg c_{\text{sp}}K'_{MY} \geq 6$ 条件稳定常数足够大 反应完成度 $\geq$ 99.9%
配位（选择性） $\Delta\lg cK' \geq 6$ 两种金属条件常数差距够大 可选择性滴定一种
氧化还原 $\Delta\varphi^{0'} \geq 0.4\text{V}$ 条件电位差足够大 $\lg K' \geq 6$（$n=1$时）
统一本质：所有判据归根结底都要求 $K'_{\text{反应}} \geq 10^6$，即反应在化学计量点的完成度 $\geq 99.9\%$，才能产生足够大的突跃供指示剂检测。

第2题：NaOH吸收CO$_2$后酚酞终点滴HCl → 结果？ —— 答案 A

第3题：属于随机误差的是？ —— 答案 D

第4题：强酸-强碱滴定，终点误差最小的指示剂？ —— 答案 D
答案：D. 酚酞 < 甲基红 < 甲基橙

化学计量点 pH = 7.00。指示剂终点越接近7.00，误差越小：

  • 酚酞终点 pH ≈ 9.0，偏离 2.0
  • 甲基红终点 pH ≈ 5.5，偏离 1.5
  • 甲基橙终点 pH ≈ 4.0，偏离 3.0

终点误差：甲基橙 > 酚酞 > 甲基红，即误差从小到大排序为 D。

考点：终点误差与指示剂变色点的关系

第7题：弱酸浓度稀释10倍后突跃变化？ —— 答案 B
答案：B. 下限不变，上限降约1个pH

突跃下限由 pKa 决定（Henderson-Hasselbalch 方程，半当量点附近），与浓度无关。突跃上限由计量点后过量碱浓度决定，浓度降低10倍 → 上限降约1个pH单位。

例：0.1 mol/L 时突跃 7.74~9.7 → 0.01 mol/L 时约 7.74~8.7

考点：浓度对滴定突跃的影响

第8题：H$_2$PO$_4^-$ 分布系数最大时的pH？ —— 答案 B
答案：B. pH = 4.66

多元酸中间型体分布系数最大时：

$$\text{pH} = \frac{\text{p}K_{a1} + \text{p}K_{a2}}{2} = \frac{2.12 + 7.20}{2} = 4.66$$

考点：分布系数与 pH 的关系

第9题：缓冲能力最强的NH$_3$/NH$_4$Cl混合比？ —— 答案 D
答案：D. V(NH$_3$) : V(NH$_4$Cl) = 1 : 10

缓冲能力最强 → 缓冲比 = 1 → $n(\text{NH}_3) = n(\text{NH}_4^+)$

$$\frac{1.0 \times V_1}{0.1 \times V_2} = 1 \implies V_1 : V_2 = 0.1 : 1.0 = 1 : 10$$

考点：缓冲能力与缓冲比的关系

第10题：不能减免系统误差的是？ —— 答案 D

计算第3题（10分）：弱酸 HA 的滴定

已知：0.1000 mol/L NaOH 滴定 20.00 mL 0.1000 mol/L HA，加入 18.00 mL NaOH 时 pH = 5.82

(1) 滴定反应方程式（2分）
$$\text{HA} + \text{NaOH} \to \text{NaA} + \text{H}_2\text{O}$$
(2) 计算 pKa（2分）

$n(\text{HA}) = 0.1000 \times 20.00 = 2.000$ mmol

$n(\text{NaOH}) = 0.1000 \times 18.00 = 1.800$ mmol

剩余 HA = 2.000 - 1.800 = 0.200 mmol，生成 A$^-$ = 1.800 mmol

$$\text{pH} = \text{p}K_a + \lg\frac{c(\text{A}^-)}{c(\text{HA})} \implies 5.82 = \text{p}K_a + \lg\frac{1.800}{0.200} = \text{p}K_a + 0.954$$ $$\boxed{\text{p}K_a = 5.82 - 0.954 = 4.87}$$
(3) 化学计量点pH及指示剂选择（2分）

计量点时 HA 完全中和为 NaA：

$$c(\text{NaA}) = \frac{2.000}{20.00 + 20.00} = 0.0500 \text{ mol/L}$$

$K_b = K_w / K_a = 10^{-14} / 10^{-4.87} = 10^{-9.13}$

验证近似条件：$c/K_b \gg 100$ ✓，$cK_b \geq 10K_w$ ✓

$$[\text{OH}^-] = \sqrt{c \cdot K_b} = \sqrt{0.0500 \times 10^{-9.13}} = \sqrt{3.71 \times 10^{-11}} = 6.09 \times 10^{-6}$$ $$\text{pOH} = 5.22 \implies \boxed{\text{pH}_{sp} = 14 - 5.22 = 8.78}$$

计量点在碱性区（pH ≈ 8.78） → 选用酚酞（变色范围 8.0~9.8）

(4) 滴定曲线要点（2分）
  • 起始 pH ≈ $\frac{1}{2}(4.87 - \lg 0.1) = \frac{1}{2}(4.87 + 1) \approx 2.94$
  • 半当量点（V = 10 mL）：pH = pKa = 4.87
  • 化学计量点（V = 20 mL）：pH = 8.78
  • 突跃范围约 7.7~10
  • 曲线形状：起始平缓 → 半当量点附近有拐点 → 突跃 → 过量碱区平缓上升
(5) 终点误差计算（2分）

酚酞终点 pH$_{ep}$ = 9.1，计量点 pH$_{sp}$ = 8.78

$\Delta\text{pH} = 9.1 - 8.78 = 0.32$

$K_t = K_a / K_w = 10^{-4.87} / 10^{-14} = 10^{9.13}$

$$E_t = \frac{10^{-\Delta\text{pH}} - 10^{\Delta\text{pH}}}{\sqrt{K_t} \cdot c_{sp}} \times 100\%$$ $$= \frac{10^{-0.32} - 10^{0.32}}{\sqrt{10^{9.13}} \times 0.0500} \times 100\% = \frac{0.479 - 2.089}{10^{4.565} \times 0.0500} \times 100\%$$ $$= \frac{-1.610}{36760 \times 0.0500} \times 100\% = \frac{-1.610}{1838} \times 100\% = \boxed{-0.09\%}$$

实验设计题（8分）：HCl 和 H$_3$BO$_3$ 混合溶液的测定

原理分析
  • HCl 是强酸，可直接用 NaOH 滴定
  • H$_3$BO$_3$ 极弱酸（$\text{p}K_a = 9.24$），$cK_a = 0.1 \times 10^{-9.24} = 10^{-10.24} < 10^{-8}$，不能直接滴定
  • 解决方法：加入丙三醇（甘油），H$_3$BO$_3$ 与甘油络合形成强酸性硼酸甘油酯，表观 $K_a$ 增大至 $\sim 10^{-5}$，满足滴定条件
实验流程

第一步：滴定 HCl

  • 取混合液适量 → 加甲基橙指示剂
  • 用 NaOH 标准液（$c$ mol/L）滴定至溶液由红色→橙黄色（pH ≈ 4.0）
  • 记录消耗体积 $V_1$ mL
  • 反应：HCl + NaOH → NaCl + H$_2$O
  • 此 pH 下 H$_3$BO$_3$ 太弱，不参与反应

第二步：加入甘油

  • 向同一锥形瓶中加入过量丙三醇，充分混匀
  • H$_3$BO$_3$ + 甘油 → 硼酸甘油酯络合酸（酸性增强）

第三步：滴定 H$_3$BO$_3$

  • 加酚酞指示剂
  • 继续用 NaOH 标准液滴定至溶液由无色→粉红色（pH ≈ 9）
  • 记录消耗体积 $V_2$ mL

公式书写规范

  • Henderson-Hasselbalch 方程：写为 $\text{pH} = \text{p}K_a + \lg\frac{c_b}{c_a}$，不要用 $\log$ 或 $\ln$
  • 标准偏差公式：分母是 $n-1$ 不是 $n$（样本标准偏差用 Bessel 校正）
  • t 检验公式：$t = \frac{|\bar{x} - \mu|}{s}\sqrt{n}$，注意 $\sqrt{n}$ 在分子
  • 终点误差公式：分清强酸-强碱和弱酸-强碱用不同公式

计算过程展示规范

  1. 列出已知量（含单位）
  2. 写出公式（原始形式）
  3. 代入数据（保留足够多的中间位数）
  4. 得出结果（正确有效数字 + 单位）
  5. 必要时验证（如检验近似条件是否满足）
得分技巧：即使最终结果算错，写出正确的公式和合理的代入过程仍可得步骤分。因此一定要写过程，不要只写答案。

自测题 1：有效数字

pH = 3.46，求 $[\text{H}^+]$，保留正确的有效数字位数。

查看答案

pH 小数部分 ".46" 为 2 位，$[\text{H}^+]$ 保留 2 位有效数字。

$$[\text{H}^+] = 10^{-3.46} = 3.5 \times 10^{-4}\;\text{mol/L}$$

自测题 2：缓冲溶液 pH

用 0.20 mol/L HAc（$pK_a = 4.74$）与 0.10 mol/L NaAc 等体积混合，求混合溶液 pH。

查看答案

等体积混合后：$c(\text{HAc}) = 0.10$，$c(\text{NaAc}) = 0.05$

$$\text{pH} = pK_a + \lg\frac{c_b}{c_a} = 4.74 + \lg\frac{0.05}{0.10} = 4.74 - 0.30 = 4.44$$

自测题 3：能否直接滴定？

浓度为 0.1 mol/L 的以下酸，哪些能用 NaOH 直接滴定？
A. HF（$K_a = 6.8\times10^{-4}$）  B. HCN（$K_a = 6.2\times10^{-10}$）  C. $\text{H}_2\text{S}$（$K_{a1}=1.3\times10^{-7}$）

查看答案

判据：$cK_a \geq 10^{-8}$

  • A: $0.1 \times 6.8\times10^{-4} = 6.8\times10^{-5} \gg 10^{-8}$ ✓ 能滴定
  • B: $0.1 \times 6.2\times10^{-10} = 6.2\times10^{-11} < 10^{-8}$ ✗ 不能
  • C: $0.1 \times 1.3\times10^{-7} = 1.3\times10^{-8} \geq 10^{-8}$ ✓ 勉强能滴定第一步

答：A 和 C（第一步）能直接滴定。

自测题 4：t 检验

5 次测量某钢样的碳含量得 $\bar{x} = 0.326\%$，$s = 0.008\%$，标准值 $\mu_0 = 0.310\%$。问在 95% 置信度下是否存在系统误差？（$t_{0.05,4} = 2.776$）

查看答案
$$t_{\text{calc}} = \frac{|\bar{x} - \mu_0|}{s}\sqrt{n} = \frac{|0.326 - 0.310|}{0.008}\sqrt{5} = \frac{0.016}{0.008} \times 2.236 = 4.47$$

$t_{\text{calc}} = 4.47 > t_{0.05,4} = 2.776$，存在显著性差异，有系统误差。

自测题 5：EDTA 条件稳定常数

在 pH = 5.0 时用 EDTA 滴定 $\text{Zn}^{2+}$。已知 $\lg K_{\text{ZnY}} = 16.50$，pH = 5 时 $\lg\alpha_{Y(H)} = 6.6$。求 $\lg K'_{\text{ZnY}}$，能否准确滴定？

查看答案
$$\lg K'_{\text{ZnY}} = \lg K_{\text{ZnY}} - \lg\alpha_{Y(H)} = 16.50 - 6.6 = 9.9$$

判据：$\lg cK' \geq 6$，$\lg(0.01 \times 10^{9.9}) = \lg(10^{7.9}) = 7.9 \geq 6$ ✓

自测题 6：氧化还原计量点电位

用 $\text{Ce}^{4+}$（$\varphi^{0'} = 1.44$ V，$n_1 = 1$）滴定 $\text{Fe}^{2+}$（$\varphi^{0'} = 0.68$ V，$n_2 = 1$），求化学计量点电位。

查看答案
$$\varphi_{sp} = \frac{n_1\varphi_1^{0'} + n_2\varphi_2^{0'}}{n_1 + n_2} = \frac{1 \times 1.44 + 1 \times 0.68}{2} = 1.06\;\text{V}$$

自测题 7：Beer 定律计算

某溶液浓度 $c = 2.0\times10^{-5}$ mol/L，在 1 cm 比色皿中测得透射比 $T = 25\%$。求摩尔吸光系数 $\kappa$。

查看答案
$$A = -\lg T = -\lg 0.25 = 0.602$$ $$\kappa = \frac{A}{bc} = \frac{0.602}{1 \times 2.0\times10^{-5}} = 3.0 \times 10^{4}\;\text{L/(mol·cm)}$$

自测题 8：沉淀溶解度

$\text{AgCl}$ 的 $K_{sp} = 1.8\times10^{-10}$。在 0.10 mol/L NaCl 溶液中，$\text{AgCl}$ 的溶解度是多少？

查看答案

$\text{AgCl} \rightleftharpoons \text{Ag}^+ + \text{Cl}^-$，$[\text{Cl}^-] \approx 0.10$ mol/L（同离子效应）

$$[\text{Ag}^+] = \frac{K_{sp}}{[\text{Cl}^-]} = \frac{1.8\times10^{-10}}{0.10} = 1.8\times10^{-9}\;\text{mol/L}$$

$s = [\text{Ag}^+] = 1.8\times10^{-9}$ mol/L（比纯水中 $s = 1.3\times10^{-5}$ 小了近 4 个数量级）

自测题 9：终点误差（弱酸-强碱）

0.1 mol/L HAc（$K_a = 1.8\times10^{-5}$）被 NaOH 滴定，酚酞终点 pH = 9.0，化学计量点 pH = 8.72。求终点误差 $E_t$。（$K_t = K_a/K_w = 1.8\times10^9$，$c_{sp} = 0.05$ mol/L）

查看答案

$\Delta\text{pH} = 9.0 - 8.72 = 0.28$

$$E_t = \frac{10^{-\Delta\text{pH}} - 10^{\Delta\text{pH}}}{\sqrt{K_t} \cdot c_{sp}} \times 100\%$$ $$= \frac{10^{-0.28} - 10^{0.28}}{\sqrt{1.8\times10^9} \times 0.05} \times 100\% = \frac{0.525 - 1.905}{42426 \times 0.05} \times 100\%$$ $$= \frac{-1.38}{2121} \times 100\% = -0.065\%$$

$|E_t| < 0.1\%$，可以接受。
