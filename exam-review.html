基准物质与标准溶液 高频

PPT：基准物质的条件

基准物质必须满足的条件：

  1. 纯度足够高（99.9% 以上）
  2. 组成与化学式完全符合（不含结晶水变化）
  3. 性质稳定（不吸湿、不与CO$_2$/O$_2$反应）
  4. 参加反应时按反应式定量进行，无副反应

常见基准物质：
Na$_2$CO$_3$（标定HCl）、邻苯二甲酸氢钾（标定NaOH）、Na$_2$C$_2$O$_4$（标定KMnO$_4$）、K$_2$Cr$_2$O$_7$（标定Na$_2$S$_2$O$_3$）、纯Zn/CaCO$_3$（标定EDTA）

滴定分析基本计算 必考

PPT：滴定分析计算

PPT：滴定计算示例

核心公式：

$$n(\text{滴定剂}) \times \text{化学计量比} = n(\text{被测物})$$ $$c_1 V_1 = c_2 V_2 \times \frac{n_1}{n_2}$$

质量分数：

$$w = \frac{c \cdot V \cdot M_{\text{被测物}} \times \text{化学计量比}}{m_{\text{样品}}} \times 100\%$$

滴定度 $T$：

$$T_{\text{A/B}} = \frac{m_A}{V_B} \quad (\text{g/mL})$$
注意：V 的单位必须统一为 L 或 mL，摩尔质量用 g/mol。计算中保留足够中间位数，最终结果按有效数字规则修约。

$n$(Na₂CO₃)$=0.2120/106.0=2.000\times10^{-3}$ mol
$n$(HCl)$=2\times2.000\times10^{-3}=4.000\times10^{-3}$ mol
$c$(HCl)$=4.000\times10^{-3}/0.02000=\boxed{0.2000}$ mol/L

四、第三章 误差与分析数据处理

误差的分类与特征 必考

PPT：误差分类

特征系统误差随机误差
方向单向（恒定或有规律变化）双向（时大时小，时正时负）
规律重复测量时重复出现服从正态分布
消除对照实验、空白实验、仪器校正增加平行实验次数（取平均）
影响影响准确度影响精密度

PPT：系统误差的来源

系统误差三大来源：
① 方法误差：终点与计量点不重合、反应不完全、副反应干扰
② 仪器误差：砝码磨损、滴定管不准、容量瓶未校正
③ 操作误差：对指示剂颜色判断偏差（注意：这是个人的恒定偏差，属系统误差）
消除方法：对照实验（用标准试样检验）、空白实验（扣除试剂本底）、仪器校正、更换方法

PPT：随机误差的统计特征

随机误差四大特征（正态分布）：

  • 对称性：正负误差出现概率相等
  • 单峰性：小误差出现概率大于大误差
  • 有界性：大误差出现概率趋近于零
  • 抵偿性：无限多次测量，随机误差的平均值趋于零
易混点：「读数时最后一位估测不准」→ 随机误差（不是系统误差！每次估读偏高偏低是随机的）
过失误差（粗大误差）：操作失误（如加错试剂、读数看错）→ 不遵循正态分布 → 应用 Q/Grubbs 检验剔除

准确度与精密度 必考

准确度 (Accuracy)精密度 (Precision)
衡量测定值与真值的接近程度多次测定值之间的一致程度
表示误差（绝对/相对）偏差（标准偏差 $s$、RSD）
影响因素系统误差随机误差

四种组合：

  • 精密度好 + 准确度好 → 理想结果 ✓
  • 精密度好 + 准确度差 → 存在系统误差
  • 精密度差 + 准确度好 → 偶然巧合，不可靠
  • 精密度差 + 准确度差 → 最差情况
关系：精密度是准确度的前提条件，但精密度好不保证准确度好（可能有系统误差）。

有效数字规则 必考

基本规则：

  • 从第一个非零数字起计数：0.0025 → 2位；25.00 → 4位
  • 末尾的零有效：1.00 → 3位；100.0 → 4位
  • "1000"有歧义 → 用科学计数法：$1.0 \times 10^3$（2位）
  • pH、pKa 等对数值：有效数字只看小数部分！

运算规则：

  • 加减法：结果小数位数与参与运算数中小数位数最少的相同
  • 乘除法：结果有效数字位数与参与运算数中有效数字最少的相同

修约规则（四舍六入五成双）：

  • ≤4 舍去，≥6 进位
  • =5 时看5后面：5后有非零数字 → 进位；5后全零或无 → 看5前面，奇进偶不进
典型例题：
• pH = 11.05 → 2位有效数字（小数部分 .05）
• pKa = 4.74 → 2位有效数字
• $[\text{H}^+] = 10^{-11.01} = 9.8 \times 10^{-12}$ mol/L（保留2位）
• 2.345 → 2.34（4偶不进）；2.335 → 2.34（3奇进位）；2.3451 → 2.35（5后有1，进位）

PPT：t 分布（小样本统计）

t 分布 vs 正态分布：

  • 用样本标准偏差 $s$ 代替总体 $\sigma$ → 引入不确定性 → t 分布
  • 自由度 $f$ 越小 → 曲线越矮胖（尾部更厚）→ 同置信度需更大 $t$ 值
  • $f \to \infty$ 时 $t$ 分布趋近正态分布

置信区间 必考

PPT：置信区间

$$\mu = \bar{x} \pm \frac{t_{(\alpha, f)} \cdot s}{\sqrt{n}}$$
  • $f = n - 1$（自由度）
  • $\alpha$：显著性水平，常用 0.05
  • 含义：有 $(1-\alpha)\times 100\%$ 的把握，真值落在此区间内
常用 t 值速查：
$t_{0.05, 3} = 3.182$，$t_{0.05, 4} = 2.776$，$t_{0.05, 5} = 2.571$，$t_{0.05, 9} = 2.262$，$t_{0.05, 19} = 2.093$

统计检验三步法：Q → F → t 必考

PPT：Q 检验

PPT：Q 检验临界值表

PPT：Grubbs 检验

PPT：F 检验

PPT：F 检验例题

PPT：t 检验

PPT：配对 t 检验

Step 1 — Q 检验（剔除离群值）：

$$Q = \frac{|x_{\text{可疑}} - x_{\text{邻近}}|}{x_{\max} - x_{\min}}$$

若 $Q_{\text{计}} > Q_{\text{表}}$，剔除；否则保留。

Step 2 — F 检验（比较两组精密度）：

$$F = \frac{s_1^2}{s_2^2} \quad (s_1 > s_2)$$

若 $F_{\text{计}} > F_{\text{表}}$，两组精密度有显著差异 → 不能合并数据。

Step 3 — t 检验（比较准确度）：

① 样本均值 vs 标准值：

$$t = \frac{|\bar{x} - \mu_0|}{s} \sqrt{n}$$

② 两组样本均值比较（F 检验通过后用合并标准偏差）：

$$s_{\text{合}} = \sqrt{\frac{(n_1-1)s_1^2 + (n_2-1)s_2^2}{n_1+n_2-2}}, \quad t = \frac{|\bar{x}_1 - \bar{x}_2|}{s_{\text{合}}} \sqrt{\frac{n_1 n_2}{n_1+n_2}}$$
顺序不能反！先 Q（清洗）→ 再 F（比精密度）→ 最后 t（比准确度）。含离群值做 t 检验会得错误结论。
完整统计检验例题（PPT典型流程）：
甲法6次测定：$\bar{x}_1=54.26\%$，$s_1=0.041\%$；乙法5次测定：$\bar{x}_2=54.10\%$，$s_2=0.032\%$
Step 1(Q)：先对各组数据分别做 Q 检验（略），假设无离群值
Step 2(F)：$F=s_1^2/s_2^2=0.041^2/0.032^2=1.64$，$F_{0.05}(5,4)=6.26$，$1.64<6.26$ → 精密度无显著差异 ✓
Step 3(t)：合并 $s_合=\sqrt{\frac{5\times0.041^2+4\times0.032^2}{6+5-2}}=0.037\%$
$t=\frac{|54.26-54.10|}{0.037}\sqrt{\frac{6\times5}{6+5}}=\frac{0.16}{0.037}\times1.651=7.14$
$t_{0.05,9}=2.262$，$7.14>2.262$ → 两法均值有显著差异
Grubbs 检验（另一种离群值检验法）：
$G = \frac{|x_{\text{可疑}} - \bar{x}|}{s}$，若 $G > G_{\text{表}}$ 则剔除。与 Q 检验目的相同，考试中按题目要求选用。
另有 4d 法（$|x_{\text{可疑}} - \bar{x}| > 4\bar{d}$ 则舍弃），适用于快速判断。

误差传递 高频

PPT：误差传递公式

系统误差传递：

  • 加减法：$\Delta R = \pm\Delta a \pm \Delta b$（绝对误差直接加减）
  • 乘除法：$\frac{\Delta R}{R} = \pm\frac{\Delta a}{a} \pm \frac{\Delta b}{b}$（相对误差传递）

随机误差传递：

  • 加减法：$s_R = \sqrt{s_a^2 + s_b^2}$
  • 乘除法：$\frac{s_R}{R} = \sqrt{\left(\frac{s_a}{a}\right)^2 + \left(\frac{s_b}{b}\right)^2}$
应用：判断哪一步操作对最终误差贡献最大，从而优化实验方案。通常相对误差最大的那一步是"瓶颈"。

一元线性回归 高频

PPT：线性回归

$$y = a + bx$$ $$b = \frac{n\sum x_i y_i - \sum x_i \sum y_i}{n\sum x_i^2 - (\sum x_i)^2}, \quad a = \bar{y} - b\bar{x}$$

相关系数：

$$r = \frac{n\sum x_i y_i - \sum x_i \sum y_i}{\sqrt{[n\sum x_i^2 - (\sum x_i)^2][n\sum y_i^2 - (\sum y_i)^2]}}$$

$|r|$ 越接近 1，线性关系越好。通常要求 $|r| > 0.999$。

Part 1：活度与活度系数 高频

PPT：酸碱反应及其平衡常数

活度（activity）：离子在化学反应中起作用的有效浓度

$$a = \gamma \cdot c$$
  • $\gamma$：活度系数（activity coefficient），衡量实际与理想的偏差
  • 溶液无限稀时 $\gamma \to 1$，$a \to c$；中性分子和溶剂 $\gamma = 1$

Debye-Hückel 公式（计算活度系数）：

$$-\lg\gamma_i = \frac{0.5115 z_i^2 \sqrt{I}}{1 + Ba_i\sqrt{I}}$$

PPT：常见离子活度系数表（25°C）

  • $I$：离子强度，$I = \frac{1}{2}\sum c_i z_i^2$
  • $z_i$：离子电荷数；$a_i$：离子有效半径(nm)
  • 规律：$I$ 一定时，电荷数越高 → $\gamma$ 越小 → 活度偏离浓度越大
  • $I < 0.01$：用 Debye-Hückel 极限公式 $-\lg\gamma_i = 0.512z_i^2\sqrt{I}$
  • $I > 0.1$：直接用活度代替浓度计算
PPT例题：MgCl$_2$ 0.030 mol/L + HCl 0.010 mol/L 溶液中 H$^+$ 的活度
$I = \frac{1}{2}(0.030\times2^2 + 0.060\times1^2 + 0.010\times1^2 + 0.010\times1^2) = 0.10$ mol/kg
$-\lg\gamma_{H^+} = \frac{0.5115\times1^2\times\sqrt{0.010}}{1+3.291\times0.9\times\sqrt{0.010}} = 0.084$
$\gamma_{H^+} = 0.82$，$a_{H^+} = 0.82\times0.010 = 0.0082$ mol/L

活度常数、浓度常数与混合常数 高频

PPT：三种平衡常数对比

以 HAc ⇌ H$^+$ + Ac$^-$ 为例：

类型表达式特点
活度常数 $K_a$$K_a = \frac{a_{H^+} \cdot a_{Ac^-}}{a_{HAc}}$热力学常数，仅与温度 $T$ 有关
浓度常数 $K_a^c$$K_a^c = \frac{[H^+][Ac^-]}{[HAc]}$与 $T$ 和离子强度 $I$ 有关
混合常数 $K_a^M$$K_a^M = \frac{a_{H^+}[Ac^-]}{[HAc]}$与 $T$、$I$ 有关，pH计常用

大小关系：$K_a < K_a^M < K_a^c$，即 $\text{p}K_a > \text{p}K_a^M > \text{p}K_a^c$

实际处理：一般分析（$I = 0.1\sim0.5$ mol/kg）用浓度（混合）常数代替活度常数。对精确度要求较高时（如标准缓冲液pH计算），需考虑 $I$ 的影响。

Part 2：质子理论 必考

PPT：酸碱半反应

Brønsted-Lowry 质子理论核心：

  • 酸：能给出质子的物质（质子供体）→ 酸 ⇌ 共轭碱 + H$^+$
  • 碱：能接受质子的物质（质子受体）→ 碱 + H$^+$ ⇌ 共轭酸
  • 酸碱半反应：不能单独存在，必须成对出现（共轭酸碱对）
PPT示例：
HAc ⇌ Ac$^-$ + H$^+$（HAc/Ac$^-$ 共轭对）
NH$_4^+$ ⇌ NH$_3$ + H$^+$（NH$_4^+$/NH$_3$ 共轭对）
HA ⇌ A$^-$ + H$^+$（一般表示）
共轭酸 ⇌ 碱 + 质子

PPT：共轭酸碱对 Ka·Kb = Kw

共轭酸碱对电离常数关系：

$$K_a \cdot K_b = K_w = 10^{-14} \quad (25°\text{C})$$ $$\text{p}K_a + \text{p}K_b = \text{p}K_w = 14.00$$
  • 酸越强（$K_a$ 越大），其共轭碱越弱（$K_b$ 越小）
  • 两性物质：既可给出质子又可接受质子（H$_2$O、HCO$_3^-$、H$_2$PO$_4^-$、NH$_4$Ac 中的离子、氨基酸）

Part 3：分布分数 $\delta$ 必考

定义：溶液中某物种的平衡浓度占总浓度（分析浓度 $c$）的分数

一元弱酸 HA（两种形态 HA 和 A$^-$）：

$$\delta_{\text{HA}} = \frac{[\text{H}^+]}{[\text{H}^+] + K_a}, \quad \delta_{\text{A}^-} = \frac{K_a}{[\text{H}^+] + K_a}$$
  • pH = pK$_a$ 时，$\delta_{\text{HA}} = \delta_{\text{A}^-} = 0.5$（两种形态各占一半）
  • pH < pK$_a$ → HA 为优势形态；pH > pK$_a$ → A$^-$ 为优势形态

PPT：一元弱酸 HF 的 δ-pH 图和优势区域图（pKa=3.17）

二元弱酸 H$_2$A（三种形态）：

$$\delta_0 = \frac{[\text{H}^+]^2}{D}, \quad \delta_1 = \frac{[\text{H}^+]K_{a1}}{D}, \quad \delta_2 = \frac{K_{a1}K_{a2}}{D}$$

其中 $D = [\text{H}^+]^2 + [\text{H}^+]K_{a1} + K_{a1}K_{a2}$

PPT：二元酸 H₂CO₃ 的 δ-pH 图（pKa1=6.38, pKa2=10.25, ΔpK=3.87）

重要规律：
① 中间型体 $\delta_1$ 在 pH = $\frac{1}{2}(\text{p}K_{a1} + \text{p}K_{a2})$ 时最大
② $\Delta\text{p}K = \text{p}K_{a2} - \text{p}K_{a1}$ 越大，中间型体分布曲线越宽（可独立存在的pH范围越大）
③ 两曲线交点处 pH 即为对应的 pK$_a$ 值

PPT：思考题——H₂A 的 pKa1=4.00, pKa2=8.00，当[H₂A]=[A²⁻]时 pH=?

解：当 $[\text{H}_2\text{A}] = [\text{A}^{2-}]$ 时 → $\delta_0 = \delta_2$ → $[\text{H}^+]^2 = K_{a1}K_{a2}$ → $\text{pH} = \frac{1}{2}(\text{p}K_{a1}+\text{p}K_{a2}) = 6.00$
思考延伸：配制 pH=4.35 的 H₃PO₄ 溶液和 pH=4.35 的 Na₃PO₄ 溶液，哪个中 [HPO₄²⁻] 更大？→ 答案：虽然 pH 相同，但 Na₃PO₄ 溶液总浓度更高（需先中和），所以 [HPO₄²⁻] 更大

5.1 强酸/强碱 pH

PPT：强酸 pH 精确式推导（从PBE出发）

强酸 HCl（$c_a$ mol/L）精确式：

PBE：$[\text{H}^+] = [\text{OH}^-] + c_a = \frac{K_w}{[\text{H}^+]} + c_a$

$$[\text{H}^+] = \frac{c_a + \sqrt{c_a^2 + 4K_w}}{2}$$

强碱 NaOH（$c_b$ mol/L）精确式：

$$[\text{OH}^-] = \frac{c_b + \sqrt{c_b^2 + 4K_w}}{2}$$

最简式条件：$c_a > 20[\text{OH}^-]$ 或 $c_a > 10^{-6}$ mol/L 时

$$[\text{H}^+] \approx c_a \quad \text{（强酸）}, \quad [\text{OH}^-] \approx c_b \quad \text{（强碱）}$$

PPT：极稀强酸 pH 计算（$2.0\times10^{-8}$ mol/L HCl）

PPT经典例题：$2.0\times10^{-8}$ mol/L HCl 的 pH
错误做法：$[\text{H}^+]=2.0\times10^{-8}$，pH=7.70 → pH>7? 酸溶液怎么会碱性？ ✗
正确做法（精确式）：
$[\text{H}^+] = \frac{2.0\times10^{-8}+\sqrt{(2.0\times10^{-8})^2+4\times10^{-14}}}{2} = 10^{-6.96}$
pH = 6.96（略小于7，合理！）✓

PPT：强碱 pH 计算示例

PPT例：$2.0\times10^{-5}$ mol/L NaOH → $c > 10^{-6}$ → $[\text{OH}^-] = 2.0\times10^{-5}$，pOH=4.70，pH=9.30
关键判断：$c > 10^{-6}$ mol/L 用最简式；$c < 10^{-6}$ mol/L 必须用精确式。极稀强酸的 pH 无限趋近但永远不会超过 7。

5.2 一元弱酸（碱）pH 计算——完整推导链 必考

PPT：弱酸精确式（从PBE推导）

Step 1 — 精确式（三次方程）：

PBE：$[\text{H}^+] = [\text{A}^-] + [\text{OH}^-]$

代入 $K_a$ 和 $K_w$：

$$[\text{H}^+] = \frac{K_a[\text{HA}]}{[\text{H}^+]} + \frac{K_w}{[\text{H}^+]}$$ $$[\text{H}^+]^2 = K_a[\text{HA}] + K_w$$ $$[\text{H}^+] = \sqrt{K_a[\text{HA}] + K_w} = \sqrt{K_a(c - [\text{H}^+]\cdot\frac{c}{[\text{H}^+]+K_a}) + K_w}$$

整理得精确三次方程：$[\text{H}^+]^3 + K_a[\text{H}^+]^2 - (K_ac + K_w)[\text{H}^+] - K_aK_w = 0$

PPT：近似处理——忽略 Kw 项

Step 2 — 近似式（忽略水的解离）：

条件：$K_a[\text{HA}] \approx K_ac \geq 20K_w$（即 $cK_a \geq 20K_w = 2\times10^{-13}$）

$$[\text{H}^+] = \sqrt{K_a[\text{HA}]} = \sqrt{K_a(c - [\text{H}^+])}$$ $$[\text{H}^+] = \frac{-K_a + \sqrt{K_a^2 + 4K_ac}}{2}$$

PPT：最简式——条件 c/Ka ≥ 400

Step 3 — 最简式（进一步忽略解离消耗）：

条件：$K_a/c < 2.5\times10^{-3}$，即 $c/K_a \geq 400$（解离度 $\alpha < 5\%$，$c - [\text{H}^+] \approx c$）

$$\boxed{[\text{H}^+] = \sqrt{cK_a}}$$

PPT：考虑水解离的近似式

特殊情况 — 极弱酸/极稀（$cK_a < 20K_w$）：

水的解离不可忽略，但 $c/K_a \geq 400$ 仍成立时：

$$[\text{H}^+] = \sqrt{K_ac + K_w}$$

5.4 两性物质 pH 计算——精确→近似→最简全链 必考

PPT：两性物质（酸式盐 NaHA）精确式推导

以二元酸的酸式盐 NaHA 为例（零水准 HA⁻, H₂O）：

PBE：$[\text{H}^+] + [\text{H}_2\text{A}] = [\text{OH}^-] + [\text{A}^{2-}]$

代入各平衡：

$$[\text{H}^+] = \sqrt{\frac{K_{a1}(K_{a2}[\text{HA}^-] + K_w)}{[\text{HA}^-] + K_{a1}}} \quad \text{（精确式）}$$

PPT：两性物质近似式

近似式（$[\text{HA}^-] \approx c$，酸式/碱式解离都很小）：

$$[\text{H}^+] = \sqrt{\frac{K_{a1}(K_{a2}c + K_w)}{c + K_{a1}}}$$

进一步简化：

  • 若 $K_{a2}c \geq 20K_w$（可忽略 $K_w$）：$[\text{H}^+] = \sqrt{\frac{K_{a1}K_{a2}c}{c + K_{a1}}}$
  • 若同时 $c \geq 20K_{a1}$（可忽略分母中 $K_{a1}$）：

PPT：两性物质最简式及其严格使用条件

$$\boxed{[\text{H}^+] = \sqrt{K_{a1} \cdot K_{a2}}}$$
最简式使用条件（PPT原文强调）：
① $K_{a2}c \geq 20K_w$（即 $cK_{a2} \geq 2\times10^{-13}$）
② $c \geq 20K_{a1}$
只有在酸式盐浓度不是很小时，即 $c \geq 20K_{a1}$ 且水解所产生的 H⁺ 可忽略的情况下才可使用！
结果不含浓度 $c$，但验证条件必须用到 $c$！

PPT：混合弱酸 pH 近似公式

$$[\text{H}^+] = \sqrt{K_{\text{HA}}c_{\text{HA}} + K_{\text{HB}}c_{\text{HB}}}$$

若 $K_{\text{HA}}c_{\text{HA}} \gg K_{\text{HB}}c_{\text{HB}}$ → $[\text{H}^+] \approx \sqrt{K_{\text{HA}}c_{\text{HA}}}$（按较强酸处理）

② 一元弱酸 + 一元强酸混合（HA + HCl）：

PPT：弱酸 + 强酸混合溶液 pH 公式

PBE：$[\text{H}^+] = [\text{OH}^-] + [\text{A}^-] + c_a$（$c_a$ 为强酸浓度）

溶液呈酸性，$[\text{OH}^-]$ 可忽略：

$$[\text{H}^+] \approx [\text{A}^-] + c_a = \frac{K_a}{K_a+[\text{H}^+]} \cdot c + c_a$$

整理得近似式：

$$[\text{H}^+] = \frac{(c_a - K_a) + \sqrt{(c_a - K_a)^2 + 4K_a(c_a + c)}}{2}$$
简化情况：若 $c_a \gg K_a$ 且 $c_a \gg [\text{A}^-]$（即强酸足够强），则 $[\text{H}^+] \approx c_a$，弱酸的解离被完全抑制。

③ 一元弱碱 + 一元强碱混合（B + NaOH）：

同理，将上述公式中 $[\text{H}^+]$ 换成 $[\text{OH}^-]$，$K_a$ 换成 $K_b$，$c_a$ 换成 $c_b$。

Part 6：缓冲溶液 必考

PPT：缓冲溶液原理

Henderson-Hasselbalch 方程：

$$\text{pH} = \text{p}K_a + \lg\frac{c_b}{c_a} = \text{p}K_a + \lg\frac{c(\text{A}^-)}{c(\text{HA})}$$

缓冲溶液 pH 计算方法（PPT原文）：

  1. 先按最简式计算 $[\text{H}^+]$ 或 $[\text{OH}^-]$
  2. 然后验证 $[\text{H}^+]$（或 $[\text{OH}^-]$）是否远小于 $c_a$ 和 $c_b$
  3. 若不满足，需要修正（$c_a' = c_a + [\text{H}^+]$，$c_b' = c_b - [\text{H}^+]$）

PPT：缓冲液 pH 计算示例

PPT例：0.10 mol/L NH$_4$Cl + 0.20 mol/L NH$_3$·H$_2$O（pK$_b$=4.75）
$[\text{OH}^-] = K_b \cdot \frac{c_b}{c_a} = 10^{-4.75}\times\frac{0.20}{0.10} = 10^{-4.45}$
验证：$[\text{OH}^-] = 10^{-4.45} \ll c_a = 0.10$ ✓，最简式合理
pH = 14 - pOH = 14 - 4.45 = 9.55

缓冲溶液核心要点汇总：

要素内容
选择缓冲对$\text{p}K_a$ 尽量接近目标 pH（$|\text{pH}-\text{p}K_a| < 1$）
有效缓冲范围$\text{p}K_a \pm 1$（缓冲比在 1:10 ~ 10:1 之间）
缓冲能力最强缓冲比 = 1（$c_a = c_b$），此时 pH = pK$_a$
总浓度影响总浓度越大 → 缓冲容量越大（能抵御更多酸碱加入）
缓冲容量 $\beta$$\beta = 2.303C_{\text{总}}\frac{K_a[\text{H}^+]}{(K_a+[\text{H}^+])^2}$，缓冲比=1时$\beta$最大
经典考题：配 pH=9.00 缓冲液 → 选 NH$_3$·H$_2$O/NH$_4$Cl（pK$_a$=9.26，范围 8.3~10.3）

PPT：Henderson-Hasselbalch 方程推导

PPT：缓冲容量 β 的定义与公式

缓冲容量 $\beta$（定量描述缓冲能力）：

$$\beta = \frac{dC_b}{d\text{pH}} = 2.303\left(\frac{K_w}{[\text{H}^+]} + [\text{H}^+] + \frac{C_{\text{总}}K_a[\text{H}^+]}{(K_a + [\text{H}^+])^2}\right)$$
  • $\beta$ 最大时：$[\text{H}^+] = K_a$，即缓冲比 = 1（$c_a = c_b$），此时 $\beta_{\max} = 0.576C_{\text{总}}$
  • $C_{\text{总}}$ 越大 → $\beta$ 越大 → 能抵御更多酸碱的加入
  • 缓冲比偏离 1 越远 → $\beta$ 越小 → 缓冲范围外 $\beta \approx 0$

PPT：缓冲液 pH 计算示例 2

PPT例题：向 100 mL pH=5.00 的 HAc/NaAc 缓冲液（$C_{\text{总}}$=0.20 mol/L）中加入 1.00 mL 1.0 mol/L NaOH，求新 pH

加入 NaOH：$n(\text{NaOH}) = 0.001$ mol → HA 减少 0.001 mol，A⁻ 增加 0.001 mol
新 $c_a = (0.10\times0.100 - 0.001)/0.101$，新 $c_b = (0.10\times0.100 + 0.001)/0.101$
pH = pK$_a$ + lg($c_b'/c_a'$) ≈ 变化很小（这就是缓冲作用！）

PPT：终点误差定义——ep(终点)与sp(计量点)不一致引起的误差

定义：

$$E_t = \frac{\text{终点时过量（或不足）的滴定剂的量}}{\text{化学计量点应加入的滴定剂的量}} \times 100\%$$

$E_t > 0$：过滴定（多加了）；$E_t < 0$：欠滴定（少加了）

$$E_t = \frac{[\text{OH}^-]_{ep} - [\text{H}^+]_{ep}}{c_{\text{sp}}} \times 100\%$$

强酸滴定强碱：

$$E_t = \frac{[\text{H}^+]_{ep} - [\text{OH}^-]_{ep}}{c_{\text{sp}}} \times 100\%$$

其中 $c_{\text{sp}}$ 为化学计量点时被测物浓度（考虑体积稀释后）

林邦公式：

$$E_t = \frac{10^{\Delta\text{pH}} - 10^{-\Delta\text{pH}}}{\sqrt{K_t} \cdot c_{\text{sp,HA}}} \times 100\%$$

其中 $\Delta\text{pH} = \text{pH}_{ep} - \text{pH}_{sp}$，$K_t = K_a/K_w$

注意分子中的正负号！PPT上两种写法等价：
写法1：$\frac{10^{\Delta pH}-10^{-\Delta pH}}{\sqrt{K_t}\cdot c_{sp}}$
写法2：$\frac{10^{-\Delta pH}-10^{\Delta pH}}{\sqrt{K_t}\cdot c_{sp}}$（取决于$E_t$符号约定方向）
关键是看 $\Delta\text{pH}$ 的正负和最终 $E_t$ 的正负是否对应过滴定/欠滴定。

EDTA 的酸效应系数：

$$\alpha_{Y(H)} = \frac{[\text{Y}']}{[\text{Y}^{4-}]} = 1 + \frac{[\text{H}^+]}{K_{a6}} + \frac{[\text{H}^+]^2}{K_{a6}K_{a5}} + \cdots$$

共存金属离子 N 消耗 EDTA → 使有效 EDTA 减少

金属离子副反应系数：

通常只考虑酸效应：$\lg K'_{MY} \approx \lg K_{MY} - \lg\alpha_{Y(H)}$

PPT：影响条件电位的因素

Nernst 方程（25°C）：

$$\varphi = \varphi^0 + \frac{0.059}{n}\lg\frac{[\text{Ox}]}{[\text{Red}]}$$

条件电位 $\varphi^{0'}$：当 $c_{\text{Ox}} = c_{\text{Red}} = 1$ mol/L 时的实际电位，考虑了离子强度、酸度、络合、沉淀等副反应。

  • 沉淀效应：氧化态生成沉淀 → $\varphi^{0'}$ 降低；还原态生成沉淀 → $\varphi^{0'}$ 升高
  • 络合效应：取决于氧化态/还原态哪个络合更稳定

加 H$_3$PO$_4$ → Fe$^{3+}$ 与 PO$_4^{3-}$ 络合 → $\varphi^{0'}$ 降低 → 突跃增大（K$_2$Cr$_2$O$_7$ 法关键）

溶解度与溶度积 高频

PPT：溶度积类型（Kap、Ksp、K'sp）

PPT：溶解度影响因素

基本概念：

  • 活度积 $K_{ap}$：热力学常数，与温度有关，与离子强度无关
  • 溶度积 $K_{sp}$：用浓度表示，$K_{sp} = K_{ap}/(\gamma_+ \cdot \gamma_-)$
  • 条件溶度积 $K'_{sp}$：考虑副反应，$K'_{sp} = K_{sp} \cdot \alpha_M \cdot \alpha_A$
  • 固有溶解度 $s_0$：MA(s) ⇌ MA(aq)，与盐效应无关

溶解度计算：

  • MA 型：$s = \sqrt{K'_{sp}}$
  • MA$_2$ 型：$s = \left(\frac{K'_{sp}}{4}\right)^{1/3}$
  • M$_2$A 型：$s = \left(\frac{K'_{sp}}{4}\right)^{1/3}$

Lambert-Beer 定律 必考

PPT：Beer 定律

PPT：灵敏度指标（κ 和 Sandell 灵敏度）

基本公式：

$$A = -\lg T = \lg\frac{I_0}{I_t} = \kappa \cdot b \cdot c$$
  • $T$：透射比（$I_t/I_0$），$A$：吸光度
  • $\kappa$：摩尔吸光系数（L·mol$^{-1}$·cm$^{-1}$），$c$ 用 mol/L
  • $a$：吸光系数（L·g$^{-1}$·cm$^{-1}$），$c$ 用 g/L
  • 关系：$\kappa = M \cdot a$
  • $b$：光程（比色皿厚度），通常 1 cm

$\kappa$ 的意义：反映灵敏度，$\kappa$ 越大 → 灵敏度越高

$\kappa$ 范围灵敏度
< $10^4$低灵敏度
$10^4 \sim 5\times10^4$中等
$6\times10^4 \sim 10^5$高灵敏度

Sandell 灵敏度：$S = M/(\kappa \times 1000)$（$\mu$g/cm$^2$），$S$ 越小越灵敏

吸光度加和性 高频

多组分混合溶液：$A_{\text{总}} = A_1 + A_2 + \cdots = \kappa_1 b c_1 + \kappa_2 b c_2 + \cdots$

应用：多组分同时测定 → 在不同波长处测 $A$，联立方程求各组分浓度

物理因素：

  • 高浓度示差法：用浓度略低的标准液调 $T = 100\%$（代替溶剂空白）

络合物组成测定 高频

PPT：摩尔比法与连续变化法

摩尔比法（饱和法）：

  • 固定 $[\text{M}]$，改变 $[\text{R}]/[\text{M}]$
  • $A$ vs $[\text{R}]/[\text{M}]$ 作图，两直线交点处 $[\text{R}]/[\text{M}]$ = 络合比 $n$

连续变化法（Job 法）：

  • 保持 $[\text{M}] + [\text{R}]$ = 常数，改变 $[\text{M}]/[\text{R}]$ 比值
  • $A$ vs $x_M$（M 的摩尔分数）作图，最大值处 $x_M = 1/(1+n)$
  • 如 $x_M = 0.5$ → $n = 1$（1:1 络合）

十、十大必考高频知识点速览

1. 有效数字规则（含对数值） 必考

pH/pKa 只看小数部分位数。修约用"四舍六入五成双"。详见 第三章。

2. pH 计算公式选择决策树 必考

先判 $c/K_a$ 决定近似级别，两性物质用 $\sqrt{K_{a1}K_{a2}}$（验证条件）。详见 第五章。

3. 缓冲溶液 pH 与缓冲能力 必考

Henderson-Hasselbalch 方程；缓冲比 = 1 时能力最强；选 pK$_a$ 接近目标 pH。详见 第五章。

4. 质子条件式（PBE）书写 必考

确定零水准→得/失质子产物分左右→注意系数。详见 第五章。

5. 四大滴定可行性判据 必考

酸碱 $cK_a\geq10^{-8}$；分步 $K_{a1}/K_{a2}>10^5$；配位 $\lg cK'\geq6$；氧化还原 $\Delta\varphi\geq0.4$V。详见 判据对比表。

6. 终点误差计算（林邦公式） 必考

强酸-强碱与弱酸-强碱用不同公式。$E_t>0$ 过滴定，$E_t<0$ 欠滴定。详见 第五章。

7. 统计检验 Q→F→t 三步法 必考

顺序不能反。先清洗数据(Q)→比精密度(F)→比准确度(t)。详见 第三章。

8. EDTA 条件稳定常数计算 必考

$\lg K'_{MY} = \lg K_{MY} - \lg\alpha_{Y(H)}$；准确滴定要求 $\lg c_{sp}K'\geq6$。详见 第六章。

9. 氧化还原三大方法对比 必考

KMnO$_4$法（自身指示剂、标定三度一点）；K$_2$Cr$_2$O$_7$法（S-P混酸）；碘量法（淀粉指示剂时机）。详见 第七章。

10. Beer 定律与最佳测量范围 必考

$A = \kappa bc$；$A = 0.2\sim0.8$（$T = 15\%\sim65\%$）时误差最小。详见 第十章。

十一、十五大易错陷阱

陷阱 1：pH 有效数字只看小数部分

pH = 11.05 → 2位，不是4位！整数部分(11)只反映数量级($10^{-11}$)。

陷阱 2：四舍六入五成双

2.345→2.34（4偶不进）；2.335→2.34（3奇进位）；2.3451→2.35（5后有1，直接进）。

陷阱 3：增加平行实验 ≠ 消除系统误差

平行实验只减小随机误差。消除系统误差靠：对照实验、空白实验、仪器校正、更换方法。

陷阱 4：NaOH 吸收 CO$_2$ 后的影响

酚酞终点滴 HCl → 偏高（Na$_2$CO$_3$ 只反应到 NaHCO$_3$）。甲基橙终点 → 无影响（Na$_2$CO$_3$ 完全反应）。

陷阱 5：弱酸稀释10倍，突跃下限不变，上限降1个pH

下限由 pK$_a$ 决定（与浓度无关），上限由过量碱浓度决定。

陷阱 6：两性物质最简式不含浓度，但要验证条件

$[\text{H}^+]=\sqrt{K_{a1}K_{a2}}$ 使用条件：$c \geq 10K_{a1}$ 且 $cK_{a2} \geq 10K_w$。

陷阱 7：统计检验顺序 Q→F→t 不能反

含离群值做 t 检验会得到错误结论；F 检验不通过时 t 检验公式不同。

陷阱 8：终点误差 $E_t$ 正负号

$E_t > 0$ 过滴定（多加了），$E_t < 0$ 欠滴定（少加了）。需根据公式仔细判断。

陷阱 9：配位滴定 $K'_{MY}$ 随 pH 变化

pH 越高→$\alpha_{Y(H)}$ 越小→$K'_{MY}$ 越大→越有利。但 pH 太高金属离子水解！

陷阱 10：碘量法淀粉指示剂加入时机

直接碘量法：开始就加。间接碘量法：近终点（浅黄色时）才加！高浓度 I$_2$ 与淀粉形成难分解络合物。

陷阱 11：佛尔哈德法测 Cl$^-$ 的沉淀转化

AgSCN 比 AgCl 更难溶，SCN$^-$ 会转化 AgCl → AgSCN → 结果偏高。需过滤或加硝基苯包裹。

陷阱 12：共沉淀 vs 后沉淀

陈化可减少表面吸附和包藏（共沉淀），但加重后沉淀。处理后沉淀要缩短放置时间。

陷阱 13：Beer 定律偏离——非单色光

实际光源有带宽→高浓度时产生负偏离（测得 $A$ 偏小）。应选好的单色器、在 $\lambda_{\max}$ 处测量。

陷阱 14：K$_2$Cr$_2$O$_7$ 法中 SnCl$_2$ 过量处理

加 HgCl$_2$ 后应出现白色丝状 Hg$_2$Cl$_2$ 沉淀。若出现灰/黑色（Hg）→ Sn$^{2+}$ 过量太多，必须重做！

陷阱 15：KMnO$_4$ 标定条件"三度一点"

温度 70~80°C（非沸腾！）、酸度 ~1 mol/L H$_2$SO$_4$（不用 HCl/HNO$_3$）、速度先慢后快（Mn$^{2+}$ 自催化）、终点粉红30s不褪。

  • 吸光度加和性：混合溶液$A_总=A_1+A_2$

计算题核心考点

  • 水硬度测定：EDTA滴定Ca²⁺+Mg²⁺总量（pH10/EBT），再pH12/NN单独测Ca²⁺，差减得Mg²⁺
  • Zn/Al分别测定：pH5.5滴Zn²⁺(XO)，加F⁻掩蔽Al³⁺；或加过量EDTA加热测Al³⁺总量后减
  • 钢中铬含量（K₂Cr₂O₇法）：先氧化Cr³⁺→Cr₂O₇²⁻，再加过量Fe²⁺，最后K₂Cr₂O₇回滴
  • BaSO₄重量分析：$w(\text{S})=m(\text{BaSO}_4)\times F/m_样$，$F=M_S/M_{BaSO_4}$
  • Zn+Pb实验设计：pH5~6用XO先滴Zn+Pb总量→加KCN掩蔽Zn→另取一份用EDTA滴定

小测2 易错考点补充

  • 返滴定法求Al³⁺：加入的 EDTA 总量 − 回滴消耗的 Zn²⁺ = 与 Al³⁺ 反应的 EDTA → n(Al³⁺)
  • lgK'MY 不同pH下不同：同一金属在不同pH下 lgK' 值变化巨大，要查对应pH的 lg$\alpha_{Y(H)}$
  • 碘量法中 2S₂O₃²⁻ ~ I₂：化学计量关系是 2:1，不是 1:1！
  • 重量分析中"灼烧"：沉淀形→称量形的转化（如 Al(OH)₃ 灼烧→ Al₂O₃）

2022期末 实验设计题要点

Na₃PO₄ + NaOH 混合液测定：
① 酚酞终点：NaOH 完全中和 + Na₃PO₄→Na₂HPO₄（消耗 1mol HCl/mol Na₃PO₄）
② 甲基橙终点：Na₂HPO₄→NaH₂PO₄（又消耗 1mol HCl/mol Na₃PO₄）
因此 $V_1 > V_2$：n(NaOH)=c(V₁−V₂)，n(Na₃PO₄)=cV₂
本质就是双指示剂法的变体！理解原理后，只需根据各步反应写出 V₁、V₂ 与组分的对应关系。

2022期末 条件电位计算要点

络合效应对条件电位的影响：
$\varphi^{0'} = \varphi^0 + \frac{0.059}{n}\lg\frac{\alpha_{\text{Red}}}{\alpha_{\text{Ox}}}$
• 若氧化态络合更稳定（$\alpha_{\text{Ox}} > \alpha_{\text{Red}}$）→ $\varphi^{0'}$ 降低
• 若还原态络合更稳定（$\alpha_{\text{Red}} > \alpha_{\text{Ox}}$）→ $\varphi^{0'}$ 升高
沉淀效应：类似处理，用 $K_{sp}$ 计算等效 $\alpha$
