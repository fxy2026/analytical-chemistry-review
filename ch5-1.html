第五章(上) 酸碱平衡

本章是分析化学考试中最核心的章节之一，涉及大量计算公式和判据条件。务必掌握分布分数、质子条件式和pH计算公式的完整体系。

一、活度与活度系数

1.1 基本概念

在真实溶液中，离子间的静电作用使其有效浓度(活度)低于实际浓度。活度 $a$ 与浓度 $c$ 的关系为：

$$a = \gamma \cdot c$$

其中 $\gamma$ 为活度系数（无量纲），$0 < \gamma \leq 1$。在无限稀释溶液中 $\gamma \to 1$，$a \to c$。

1.2 离子强度

离子强度 $I$ 描述溶液中离子氛的强弱：

$$I = \frac{1}{2}\sum c_i z_i^2$$

其中 $c_i$ 为第 $i$ 种离子的摩尔浓度，$z_i$ 为其电荷数。

例：计算 0.10 mol/L $\text{Na}_2\text{SO}_4$ 溶液的离子强度。
解：
$c(\text{Na}^+) = 0.20\;\text{mol/L}$，$z = +1$；$c(\text{SO}_4^{2-}) = 0.10\;\text{mol/L}$，$z = -2$
$$I = \frac{1}{2}(0.20 \times 1^2 + 0.10 \times 2^2) = \frac{1}{2}(0.20 + 0.40) = 0.30$$

1.3 Debye-Huckel 公式

Debye-Huckel 极限公式（$I < 0.1$）
$$\lg \gamma_i = -0.512\, z_i^2 \sqrt{I}$$

适用于离子强度较低的稀溶液。离子电荷越高、离子强度越大，$\gamma$ 越小。

扩展Debye-Huckel公式（$I < 0.2$）：

$$\lg \gamma_i = \frac{-0.512\, z_i^2 \sqrt{I}}{1 + \alpha B \sqrt{I}}$$

其中 $\alpha$ 为离子有效半径（pm），$B$ 为常数。

在分析化学的一般计算中，当溶液浓度较低时，常近似令 $\gamma = 1$，即 $a \approx c$。但对于高电荷离子或高离子强度溶液需考虑活度校正。

二、酸碱质子理论 (Bronsted-Lowry)

2.1 基本定义

  • 酸（acid）：能给出质子（$\text{H}^+$）的物质 —— 质子给予体（proton donor）
  • 碱（base）：能接受质子（$\text{H}^+$）的物质 —— 质子接受体（proton acceptor）
$$\text{Acid} \rightleftharpoons \text{Base} + \text{H}^+$$

酸失去质子后变成共轭碱，碱得到质子后变成共轭酸，形成共轭酸碱对。

2.2 共轭酸碱对举例

酸 (Acid)共轭碱 (Conjugate Base)
$\text{HCl}$$\rightleftharpoons$$\text{Cl}^-$
$\text{HAc}$$\rightleftharpoons$$\text{Ac}^-$
$\text{NH}_4^+$$\rightleftharpoons$$\text{NH}_3$
$\text{H}_2\text{PO}_4^-$$\rightleftharpoons$$\text{HPO}_4^{2-}$
$\text{H}_2\text{O}$$\rightleftharpoons$$\text{OH}^-$
$\text{H}_3\text{O}^+$$\rightleftharpoons$$\text{H}_2\text{O}$

2.3 共轭酸碱对的 $K_a$ 与 $K_b$ 关系

核心关系式
$$K_a \cdot K_b = K_w$$

即：

$$pK_a + pK_b = pK_w = 14.00 \quad (25°\text{C})$$

酸越强（$K_a$ 越大），其共轭碱越弱（$K_b$ 越小），反之亦然。

2.4 两性物质

两性物质（amphoteric substance）既可作为酸也可作为碱。例如：

  • $\text{H}_2\text{O}$：既是酸（给出$\text{H}^+$变$\text{OH}^-$）又是碱（接受$\text{H}^+$变$\text{H}_3\text{O}^+$）
  • $\text{HCO}_3^-$：作酸 $\rightarrow \text{CO}_3^{2-} + \text{H}^+$；作碱 $+ \text{H}^+ \rightarrow \text{H}_2\text{CO}_3$
  • $\text{H}_2\text{PO}_4^-$、$\text{HPO}_4^{2-}$：磷酸的中间形体
  • 氨基酸：含 $-\text{NH}_2$（碱性）和 $-\text{COOH}$（酸性）

四、分布分数 (Distribution Fraction)

分布分数是本章最重要的概念之一，考试必考！必须熟练掌握一元、二元、三元酸的分布分数公式。

4.1 一元酸 HA

一元弱酸 $\text{HA} \rightleftharpoons \text{H}^+ + \text{A}^-$，$K_a$
$$\delta_{\text{HA}} = \frac{[\text{HA}]}{c} = \frac{[\text{H}^+]}{[\text{H}^+] + K_a}$$ $$\delta_{\text{A}^-} = \frac{[\text{A}^-]}{c} = \frac{K_a}{[\text{H}^+] + K_a}$$

其中 $c = [\text{HA}] + [\text{A}^-]$ 为分析浓度。

当 $[\text{H}^+] = K_a$ 即 $\text{pH} = pK_a$ 时，$\delta_{\text{HA}} = \delta_{\text{A}^-} = 0.5$（两曲线交点）。

4.2 二元酸 $\text{H}_2\text{A}$

$\text{H}_2\text{A} \rightleftharpoons \text{H}^+ + \text{HA}^- \rightleftharpoons 2\text{H}^+ + \text{A}^{2-}$

令 $h = [\text{H}^+]$，分母 $D = h^2 + h K_{a1} + K_{a1} K_{a2}$

$$\delta_{\text{H}_2\text{A}} = \frac{h^2}{D}$$ $$\delta_{\text{HA}^-} = \frac{h \cdot K_{a1}}{D}$$ $$\delta_{\text{A}^{2-}} = \frac{K_{a1} K_{a2}}{D}$$
中间形体 $\text{HA}^-$ 的最大分布分数出现在：
$$\text{pH} = \frac{pK_{a1} + pK_{a2}}{2}$$ 这是一个非常重要的结论，常用于判断两性物质溶液的 pH 近似值。

4.3 三元酸 $\text{H}_3\text{A}$（以 $\text{H}_3\text{PO}_4$ 为例）

三元酸分布分数

令 $h = [\text{H}^+]$，分母 $D = h^3 + h^2 K_{a1} + h K_{a1}K_{a2} + K_{a1}K_{a2}K_{a3}$

$$\delta_{\text{H}_3\text{A}} = \frac{h^3}{D}$$ $$\delta_{\text{H}_2\text{A}^-} = \frac{h^2 K_{a1}}{D}$$ $$\delta_{\text{HA}^{2-}} = \frac{h \cdot K_{a1}K_{a2}}{D}$$ $$\delta_{\text{A}^{3-}} = \frac{K_{a1}K_{a2}K_{a3}}{D}$$

$\text{H}_3\text{PO}_4$：$pK_{a1}=2.12$，$pK_{a2}=7.20$，$pK_{a3}=12.36$

$\text{H}_2\text{PO}_4^-$ 的最大值在 $\text{pH} = (2.12+7.20)/2 = 4.66$

$\text{HPO}_4^{2-}$ 的最大值在 $\text{pH} = (7.20+12.36)/2 = 9.78$

规律总结：对于多元酸 $\text{H}_n\text{A}$，第 $i$ 个中间形体 $\text{H}_{n-i}\text{A}^{i-}$ 的分布分数最大值出现在 $\text{pH} = (pK_{ai} + pK_{a,i+1})/2$。各形体的 $\delta$ 之和恒等于1。

五、质子条件式 (PBE)

质子条件式是书写精确pH计算方程的基础，考试中经常要求直接写出PBE。务必掌握方法！

5.1 书写方法

  1. 选择参考水准（零水准）：通常选溶液中大量存在的物种（溶质的原始形态 + $\text{H}_2\text{O}$）
  2. 得质子产物列在等式左边（参考水准物种得到质子后的产物）
  3. 失质子产物列在等式右边（参考水准物种失去质子后的产物）
  4. 等式两边各物种的浓度之和相等（得到和失去的质子数要配平）

5.2 经典例题

例1：$\text{H}_3\text{PO}_4$ 溶液
参考水准：$\text{H}_3\text{PO}_4$，$\text{H}_2\text{O}$
PBE：
$\text{H}_3\text{PO}_4$ 失去1个质子 $\to \text{H}_2\text{PO}_4^-$，失去2个 $\to \text{HPO}_4^{2-}$，失去3个 $\to \text{PO}_4^{3-}$
$\text{H}_2\text{O}$ 得到1个质子 $\to \text{H}_3\text{O}^+$（即$\text{H}^+$），失去1个 $\to \text{OH}^-$ $$[\text{H}^+] = [\text{H}_2\text{PO}_4^-] + 2[\text{HPO}_4^{2-}] + 3[\text{PO}_4^{3-}] + [\text{OH}^-]$$ 注意：$\text{HPO}_4^{2-}$ 比参考水准少2个质子，所以系数为2；$\text{PO}_4^{3-}$ 系数为3。
例2：$\text{NaH}_2\text{PO}_4$ 溶液
参考水准：$\text{H}_2\text{PO}_4^-$，$\text{H}_2\text{O}$（$\text{Na}^+$ 不参与质子转移）
PBE：
得质子：$\text{H}_2\text{PO}_4^-$ + $\text{H}^+$ $\to$ $\text{H}_3\text{PO}_4$
失质子：$\text{H}_2\text{PO}_4^-$ $\to$ $\text{HPO}_4^{2-}$ + $\text{H}^+$；$\to$ $\text{PO}_4^{3-}$ + 2$\text{H}^+$ $$[\text{H}_3\text{PO}_4] + [\text{H}^+] = [\text{HPO}_4^{2-}] + 2[\text{PO}_4^{3-}] + [\text{OH}^-]$$
例3：$\text{NaHCO}_3$ 溶液
参考水准：$\text{HCO}_3^-$，$\text{H}_2\text{O}$
PBE：
$$[\text{H}_2\text{CO}_3] + [\text{H}^+] = [\text{CO}_3^{2-}] + [\text{OH}^-]$$
例4：$\text{Na}_2\text{HPO}_4$ 溶液
参考水准：$\text{HPO}_4^{2-}$，$\text{H}_2\text{O}$
PBE：
得质子方向：$\text{HPO}_4^{2-}$ $\to$ $\text{H}_2\text{PO}_4^-$（得1个），$\to$ $\text{H}_3\text{PO}_4$（得2个） $$[\text{H}_2\text{PO}_4^-] + 2[\text{H}_3\text{PO}_4] + [\text{H}^+] = [\text{PO}_4^{3-}] + [\text{OH}^-]$$
例5：$\text{NH}_4\text{Ac}$ 溶液
参考水准：$\text{NH}_4^+$，$\text{Ac}^-$，$\text{H}_2\text{O}$
PBE：
$\text{NH}_4^+$ 失去质子 $\to$ $\text{NH}_3$
$\text{Ac}^-$ 得到质子 $\to$ $\text{HAc}$ $$[\text{HAc}] + [\text{H}^+] = [\text{NH}_3] + [\text{OH}^-]$$

六、pH 计算公式体系

本节是整章的核心！所有公式和判据条件都需要熟记。考试中约30%-40%的计算题出自此节。

6.1 强酸/强碱

强酸：$c \geq 10^{-6}$ mol/L 时，$[\text{H}^+] = c$，$\text{pH} = -\lg c$

强碱：$c \geq 10^{-6}$ mol/L 时，$[\text{OH}^-] = c$，$\text{pOH} = -\lg c$，$\text{pH} = 14 + \lg c$

极稀强酸（$c < 10^{-6}$）：需考虑水的自偶解离

$$[\text{H}^+] = \frac{c + \sqrt{c^2 + 4K_w}}{2}$$

6.2 一元弱酸 HA

精确公式（由PBE推导）
$$[\text{H}^+] = \sqrt{K_a(c - [\text{H}^+] + [\text{OH}^-]) + K_w}$$

简化条件与对应公式：

条件含义公式
$cK_a \geq 20K_w$可忽略水的解离不需要 $K_w$ 项
$c/K_a \geq 400$可忽略解离度（弱酸近似）分母中 $c-[\text{H}^+] \approx c$

最简公式（两个条件都满足时）：

$$[\text{H}^+] = \sqrt{K_a \cdot c} \quad \Rightarrow \quad \text{pH} = \frac{1}{2}(pK_a - \lg c)$$

仅满足 $cK_a \geq 20K_w$ 但 $c/K_a < 400$（需解一元二次方程）：

$$[\text{H}^+]^2 + K_a[\text{H}^+] - K_a c = 0$$ $$[\text{H}^+] = \frac{-K_a + \sqrt{K_a^2 + 4K_ac}}{2}$$

$cK_a < 20K_w$（极弱酸或极稀溶液，需保留水的解离）：

$$[\text{H}^+]^2 + K_a[\text{H}^+] - (K_ac + K_w) = 0$$

6.3 一元弱碱 B

将弱酸公式中的 $[\text{H}^+]$ 换为 $[\text{OH}^-]$，$K_a$ 换为 $K_b$：

最简公式（$cK_b \geq 20K_w$ 且 $c/K_b \geq 400$）：

$$[\text{OH}^-] = \sqrt{K_b \cdot c} \quad \Rightarrow \quad \text{pOH} = \frac{1}{2}(pK_b - \lg c)$$ $$\text{pH} = 14 - \text{pOH} = 14 - \frac{1}{2}(pK_b - \lg c)$$

别忘了 $K_b = K_w / K_a$。

6.4 两性物质

两性物质 $\text{HA}^-$（如 $\text{NaHCO}_3$、$\text{NaH}_2\text{PO}_4$）

精确公式：

$$[\text{H}^+] = \sqrt{\frac{K_{a1}(K_{a2}c + K_w)}{c + K_{a1}}}$$

简化条件与公式：

条件简化公式
$c \gg K_{a1}$ 且 $cK_{a2} \gg K_w$$[\text{H}^+] = \sqrt{K_{a1} K_{a2}}$，$\text{pH} = \frac{1}{2}(pK_{a1} + pK_{a2})$
$c \gg K_{a1}$ 但 $cK_{a2}$ 与 $K_w$ 相当$[\text{H}^+] = \sqrt{K_{a1}(K_{a2} + K_w/c)}$
$c$ 与 $K_{a1}$ 相当使用完整公式
判断两性物质溶液酸碱性：
若 $K_{a2} > K_b = K_w/K_{a1}$，即 $K_{a1}K_{a2} > K_w$，则溶液呈酸性；反之呈碱性。
例如 $\text{NaHCO}_3$：$K_{a1}=4.3\times10^{-7}$，$K_{a2}=5.6\times10^{-11}$，$K_{a1}K_{a2}=2.4\times10^{-17} > K_w=10^{-14}$？不对，$2.4\times10^{-17} < 10^{-14}$，所以 $\text{NaHCO}_3$ 溶液呈碱性。

6.5 多元酸

多元酸（如 $\text{H}_3\text{PO}_4$）通常可按一元弱酸处理，只使用 $K_{a1}$，前提是验证：

$$\frac{2K_{a2}}{[\text{H}^+]} < 0.05$$

即第二步解离对 $[\text{H}^+]$ 的贡献不超过5%。因大多数多元酸的 $K_{a1}/K_{a2} > 10^4$，此条件通常满足。

七、缓冲溶液

7.1 Henderson-Hasselbalch 方程

缓冲溶液pH计算

由弱酸 HA 及其共轭碱 $\text{A}^-$（如NaA）组成的缓冲溶液：

$$\text{pH} = pK_a + \lg\frac{c_b}{c_a} = pK_a + \lg\frac{[\text{A}^-]}{[\text{HA}]}$$

其中 $c_b$ 为共轭碱浓度，$c_a$ 为弱酸浓度。

对于弱碱缓冲体系：

$$\text{pOH} = pK_b + \lg\frac{c_a}{c_b}$$

7.2 缓冲容量 $\beta$

定义：使1 L缓冲溶液pH改变1个单位所需的强酸或强碱的物质的量。

$$\beta = \frac{dc_b}{d(\text{pH})} = -\frac{dc_a}{d(\text{pH})}$$

缓冲容量公式：

$$\beta = 2.303 \cdot \frac{c \cdot K_a \cdot [\text{H}^+]}{(K_a + [\text{H}^+])^2}$$

7.3 最大缓冲容量

当 $\text{pH} = pK_a$（即 $c_a = c_b$）时，缓冲容量最大
$$\beta_{\max} = 2.303 \times \frac{c}{4} = 0.576c$$

其中 $c$ 为缓冲体系的总浓度。

7.4 有效缓冲范围

$$\text{pH} = pK_a \pm 1$$

即 $c_b / c_a$ 在 $1/10$ 到 $10$ 之间时缓冲效果良好。超出此范围，缓冲能力急剧下降。

7.5 缓冲溶液的配制

  1. 选择缓冲对：选 $pK_a$ 尽量接近所需 pH 的弱酸及其共轭碱
  2. 计算比例：由 Henderson-Hasselbalch 方程算出 $c_b/c_a$
  3. 确定浓度：总浓度一般 0.01~0.1 mol/L，浓度越高缓冲容量越大
  4. 用 pH 计校准：配好后用标准缓冲液校准的 pH 计测量，必要时微调

八、综合例题

例题1：计算 0.10 mol/L $\text{HAc}$ 溶液的 pH。已知 $K_a = 1.8 \times 10^{-5}$。
解：
Step 1：判断条件
$cK_a = 0.10 \times 1.8\times10^{-5} = 1.8\times10^{-6} \gg 20K_w = 2.0\times10^{-13}$ ✓ 忽略水解离
$c/K_a = 0.10 / 1.8\times10^{-5} = 5556 > 400$ ✓ 可用最简式

Step 2：计算 $$[\text{H}^+] = \sqrt{K_a \cdot c} = \sqrt{1.8\times10^{-5} \times 0.10} = \sqrt{1.8\times10^{-6}}$$ $$= 1.34 \times 10^{-3}\;\text{mol/L}$$ $$\text{pH} = -\lg(1.34\times10^{-3}) = 2.87$$ Step 3：验证
$[\text{H}^+]/c = 1.34\times10^{-3}/0.10 = 1.3\%$，解离度很小，近似合理。
例题2：计算 0.10 mol/L $\text{NaHCO}_3$ 溶液的 pH。已知 $\text{H}_2\text{CO}_3$：$K_{a1}=4.3\times10^{-7}$，$K_{a2}=5.6\times10^{-11}$。
解：
$\text{NaHCO}_3$ 是两性物质，$\text{HCO}_3^-$ 是 $\text{H}_2\text{CO}_3$ 的中间形体。

检查简化条件：
$c = 0.10 \gg K_{a1} = 4.3\times10^{-7}$ ✓
$cK_{a2} = 0.10 \times 5.6\times10^{-11} = 5.6\times10^{-12}$
$K_w = 1.0\times10^{-14}$
$cK_{a2}/K_w = 560 \gg 1$ ✓ 可忽略 $K_w$ 项

用最简两性公式： $$[\text{H}^+] = \sqrt{K_{a1} \cdot K_{a2}} = \sqrt{4.3\times10^{-7} \times 5.6\times10^{-11}}$$ $$= \sqrt{2.408\times10^{-17}} = 4.91\times10^{-9}$$ $$\text{pH} = -\lg(4.91\times10^{-9}) = 8.31$$

溶液呈碱性，符合预期（$K_{a1}K_{a2} < K_w$）。

例题3：配制 pH = 5.00 的缓冲溶液，现有 0.20 mol/L $\text{HAc}$（$pK_a = 4.74$）和 0.20 mol/L $\text{NaAc}$ 溶液。需各取多少 mL 配成 100 mL 缓冲液？
解：
由 Henderson-Hasselbalch 方程： $$\text{pH} = pK_a + \lg\frac{c_b}{c_a}$$ $$5.00 = 4.74 + \lg\frac{c_b}{c_a}$$ $$\lg\frac{c_b}{c_a} = 0.26 \quad \Rightarrow \quad \frac{c_b}{c_a} = 10^{0.26} = 1.82$$ 设取 $V_1$ mL $\text{HAc}$，$V_2$ mL $\text{NaAc}$，$V_1 + V_2 = 100$
两溶液浓度相同，故 $V_2/V_1 = 1.82$
$V_1 = 100/(1+1.82) = 35.5$ mL（$\text{HAc}$）
$V_2 = 64.5$ mL（$\text{NaAc}$）

验证：缓冲范围 $pK_a \pm 1 = 3.74 \sim 5.74$，pH = 5.00 在范围内 ✓
$c_b/c_a = 1.82$，在 $1/10 \sim 10$ 之间 ✓
例题4：计算 $1.0\times10^{-3}$ mol/L $\text{NH}_3 \cdot \text{H}_2\text{O}$ 的 pH。已知 $K_b = 1.8\times10^{-5}$。
解：
检查条件：
$cK_b = 1.0\times10^{-3} \times 1.8\times10^{-5} = 1.8\times10^{-8}$
$20K_w = 2.0\times10^{-13}$
$cK_b \geq 20K_w$ ✓
$c/K_b = 1.0\times10^{-3}/1.8\times10^{-5} = 55.6 < 400$ ✗ 不能用最简式！

需解一元二次方程： $$[\text{OH}^-]^2 + K_b[\text{OH}^-] - K_b c = 0$$ $$[\text{OH}^-]^2 + 1.8\times10^{-5}[\text{OH}^-] - 1.8\times10^{-8} = 0$$ $$[\text{OH}^-] = \frac{-1.8\times10^{-5} + \sqrt{(1.8\times10^{-5})^2 + 4\times1.8\times10^{-8}}}{2}$$ $$= \frac{-1.8\times10^{-5} + \sqrt{3.24\times10^{-10} + 7.2\times10^{-8}}}{2}$$ $$= \frac{-1.8\times10^{-5} + \sqrt{7.23\times10^{-8}}}{2}$$ $$= \frac{-1.8\times10^{-5} + 2.69\times10^{-4}}{2} = 1.26\times10^{-4}$$ $$\text{pOH} = -\lg(1.26\times10^{-4}) = 3.90$$ $$\text{pH} = 14.00 - 3.90 = 10.10$$

若用最简式：$[\text{OH}^-]=\sqrt{1.8\times10^{-8}}=1.34\times10^{-4}$，pH=10.13，误差约0.03，虽然不大但不够严谨。

九、常见考试陷阱

陷阱1：忘记检查简化条件
直接套用 $[\text{H}^+]=\sqrt{K_ac}$ 而不检查 $c/K_a \geq 400$ 和 $cK_a \geq 20K_w$，导致结果误差过大。特别是浓度很小或 $K_a$ 较大的弱酸。
陷阱2：两性物质用错公式
$\text{NaHCO}_3$、$\text{NaH}_2\text{PO}_4$ 等是两性物质，不能简单用弱酸或弱碱公式！必须用两性物质公式 $[\text{H}^+] = \sqrt{K_{a1}K_{a2}}$（满足简化条件时）。
陷阱3：PBE 中系数写错
多元酸的 PBE 中，失去/得到多个质子的物种前要乘以相应系数。例如 $\text{H}_3\text{PO}_4$ 溶液的 PBE 中 $[\text{HPO}_4^{2-}]$ 前系数为2，$[\text{PO}_4^{3-}]$ 前系数为3。
陷阱4：分布分数的分母
写分布分数时，分母是所有形体浓度之和。二元酸分母 $D = h^2 + hK_{a1} + K_{a1}K_{a2}$ 中不要漏掉任何一项。
陷阱5：$K_a$、$K_b$ 关系搞混
$K_a \cdot K_b = K_w$ 是对共轭酸碱对成立。$\text{NH}_4^+$ 的 $K_a$ 和 $\text{NH}_3$ 的 $K_b$ 满足此关系，但 $\text{H}_2\text{CO}_3$ 的 $K_{a1}$ 和 $\text{HCO}_3^-$ 的 $K_{b}$ 不构成共轭关系（$\text{HCO}_3^-$ 的共轭酸是 $\text{H}_2\text{CO}_3$，对应 $K_{b2}=K_w/K_{a1}$）。
陷阱6：缓冲溶液稀释的影响
Henderson-Hasselbalch 方程表明，缓冲溶液的 pH 主要取决于 $c_b/c_a$ 的比值，稀释时比值不变所以 pH 几乎不变。但总浓度降低导致缓冲容量减小。
考前速记清单：
  • $K_a \cdot K_b = K_w = 10^{-14}$（25°C）
  • 最简式条件：$cK_a \geq 20K_w$ 且 $c/K_a \geq 400$
  • 两性物质：$\text{pH} = \frac{1}{2}(pK_{a1}+pK_{a2})$
  • 缓冲范围：$pK_a \pm 1$
  • 中间形体最大分布：$\text{pH}=(pK_{ai}+pK_{a,i+1})/2$
  • 多元酸按一元处理条件：$2K_{a2}/[\text{H}^+]<0.05$
