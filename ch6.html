第六章 配位滴定法

6.1 EDTA的性质与配位特点

EDTA的结构

EDTA（乙二胺四乙酸）是一种六元酸（$\text{H}_6\text{Y}^{2+}$），含有 2 个氨基氮和 4 个羧基氧，共提供 6 个配位原子。通常使用其二钠盐 $\text{Na}_2\text{H}_2\text{Y}\cdot 2\text{H}_2\text{O}$（$M = 372.24$）。

EDTA 的六个质子的酸性：前两个质子来自氨基 $\text{NH}^+$（$\text{p}K_{a1}=0.9$, $\text{p}K_{a2}=1.6$），后四个来自羧基（$\text{p}K_{a3}=2.07$, $\text{p}K_{a4}=2.75$, $\text{p}K_{a5}=6.24$, $\text{p}K_{a6}=10.34$）。

EDTA配合物的特点

  • 广泛性：能与几乎所有金属离子形成配合物
  • 1:1 配位：无论金属离子电荷或半径如何，均形成 $\text{MY}^{n-4}$ 型 1:1 配合物
  • 稳定性高：多数 $\lg K_{MY}$ 在 15 以上
  • 水溶性好：配合物均可溶于水
  • 反应速率快：配位反应速度一般较快

6.2 副反应系数

EDTA的副反应系数

在配位滴定中，EDTA 参与的副反应主要有两类：

酸效应系数 $\alpha_{Y(H)}$

反映 H$^+$ 对 EDTA 配位能力的影响：

$$\alpha_{Y(H)} = \frac{[\text{Y}']}{[\text{Y}^{4-}]} = 1 + \frac{[\text{H}^+]}{K_{a6}} + \frac{[\text{H}^+]^2}{K_{a6}K_{a5}} + \cdots$$

酸度越大，$\alpha_{Y(H)}$ 越大，EDTA 配位能力越弱。

共存离子效应系数 $\alpha_{Y(N)}$

溶液中共存金属离子 N 与 EDTA 形成配合物，消耗 Y：

$$\alpha_{Y(N)} = 1 + K_{NY}[\text{N}]$$

若多种共存离子：$\alpha_Y = \alpha_{Y(H)} + \alpha_{Y(N_1)} + \alpha_{Y(N_2)} + \cdots - (k-1)$，其中 $k$ 为副反应总数。通常简化为 $\alpha_Y \approx \alpha_{Y(H)} \cdot \alpha_{Y(N)}$（当各项远大于 1 时）。

金属离子的副反应系数

辅助配位效应 $\alpha_{M(L)}$

金属离子 M 与辅助配位体 L（如 $\text{NH}_3$）反应：

$$\alpha_{M(L)} = 1 + \beta_1[\text{L}] + \beta_2[\text{L}]^2 + \cdots + \beta_n[\text{L}]^n$$
羟基效应 $\alpha_{M(OH)}$

金属离子水解导致参与配位反应的游离金属减少：

$$\alpha_{M(OH)} = 1 + \beta_1^{OH}\frac{K_w}{[\text{H}^+]} + \beta_2^{OH}\left(\frac{K_w}{[\text{H}^+]}\right)^2 + \cdots$$

pH 越高，$\alpha_{M(OH)}$ 越大。

注意：$\alpha_{M(OH)}$ 通常在 pH 较高时才显著，且对不同金属影响差异很大。某些金属离子（如 $\text{Fe}^{3+}$, $\text{Al}^{3+}$）在较低 pH 下即开始水解。

6.3 条件稳定常数

条件稳定常数的定义
$$K'_{MY} = \frac{[\text{MY}]}{[\text{M}'][\text{Y}']} = \frac{K_{MY}}{\alpha_M \cdot \alpha_Y}$$

取对数形式：

$$\boxed{\lg K'_{MY} = \lg K_{MY} - \lg\alpha_M - \lg\alpha_Y}$$

其中 $\lg\alpha_M = \lg\alpha_{M(L)} + \lg\alpha_{M(OH)}$（若两者都显著），$\lg\alpha_Y = \lg\alpha_{Y(H)} + \lg\alpha_{Y(N)}$（通常 $\alpha_{Y(H)}$ 占主导）。

条件稳定常数是在特定实验条件下（特定 pH、辅助配位体浓度等）的有效稳定常数，它综合反映了各种副反应对配位平衡的影响。
例题：在 pH = 5.0 时，$\lg\alpha_{Y(H)} = 6.45$，计算 $\text{Zn}^{2+}$-EDTA 的条件稳定常数（忽略其他副反应）。
解：

$\lg K_{ZnY} = 16.5$

$\lg K'_{ZnY} = \lg K_{ZnY} - \lg\alpha_{Y(H)} = 16.5 - 6.45 = 10.05$

条件稳定常数足够大，可以准确滴定。

6.4 配位滴定曲线

配位滴定曲线以 $pM'$（$pM' = -\lg[\text{M}']$）为纵轴，滴定剂加入体积（或滴定百分数）为横轴。

滴定曲线的关键点

  • 化学计量点： $$\boxed{pM'_{sp} = \frac{1}{2}(\lg K'_{MY} + pc_{sp})}$$ 其中 $pc_{sp} = -\lg c_{sp}$，$c_{sp}$ 为化学计量点时 MY 的浓度。
  • 滴定突跃：$\lg c_{sp} K'_{MY}$ 越大，突跃越大。
  • 化学计量点前（如 50%）：$pM' = -\lg\frac{c_M}{2}$（简单稀释计算）
  • 化学计量点后（如 200%）：根据过量 Y 的浓度计算
例题：用 0.020 mol/L EDTA 滴定 20.00 mL 0.020 mol/L $\text{Zn}^{2+}$，pH = 5.0，$\lg K'_{ZnY} = 10.05$，求化学计量点的 $pZn'$。
解：

$c_{sp} = \frac{0.020 \times 20.00}{40.00} = 0.010\;\text{mol/L}$，故 $pc_{sp} = 2.00$

$pZn'_{sp} = \frac{1}{2}(10.05 + 2.00) = 6.03$

6.5 金属指示剂

原理

金属指示剂（In）本身是一种有机配位剂，在游离态（$\text{In}$）和与金属配合态（$\text{MIn}$）颜色不同。

滴定终点：$\text{MIn} + \text{Y} \rightarrow \text{MY} + \text{In}$（颜色由 MIn 的颜色变为 In 的颜色）。

指示剂使用中的问题

封闭现象（僵化）：某些金属（如 $\text{Al}^{3+}$, $\text{Fe}^{3+}$, $\text{Ni}^{2+}$, $\text{Co}^{2+}$）与指示剂形成的配合物过于稳定（$K_{MIn} \gg K_{MY}$），EDTA 无法将其从 MIn 中夺取，导致终点变色不灵敏甚至不变色。
解决：加入掩蔽剂除去干扰离子，或选用其他指示剂。
僵化现象：MIn 配合物与 EDTA 的置换反应速度太慢，导致终点拖后。
解决：加热滴定或加入有机溶剂。
指示剂选择原则：$pM'_t$（指示剂变色点对应的 pM'）应尽量接近 $pM'_{sp}$（化学计量点），即 $pM'_t \approx pM'_{sp}$，且落在滴定突跃范围内。

6.6 终点误差与准确滴定判据

直接滴定判据

准确滴定的充要条件
$$\boxed{\lg c_{sp} K'_{MY} \geq 6}$$

当 $\lg c_{sp} K'_{MY} \geq 6$ 时，终点误差 $E_t \leq \pm 0.1\%$，可进行准确滴定。

终点误差公式：

$$E_t = \frac{10^{\Delta pM'} - 10^{-\Delta pM'}}{c_{sp} \cdot K'_{MY}} \times 100\%$$

其中 $\Delta pM' = pM'_t - pM'_{sp}$。

例题：用 EDTA 滴定 $\text{Ca}^{2+}$，$c_{sp} = 0.01$ mol/L，用 EBT 指示剂在 pH = 10.0，$\lg\alpha_{Y(H)} = 0.45$，判断能否准确滴定。
解：

$\lg K'_{CaY} = \lg K_{CaY} - \lg\alpha_{Y(H)} = 10.7 - 0.45 = 10.25$

$\lg c_{sp} K'_{CaY} = \lg(0.01) + 10.25 = -2 + 10.25 = 8.25 \geq 6$

满足判据，可以准确滴定。

6.7 酸度控制

最低酸度（最高 pH）

受金属离子水解和氢氧化物沉淀限制。pH 过高时，金属离子会生成 $\text{M(OH)}_n$ 沉淀，导致滴定无法进行。

最高 pH 可从金属氢氧化物的 $K_{sp}$ 和金属浓度估算。

最高酸度（最低 pH）

受 $\alpha_{Y(H)}$ 限制。酸度太大会使 $\lg K'_{MY}$ 降低，不满足 $\lg c_{sp}K'_{MY} \geq 6$ 的要求。

从 $\lg K_{MY} - \lg\alpha_{Y(H)} \geq 6 - \lg c_{sp}$ 得到允许的最大 $\lg\alpha_{Y(H)}$，再查表得到最低 pH。

酸度控制的核心思路：最低 pH 由准确滴定判据决定（保证 $\lg c_{sp}K'_{MY} \geq 6$），最高 pH 由金属离子水解/沉淀决定。滴定 pH 应在两者之间。

6.8 混合离子的选择性滴定

分别滴定判据
$$\boxed{\Delta\lg cK' = \lg c_M K'_{MY} - \lg c_N K'_{NY} \geq 6}$$

当满足此条件时，可在同一溶液中先滴定 M 再滴定 N（或只滴定 M 不受 N 干扰）。

若 $\Delta\lg cK'$ 不够大，可通过以下方式提高选择性：

  • 控制酸度（在较低 pH 下只有高稳定常数的配合物形成）
  • 使用掩蔽剂降低干扰离子的有效浓度
  • 更换滴定剂（如对某些离子有更好选择性的配位剂）
例题：溶液中含 $\text{Fe}^{3+}$ 和 $\text{Zn}^{2+}$，各 0.01 mol/L，在 pH = 2.0 时（$\lg\alpha_{Y(H)} = 13.51$），能否选择性滴定 $\text{Fe}^{3+}$？
解：

$\lg K'_{FeY} = 25.1 - 13.51 = 11.59$，$\lg c \cdot K'_{FeY} = -2 + 11.59 = 9.59 \geq 6$ ✓

$\lg K'_{ZnY} = 16.5 - 13.51 = 2.99$，$\lg c \cdot K'_{ZnY} = -2 + 2.99 = 0.99 < 6$ ✗

$\Delta\lg cK' = 9.59 - 0.99 = 8.60 \geq 6$ ✓

在 pH = 2.0 时，可以选择性滴定 $\text{Fe}^{3+}$ 而不受 $\text{Zn}^{2+}$ 干扰。

掩蔽方法分类

  • 配位掩蔽：加入辅助配位剂使干扰离子形成更稳定的配合物（$\text{CN}^-$、$\text{F}^-$、TEA 等）
  • 沉淀掩蔽：使干扰离子生成沉淀而分离（如 NaOH 沉淀 $\text{Fe(OH)}_3$）
  • 氧化还原掩蔽：改变干扰离子的氧化态，使其不再干扰（如抗坏血酸还原 $\text{Fe}^{3+}$）
解蔽：某些掩蔽剂的作用可以解除。如用甲醛解除 $\text{CN}^-$ 对 $\text{Zn}^{2+}$ 的掩蔽，释放 $\text{Zn}^{2+}$ 进行滴定。

6.10 滴定方式

四种基本方式

1. 直接滴定法

用 EDTA 标准溶液直接滴定含金属离子的待测溶液。
条件：反应快速、定量，有合适指示剂，无干扰。

2. 返滴定法（回滴法）

先加入过量已知的 EDTA，再用另一种金属标准溶液（如 $\text{Zn}^{2+}$）回滴过量的 EDTA。

适用：反应慢（如 $\text{Al}^{3+}$, $\text{Cr}^{3+}$）、无合适指示剂、或金属离子在滴定 pH 下会水解沉淀。

3. 置换滴定法

先用另一金属的 EDTA 配合物（如 $\text{MgY}$）与待测金属反应：$\text{M} + \text{MgY} \rightarrow \text{MY} + \text{Mg}$，然后滴定释放出的 $\text{Mg}^{2+}$。

适用：待测金属无合适指示剂，但置换出的金属有合适指示剂。

4. 间接滴定法

对不能直接与 EDTA 配位的物质（如 $\text{SO}_4^{2-}$, $\text{PO}_4^{3-}$），先与已知过量的金属离子沉淀或反应，再滴定剩余金属离子。

例：测 $\text{SO}_4^{2-}$：加过量 $\text{Ba}^{2+}$ 沉淀 $\text{BaSO}_4$，再滴定剩余 $\text{Ba}^{2+}$。

综合例题：水的总硬度测定。取 100.0 mL 水样，加入 pH = 10 的 $\text{NH}_3$-$\text{NH}_4\text{Cl}$ 缓冲溶液，以 EBT 为指示剂，用 0.01000 mol/L EDTA 滴定至蓝色，消耗 15.20 mL。求水的总硬度（以 $\text{CaCO}_3$ 的 mg/L 表示）。
解：

$n_{EDTA} = 0.01000 \times 15.20 = 0.1520\;\text{mmol}$

$n_{Ca+Mg} = n_{EDTA} = 0.1520\;\text{mmol}$（1:1 配位）

总硬度 $= \frac{0.1520 \times 100.08}{100.0} \times 1000 = 152.1\;\text{mg/L}$（以 $\text{CaCO}_3$ 计）
