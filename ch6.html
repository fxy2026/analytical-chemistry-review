第六章 配位滴定法

6.1 EDTA的性质与配位特点

PPT: EDTA的结构与配位特点

EDTA的结构

EDTA（乙二胺四乙酸）是一种六元酸（$\text{H}_6\text{Y}^{2+}$），含有 2 个氨基氮和 4 个羧基氧，共提供 6 个配位原子。通常使用其二钠盐 $\text{Na}_2\text{H}_2\text{Y}\cdot 2\text{H}_2\text{O}$（$M = 372.24$）。

EDTA 的六个质子的酸性：前两个质子来自氨基 $\text{NH}^+$（$\text{p}K_{a1}=0.9$, $\text{p}K_{a2}=1.6$），后四个来自羧基（$\text{p}K_{a3}=2.07$, $\text{p}K_{a4}=2.75$, $\text{p}K_{a5}=6.24$, $\text{p}K_{a6}=10.34$）。

EDTA配合物的特点

  • 广泛性：能与几乎所有金属离子形成配合物
  • 1:1 配位：无论金属离子电荷或半径如何，均形成 $\text{MY}^{n-4}$ 型 1:1 配合物
  • 稳定性高：多数 $\lg K_{MY}$ 在 15 以上
  • 水溶性好：配合物均可溶于水
  • 反应速率快：配位反应速度一般较快

EDTA在不同 pH 下的存在形态

EDTA 是六元酸，在不同 pH 下以不同形态存在。由 $\text{p}K_a$ 值可以判断各 pH 区间的主要存在形态：

6.1b 逐级稳定常数与累积稳定常数

PPT: 逐级稳定常数与累积稳定常数

逐级稳定常数 $K_i$

对于金属离子 M 与配位体 L 的逐级配位反应：

$$M + L \rightleftharpoons ML \quad K_1 = \frac{[\text{ML}]}{[\text{M}][\text{L}]}$$ $$ML + L \rightleftharpoons ML_2 \quad K_2 = \frac{[\text{ML}_2]}{[\text{ML}][\text{L}]}$$ $$\vdots$$ $$ML_{n-1} + L \rightleftharpoons ML_n \quad K_n = \frac{[\text{ML}_n]}{[\text{ML}_{n-1}][\text{L}]}$$

$K_i$ 表示相邻配合物之间的平衡关系，一般 $K_1 > K_2 > \cdots > K_n$。

累积稳定常数 $\beta_n$

累积稳定常数的定义

$\beta_n$ 表示配合物 $ML_n$ 与游离 M、L 之间的总平衡关系：

$$\beta_n = K_1 \cdot K_2 \cdots K_n = \frac{[\text{ML}_n]}{[\text{M}][\text{L}]^n}$$

即 $\beta_1 = K_1$，$\beta_2 = K_1 K_2$，$\beta_n = \prod_{i=1}^{n}K_i$。

由此可得各级配合物浓度：$[\text{ML}_n] = \beta_n[\text{M}][\text{L}]^n$。

$K$ 与 $\beta$ 的区别：$K_i$ 描述相邻两级配合物的关系，$\beta_n$ 描述第 $n$ 级配合物与最初游离态的关系。考试中注意辨别题目给出的是 $K$ 还是 $\beta$。

各级配合物的分布分数 $\delta$

PPT: 各级配合物的分布

金属离子的分析总浓度 $c_M$ 为各种形态之和：

$$c_M = [\text{M}] + [\text{ML}] + [\text{ML}_2] + \cdots + [\text{ML}_n] = [\text{M}](1 + \beta_1[\text{L}] + \beta_2[\text{L}]^2 + \cdots + \beta_n[\text{L}]^n)$$

各级配合物的分布分数：

$$\delta_M = \frac{[\text{M}]}{c_M} = \frac{1}{1 + \beta_1[\text{L}] + \beta_2[\text{L}]^2 + \cdots + \beta_n[\text{L}]^n}$$ $$\delta_{ML_i} = \frac{[\text{ML}_i]}{c_M} = \frac{\beta_i[\text{L}]^i}{1 + \beta_1[\text{L}] + \beta_2[\text{L}]^2 + \cdots + \beta_n[\text{L}]^n} = \delta_M \cdot \beta_i[\text{L}]^i$$
与酸碱分布分数的类比：配合物的分布分数公式与多元酸的分布分数 $\delta$ 完全类似。酸碱平衡中用 $[\text{H}^+]$ 和 $K_a$ 计算分布，配位平衡中用 $[\text{L}]$ 和 $\beta_i$ 计算分布。

酸碱平衡与配位平衡的统一

PPT: 酸碱平衡与络合平衡的统一

质子化反应可视为以 $\text{H}^+$ 为"配位体"的配位反应。质子化常数 $K^H$ 与酸离解常数 $K_a$ 的关系：

$$K_1^H = \frac{1}{K_{a,n}}, \quad K_2^H = \frac{1}{K_{a,n-1}}, \quad \ldots$$

例如 $\text{NH}_3 + \text{H}^+ \rightleftharpoons \text{NH}_4^+$，其 $K^H_{\text{NH}_3} = \frac{1}{K_a} = \frac{K_b}{K_w} = 10^{9.37}$。

这种统一观点使得酸效应系数 $\alpha_{Y(H)}$ 的公式形式与金属副反应系数 $\alpha_{M(L)}$ 完全一致，只是"配位体"换成了 $\text{H}^+$，"稳定常数"换成了质子化常数 $\beta^H_i$。

6.2 副反应系数

PPT: 副反应系数的定义与分类

PPT: 副反应系数的计算

PPT: 条件稳定常数

EDTA的副反应系数

在配位滴定中，EDTA 参与的副反应主要有两类：

酸效应系数 $\alpha_{Y(H)}$

反映 H$^+$ 对 EDTA 配位能力的影响：

$$\alpha_{Y(H)} = \frac{[\text{Y}']}{[\text{Y}^{4-}]} = 1 + \frac{[\text{H}^+]}{K_{a6}} + \frac{[\text{H}^+]^2}{K_{a6}K_{a5}} + \cdots$$

酸度越大，$\alpha_{Y(H)}$ 越大，EDTA 配位能力越弱。

共存离子效应系数 $\alpha_{Y(N)}$

溶液中共存金属离子 N 与 EDTA 形成配合物，消耗 Y：

$$\alpha_{Y(N)} = 1 + K_{NY}[\text{N}]$$

若多种共存离子：$\alpha_Y = \alpha_{Y(H)} + \alpha_{Y(N_1)} + \alpha_{Y(N_2)} + \cdots - (k-1)$，其中 $k$ 为副反应总数。通常简化为 $\alpha_Y \approx \alpha_{Y(H)} \cdot \alpha_{Y(N)}$（当各项远大于 1 时）。

例题：手动计算 pH = 5 时的 $\alpha_{Y(H)}$
解：逐项计算法

已知 $\text{p}K_{a1}=0.9$, $\text{p}K_{a2}=1.6$, $\text{p}K_{a3}=2.07$, $\text{p}K_{a4}=2.75$, $\text{p}K_{a5}=6.24$, $\text{p}K_{a6}=10.34$。

pH = 5，即 $[\text{H}^+] = 10^{-5}$。

$$\alpha_{Y(H)} = 1 + \frac{[\text{H}^+]}{K_{a6}} + \frac{[\text{H}^+]^2}{K_{a6}K_{a5}} + \frac{[\text{H}^+]^3}{K_{a6}K_{a5}K_{a4}} + \cdots$$

逐项计算：

  • 第 1 项：$\frac{10^{-5}}{10^{-10.34}} = 10^{5.34} = 2.19 \times 10^5$
  • 第 2 项：$\frac{(10^{-5})^2}{10^{-10.34} \times 10^{-6.24}} = 10^{10-10+16.58} = 10^{6.58} = 3.80 \times 10^6$
  • 第 3 项：$10^{-15 + 10.34 + 6.24 + 2.75} = 10^{4.33}$（相对于第 2 项较小）

实际上第 2 项最大，$\alpha_{Y(H)} \approx 10^{6.58} + 10^{5.34} + \cdots$

取对数：$\lg\alpha_{Y(H)} \approx 6.6$（查表值一致）

实际计算时直接查表即可。只需在特殊情况（如小数 pH、或非标准温度）下才需手动计算。
例题：考虑多种副反应时的 $\alpha_Y$ 计算

溶液 pH = 10，含 $\text{Mg}^{2+}$（$c = 0.01$ mol/L）作为共存离子，$\lg K_{\text{MgY}} = 8.7$。求 $\alpha_Y$。

解：

pH = 10 时，$\lg\alpha_{Y(H)} = 0.5$，即 $\alpha_{Y(H)} = 10^{0.5} = 3.16$

$\alpha_{Y(N)} = 1 + K_{\text{MgY}}[\text{Mg}^{2+}] = 1 + 10^{8.7} \times 0.01 = 1 + 10^{6.7} \approx 10^{6.7}$

总副反应系数（严格公式）：

$$\alpha_Y = \alpha_{Y(H)} + \alpha_{Y(N)} - 1 = 3.16 + 10^{6.7} - 1 \approx 10^{6.7}$$

此时共存离子效应远大于酸效应，$\lg\alpha_Y \approx 6.7$。

注意：当某一项副反应远大于其他项时，$\alpha_Y$ 近似等于最大的那一项。但在考试中应写出完整的加法公式 $\alpha_Y = \alpha_{Y(H)} + \alpha_{Y(N)} - 1$，而不是乘法！乘法近似 $\alpha_Y \approx \alpha_{Y(H)} \cdot \alpha_{Y(N)}$ 仅在各项都远大于 1 时使用。

金属离子的副反应系数

辅助配位效应 $\alpha_{M(L)}$

金属离子 M 与辅助配位体 L（如 $\text{NH}_3$）反应：

$$\alpha_{M(L)} = 1 + \beta_1[\text{L}] + \beta_2[\text{L}]^2 + \cdots + \beta_n[\text{L}]^n$$
羟基效应 $\alpha_{M(OH)}$

金属离子水解导致参与配位反应的游离金属减少：

$$\alpha_{M(OH)} = 1 + \beta_1^{OH}\frac{K_w}{[\text{H}^+]} + \beta_2^{OH}\left(\frac{K_w}{[\text{H}^+]}\right)^2 + \cdots$$

pH 越高，$\alpha_{M(OH)}$ 越大。

注意：$\alpha_{M(OH)}$ 通常在 pH 较高时才显著，且对不同金属影响差异很大。某些金属离子（如 $\text{Fe}^{3+}$, $\text{Al}^{3+}$）在较低 pH 下即开始水解。
金属总副反应系数 $\alpha_M$

若同时存在辅助配位效应和羟基效应：

$$\alpha_M = \alpha_{M(L)} + \alpha_{M(OH)} - 1$$

当某一项远大于另一项时，取较大值近似。

例题（PPT）：$\text{Zn}^{2+}$ 在 $\text{NH}_3$ 缓冲液中的综合副反应系数

在 0.01000 mol/L $\text{Zn(NH}_3)_4^{2+}$ 溶液中，游离氨浓度为 0.10 mol/L，已知 pH = 10，$\lg\alpha_{\text{Zn(OH)}} = 2.4$，$\lg\alpha_{Y(H)} = 0.5$。$\text{Zn-NH}_3$ 的各级累积稳定常数：$\lg\beta_1 = 2.27$, $\lg\beta_2 = 4.61$, $\lg\beta_3 = 7.01$, $\lg\beta_4 = 9.06$。计算 pH = 10 时的 $\alpha_{\text{Zn}}$。

解：

第一步：计算 $\alpha_{\text{Zn(NH}_3)}$

$$\alpha_{\text{Zn(NH}_3)} = 1 + \beta_1[\text{NH}_3] + \beta_2[\text{NH}_3]^2 + \beta_3[\text{NH}_3]^3 + \beta_4[\text{NH}_3]^4$$ $$= 1 + 10^{2.27}\times 0.10 + 10^{4.61}\times 0.01 + 10^{7.01}\times 0.001 + 10^{9.06}\times 0.0001$$ $$= 1 + 10^{1.27} + 10^{2.61} + 10^{4.01} + 10^{5.06} \approx 10^{5.10}$$

第二步：已知 $\alpha_{\text{Zn(OH)}} = 10^{2.4}$

第三步：金属总副反应系数

$$\alpha_{\text{Zn}} = \alpha_{\text{Zn(NH}_3)} + \alpha_{\text{Zn(OH)}} - 1 = 10^{5.10} + 10^{2.4} - 1 \approx 10^{5.10}$$

（$\alpha_{\text{Zn(NH}_3)}$ 远大于 $\alpha_{\text{Zn(OH)}}$，后者可忽略）

第四步：条件稳定常数

$$\lg K'_{\text{ZnY}} = \lg K_{\text{ZnY}} - \lg\alpha_{\text{Zn}} - \lg\alpha_{Y(H)} = 16.50 - 5.10 - 0.5 = 10.9$$
此例强调：在 $\text{NH}_3$-$\text{NH}_4\text{Cl}$ 缓冲液中滴定 $\text{Zn}^{2+}$ 时，必须同时考虑 $\alpha_{\text{Zn(NH}_3)}$ 和 $\alpha_{\text{Zn(OH)}}$，不能只考虑 $\alpha_{Y(H)}$。但通常 $\alpha_{\text{Zn(NH}_3)}$ 占主导。
例题 (Quiz 2 Q4)：求 $\alpha_{Y(\text{HCN})}$

EDTA 溶液中含有 HCN（$K_a = 6.2 \times 10^{-10}$），HCN 对 Y 有副反应。已知 $[\text{HCN}] = 0.1$ mol/L，pH = 10.0。计算 $\alpha_{Y(\text{CN})}$。

解：

首先求 $[\text{CN}^-]$ 的浓度：

$$[\text{CN}^-] = \frac{K_a \cdot [\text{HCN}]}{[\text{H}^+]} = \frac{6.2\times 10^{-10} \times 0.1}{10^{-10}} = 0.62\;\text{mol/L}$$

实际上，pH = 10 时 HCN 基本解离（$K_a / [\text{H}^+] = 6.2$），所以大部分以 $\text{CN}^-$ 形式存在。

CN$^-$ 与金属的副反应系数：$\alpha_{M(\text{CN})} = 1 + \beta_1[\text{CN}^-] + \beta_2[\text{CN}^-]^2 + \cdots$

这道题考查的是对副反应系数概念的灵活运用。CN$^-$ 作为辅助配位剂，主要影响的是金属离子一侧的副反应（$\alpha_{M(\text{CN})}$），而不是 Y 一侧。审题时要注意区分副反应影响的是 M 还是 Y。

6.3 条件稳定常数

PPT: 逐级稳定常数与累积稳定常数

条件稳定常数的定义
$$K'_{MY} = \frac{[\text{MY}]}{[\text{M}'][\text{Y}']} = \frac{K_{MY}}{\alpha_M \cdot \alpha_Y}$$

取对数形式：

$$\boxed{\lg K'_{MY} = \lg K_{MY} - \lg\alpha_M - \lg\alpha_Y}$$

其中 $\lg\alpha_M = \lg\alpha_{M(L)} + \lg\alpha_{M(OH)}$（若两者都显著），$\lg\alpha_Y = \lg\alpha_{Y(H)} + \lg\alpha_{Y(N)}$（通常 $\alpha_{Y(H)}$ 占主导）。

条件稳定常数是在特定实验条件下（特定 pH、辅助配位体浓度等）的有效稳定常数，它综合反映了各种副反应对配位平衡的影响。
例题：在 pH = 5.0 时，$\lg\alpha_{Y(H)} = 6.6$，计算 $\text{Zn}^{2+}$-EDTA 的条件稳定常数（忽略其他副反应）。
解：

$\lg K_{ZnY} = 16.5$

$\lg K'_{ZnY} = \lg K_{ZnY} - \lg\alpha_{Y(H)} = 16.5 - 6.6 = 9.9$

条件稳定常数足够大，可以准确滴定。

例题：pH = 5 时能否准确滴定 $\text{Pb}^{2+}$？（$c = 0.01$ mol/L）
解：

查表：pH = 5 时，$\lg\alpha_{Y(H)} = 6.6$

$\lg K_{\text{PbY}} = 18.0$

$\lg K'_{\text{PbY}} = 18.0 - 6.6 = 11.4$

判据：$\lg c_{sp} K'_{\text{PbY}} = \lg(0.01) + 11.4 = -2 + 11.4 = 9.4$

$9.4 \geq 6$ ，满足准确滴定条件。

pH = 5 时可以准确滴定 $\text{Pb}^{2+}$，且突跃较大（$\lg cK' = 9.4$），终点敏锐。
例题：pH = 2 时能否准确滴定 $\text{Zn}^{2+}$？（$c = 0.01$ mol/L）
解：

查表：pH = 2 时，$\lg\alpha_{Y(H)} = 13.5$

$\lg K_{\text{ZnY}} = 16.5$

$\lg K'_{\text{ZnY}} = 16.5 - 13.5 = 3.0$

判据：$\lg c_{sp} K'_{\text{ZnY}} = -2 + 3.0 = 1.0$

$1.0 < 6$，不能准确滴定。

pH 过低时，EDTA 被大量质子化，有效配位能力极差。$\text{Zn}^{2+}$ 的 $\lg K$ 不够大，无法在 pH = 2 下滴定。需要将 pH 提高到至少 4~5 才可以。
例题：求 $\text{Zn}^{2+}$ 可被准确滴定的最低 pH（$c = 0.02$ mol/L）
解：

由 $\lg c_{sp}K'_{MY} \geq 6$ 得：

$$\lg K_{MY} - \lg\alpha_{Y(H)} + \lg c \geq 6$$ $$\lg\alpha_{Y(H)} \leq \lg K_{MY} + \lg c - 6 = 16.5 + \lg(0.02) - 6 = 16.5 - 1.7 - 6 = 8.8$$

查表：$\lg\alpha_{Y(H)} = 8.8$ 对应 pH 约在 3.5 ~ 4 之间。

因此 $\text{Zn}^{2+}$ 的最低滴定 pH 约为 3.5 ~ 4。

6.4 配位滴定曲线

PPT: EDTA配位滴定曲线

配位滴定曲线以 $pM'$（$pM' = -\lg[\text{M}']$）为纵轴，滴定剂加入体积（或滴定百分数）为横轴。

滴定曲线的关键点

  • 化学计量点： $$\boxed{pM'_{sp} = \frac{1}{2}(\lg K'_{MY} + pc_{sp})}$$ 其中 $pc_{sp} = -\lg c_{sp}$，$c_{sp}$ 为化学计量点时 MY 的浓度。
  • 滴定突跃：$\lg c_{sp} K'_{MY}$ 越大，突跃越大。
  • 化学计量点前（如 50%）：$pM' = -\lg\frac{c_M}{2}$（简单稀释计算）
  • 化学计量点后（如 200%）：根据过量 Y 的浓度计算
突跃范围 $\Delta pM'$ 的估算公式

PPT: 滴定曲线与突跃范围

突跃范围（化学计量点前后 $\pm 0.1\%$ 所对应的 $pM'$ 变化）的近似公式：

$$\Delta pM' = (-\lg c_{sp,M} + 3) - (\lg K'_{MY} - 3)$$

即突跃下限 $pM'_{99.9\%} \approx -\lg c_{sp,M} + 3 = pc_{sp} + 3$，突跃上限 $pM'_{100.1\%} \approx \lg K'_{MY} - 3$。

影响突跃大小的因素

PPT: 影响pM突跃的主要因素

  • $K'_{MY}$ 越大，突跃越大（$K'$ 增大 10 倍，$pM'$ 突跃增大约 1 个单位）
  • 浓度 $c$ 越大，突跃前半段（化学计量点前）$pM'$ 起始值越小，突跃越大
  • 化学计量点后的突跃还与酸度有关：$pM'_{后} = \lg K_{MY} - \lg\alpha_{Y(H)}$ 左右

滴定曲线各段计算公式

化学计量点前（滴定分数 $\phi < 1$）

此时金属离子过量，溶液中 M' 浓度由未反应的金属离子决定：

$$[\text{M}'] = c_M \cdot \frac{1-\phi}{1+\phi/(V_0/V_0)} \approx c_M(1-\phi) \cdot \frac{V_0}{V_0+V}$$

当 $\phi = 0.5$（即加入一半 EDTA）时，约一半金属被配位：$[\text{M}'] \approx c_M / 2$（稀释因子近似忽略或精确考虑）。

化学计量点后（滴定分数 $\phi > 1$）

此时 EDTA 过量：

$$[\text{Y}'] = c_{EDTA} \cdot \frac{(\phi - 1)V_{eq}}{V_0 + V}$$ $$[\text{M}'] = \frac{[\text{MY}]}{K'_{MY} \cdot [\text{Y}']}$$
例题：用 0.020 mol/L EDTA 滴定 20.00 mL 0.020 mol/L $\text{Zn}^{2+}$，pH = 5.0，$\lg K'_{ZnY} = 10.05$，求化学计量点的 $pZn'$。
解：

$c_{sp} = \frac{0.020 \times 20.00}{40.00} = 0.010\;\text{mol/L}$，故 $pc_{sp} = 2.00$

$pZn'_{sp} = \frac{1}{2}(10.05 + 2.00) = 6.03$

例题：接上题，计算滴定 50%、99.9%、100.1%、200% 时的 $pZn'$
解：

50% 处 ($\phi = 0.5$)：

$[\text{Zn}'] = \frac{0.020 \times 20.00 - 0.020 \times 10.00}{30.00} = \frac{0.200}{30.00} = 6.67 \times 10^{-3}$

$pZn' = -\lg(6.67 \times 10^{-3}) = 2.18$

99.9% 处 ($\phi = 0.999$)：

$[\text{Zn}'] = \frac{0.020 \times (1-0.999) \times 20.00}{39.98} = \frac{0.020 \times 0.001 \times 20.00}{39.98} = 1.0 \times 10^{-5}$

$pZn' = 5.0$

100.1% 处 ($\phi = 1.001$)：

$[\text{Y}'] = \frac{0.020 \times 0.001 \times 20.00}{40.02} = 1.0 \times 10^{-5}$

$[\text{Zn}'] = \frac{c_{sp}}{K'_{ZnY} \cdot [\text{Y}']} = \frac{0.010}{10^{10.05} \times 10^{-5}} = \frac{0.010}{10^{5.05}} = 10^{-7.05}$

$pZn' = 7.05$

200% 处 ($\phi = 2$)：

$[\text{Y}'] = \frac{0.020 \times 20.00}{60.00} = 6.67 \times 10^{-3}$

$[\text{Zn}'] = \frac{0.010}{10^{10.05} \times 6.67 \times 10^{-3}} = \frac{0.010}{10^{7.87}} = 10^{-9.87}$

$pZn' = 9.87$

滴定突跃范围：$pZn' = 5.0 \sim 7.1$（99.9% ~ 100.1%），突跃 $\Delta pZn' \approx 2.1$。

6.5 金属指示剂

PPT: 金属指示剂的作用原理

原理

金属指示剂（In）本身是一种有机配位剂，在游离态（$\text{In}$）和与金属配合态（$\text{MIn}$）颜色不同。

滴定终点：$\text{MIn} + \text{Y} \rightarrow \text{MY} + \text{In}$（颜色由 MIn 的颜色变为 In 的颜色）。

EBT（铬黑 T）与 $\text{Mg}^{2+}$ 的变色机理

详细变色过程

EBT 是一种偶氮染料，含有酚羟基和偶氮基作为配位基团。在 pH 8~11 范围内：

  • 游离态 $\text{In}^{3-}$（$\text{HIn}^{2-}$）呈蓝色
  • 与 $\text{Mg}^{2+}$ 配位形成 $\text{MgIn}^-$ 呈紫红（酒红）色

滴定过程：

  1. 滴定前：溶液含 $\text{Mg}^{2+}$，加入少量 EBT，部分 $\text{Mg}^{2+}$ 与 EBT 结合为 $\text{MgIn}^-$（紫红色），溶液呈紫红色。
  2. 滴定中：EDTA 优先与游离 $\text{Mg}^{2+}$ 反应（因为 $K_{\text{MgY}} \gg K_{\text{MgIn}}$），溶液仍为紫红色。
  3. 化学计量点附近：游离 $\text{Mg}^{2+}$ 基本耗尽，EDTA 开始夺取 $\text{MgIn}^-$ 中的 $\text{Mg}^{2+}$：
$$\text{MgIn}^-(\text{紫红}) + \text{Y}^{4-} \rightarrow \text{MgY}^{2-} + \text{In}^{3-}(\text{蓝})$$

溶液从紫红色变为纯蓝色，即为终点。

指示剂变色点 $pM_t$ 的计算

$pM_t$ 的理论公式

指示剂的变色点定义为 $[\text{MIn}] = [\text{In}']$ 时对应的 $pM'$ 值：

$$\boxed{pM_t = \lg K_{\text{MIn}} - \lg\alpha_{\text{In}(H)}}$$

其中 $K_{\text{MIn}}$ 是金属-指示剂配合物的稳定常数，$\alpha_{\text{In}(H)}$ 是指示剂的酸效应系数。

当 $[\text{MIn}] / [\text{In}'] = 1$ 时，恰好是颜色转变的中间点。实际观察到的变色范围为 $pM_t \pm 1$。

指示剂选择原则：选择 $pM_t$ 落在滴定突跃范围内的指示剂，且 $pM_t$ 尽量接近 $pM'_{sp}$。若 $pM_t$ 偏离 $pM'_{sp}$ 太多，终点误差会增大。
例题：EBT 指示剂测定 $\text{Zn}^{2+}$，pH = 10 时的 $pM_t$
解：

已知：$\lg K_{\text{ZnIn}} = 12.9$（EBT 与 Zn 的稳定常数），pH = 10 时 $\lg\alpha_{\text{In}(H)} = 5.0$。

$$pM_t = \lg K_{\text{ZnIn}} - \lg\alpha_{\text{In}(H)} = 12.9 - 5.0 = 7.9$$

而 pH = 10 时，$\lg\alpha_{Y(H)} = 0.5$，$\lg K'_{\text{ZnY}} = 16.5 - 0.5 = 16.0$

若 $c = 0.01$ mol/L，$pZn'_{sp} = \frac{1}{2}(16.0 + 2.0) = 9.0$

$pM_t = 7.9$ 落在突跃范围内，EBT 可以用于指示 $\text{Zn}^{2+}$ 的滴定终点。

但 $pM_t (7.9) < pM'_{sp} (9.0)$，终点偏早，滴定结果略偏低。

指示剂使用中的问题

封闭现象（僵化）：某些金属（如 $\text{Al}^{3+}$, $\text{Fe}^{3+}$, $\text{Ni}^{2+}$, $\text{Co}^{2+}$）与指示剂形成的配合物过于稳定（$K_{MIn} \gg K_{MY}$），EDTA 无法将其从 MIn 中夺取，导致终点变色不灵敏甚至不变色。
解决：加入掩蔽剂除去干扰离子，或选用其他指示剂。
僵化现象：MIn 配合物与 EDTA 的置换反应速度太慢，导致终点拖后。
解决：加热滴定或加入有机溶剂。
指示剂选择原则：$pM'_t$（指示剂变色点对应的 pM'）应尽量接近 $pM'_{sp}$（化学计量点），即 $pM'_t \approx pM'_{sp}$，且落在滴定突跃范围内。

6.6 终点误差与准确滴定判据

直接滴定判据

准确滴定的充要条件
$$\boxed{\lg c_{sp} K'_{MY} \geq 6}$$

当 $\lg c_{sp} K'_{MY} \geq 6$ 时，终点误差 $E_t \leq \pm 0.1\%$，可进行准确滴定。

终点误差公式（林邦误差公式）

林邦公式

终点误差 $E_t$ 定量反映了指示剂变色点 $pM'_t$ 与化学计量点 $pM'_{sp}$ 偏差对滴定结果的影响：

$$\boxed{E_t = \frac{10^{\Delta pM'} - 10^{-\Delta pM'}}{c_{sp} \cdot K'_{MY}} \times 100\%}$$

其中 $\Delta pM' = pM'_t - pM'_{sp}$。

公式分析：

  • 当 $\Delta pM' > 0$（$pM'_t > pM'_{sp}$）：终点偏后（过滴定），$E_t > 0$
  • 当 $\Delta pM' < 0$（$pM'_t < pM'_{sp}$）：终点偏前（欠滴定），$E_t < 0$
  • 当 $\Delta pM' = 0$：$E_t = 0$，无终点误差
  • $c_{sp}K'_{MY}$ 越大（分母越大），$E_t$ 越小 -- 这就是为什么 $\lg cK' \geq 6$ 是准确滴定的条件
林邦公式的核心含义：终点误差的大小取决于两个因素：(1) $\Delta pM'$ -- 指示剂变色点与计量点的偏差；(2) $c_{sp}K'_{MY}$ -- 反映滴定突跃的大小。突跃越大，即使 $\Delta pM'$ 不为零，误差也可以很小。
例题：计算 $\Delta pM' = 0.5$ 时的终点误差

用 EDTA 滴定 $\text{Zn}^{2+}$，$c_{sp} = 0.010$ mol/L，pH = 10，$\lg K'_{\text{ZnY}} = 16.0$。指示剂变色点使得 $pM'_t - pM'_{sp} = +0.5$。

解：

$\Delta pM' = +0.5$

$c_{sp} \cdot K'_{MY} = 0.010 \times 10^{16.0} = 10^{14.0}$

$$E_t = \frac{10^{0.5} - 10^{-0.5}}{10^{14.0}} \times 100\% = \frac{3.16 - 0.316}{10^{14}} \times 100\% = \frac{2.84}{10^{14}} \times 100\%$$

$E_t \approx 2.84 \times 10^{-12}\;\%$，终点误差极其微小。

当 $\lg cK'$ 很大（如 14）时，即使 $\Delta pM'$ 有偏差，终点误差也可忽略。实际问题中，$\lg cK'$ 在 6~8 附近时终点误差才需要认真考虑。
例题：$\lg cK'$ 较小时的终点误差

滴定 $\text{Ca}^{2+}$，$c_{sp} = 0.010$ mol/L，pH = 10，$\lg\alpha_{Y(H)} = 0.5$，$\lg K'_{\text{CaY}} = 10.7 - 0.5 = 10.2$。指示剂 $\Delta pM' = -1.0$。

解：

$c_{sp}K' = 10^{-2} \times 10^{10.2} = 10^{8.2}$

$$E_t = \frac{10^{-1.0} - 10^{1.0}}{10^{8.2}} \times 100\% = \frac{0.1 - 10}{10^{8.2}} \times 100\% = \frac{-9.9}{10^{8.2}} \times 100\%$$

$E_t \approx -6.25 \times 10^{-6}\%$，仍然非常小。

$\Delta pM' < 0$ 说明终点偏前（欠滴定），$E_t$ 为负值。

准确滴定判据小结

判据公式含义
直接滴定$\lg c_{sp}K'_{MY} \geq 6$保证 $|E_t| \leq 0.1\%$
分别滴定$\Delta\lg cK' \geq 6$两种金属可分步滴定
最低 pH$\lg\alpha_{Y(H)} \leq \lg K_{MY} + \lg c - 6$由判据推出的酸度下限
例题：用 EDTA 滴定 $\text{Ca}^{2+}$，$c_{sp} = 0.01$ mol/L，用 EBT 指示剂在 pH = 10.0，$\lg\alpha_{Y(H)} = 0.45$，判断能否准确滴定。
解：

$\lg K'_{CaY} = \lg K_{CaY} - \lg\alpha_{Y(H)} = 10.7 - 0.45 = 10.25$

$\lg c_{sp} K'_{CaY} = \lg(0.01) + 10.25 = -2 + 10.25 = 8.25 \geq 6$

满足判据，可以准确滴定。

6.7 酸度控制

最低酸度（最高 pH）

受金属离子水解和氢氧化物沉淀限制。pH 过高时，金属离子会生成 $\text{M(OH)}_n$ 沉淀，导致滴定无法进行。

最高 pH 可从金属氢氧化物的 $K_{sp}$ 和金属浓度估算。

最高酸度（最低 pH）

受 $\alpha_{Y(H)}$ 限制。酸度太大会使 $\lg K'_{MY}$ 降低，不满足 $\lg c_{sp}K'_{MY} \geq 6$ 的要求。

从 $\lg K_{MY} - \lg\alpha_{Y(H)} \geq 6 - \lg c_{sp}$ 得到允许的最大 $\lg\alpha_{Y(H)}$，再查表得到最低 pH。

酸度控制的核心思路：最低 pH 由准确滴定判据决定（保证 $\lg c_{sp}K'_{MY} \geq 6$），最高 pH 由金属离子水解/沉淀决定。滴定 pH 应在两者之间。

3~5$\text{Pb}^{2+}$, $\text{Cu}^{2+}$, $\text{Ni}^{2+}$, $\text{Zn}^{2+}$, $\text{Al}^{3+}$HAc-NaAc (pH 4~6)

最佳 pH 值的确定

PPT: 最佳pH值的获取方法

最佳 pH 值是使 $pM'_{sp}$ 与 $pM'_t$（指示剂变色点）恰好相等（即 $\Delta pM' = 0$、$E_t = 0$）的 pH 值。具体求法：

  1. 建立 $pM'_{sp}$ - pH 关系曲线：$pM'_{sp} = \frac{1}{2}(\lg K'_{MY} + pc_{sp})$，其中 $\lg K'_{MY}$ 随 pH 变化（通过 $\alpha_{Y(H)}$、$\alpha_{M(OH)}$）
  2. 建立 $pM'_t$ - pH 关系曲线：$pM_t = \lg K_{MIn} - \lg\alpha_{In(H)}$
  3. 两条曲线的交点对应的 pH 即为最佳 pH
在最佳 pH 下，虽然 $\Delta pM' = 0$，但由于指示剂变色范围和人眼辨色能力的限制，实际仍会有约 $\pm 0.2$ 的 $\Delta pM'$ 不确定性。

缓冲溶液的作用（重要！）

PPT: 配位滴定中缓冲溶液的作用

EDTA 与金属离子反应时会释放 $\text{H}^+$（因为 EDTA 以 $\text{H}_2\text{Y}^{2-}$ 等质子化形态存在）：

$$M^{n+} + H_2Y^{2-} \rightarrow MY^{(n-4)} + 2H^+$$

这导致溶液 pH 随滴定进行而降低，引起以下问题：

  • $\alpha_{Y(H)}$ 增大 $\rightarrow$ $K'_{MY}$ 降低 $\rightarrow$ 终点误差增大
  • $K'_{MIn}$ 也发生变化 $\rightarrow$ $pM'_t$ 改变

解决方案：加入缓冲溶液维持 pH 恒定。

常用缓冲体系：
  • pH 5~6：醋酸缓冲液或六次甲基四胺
  • pH 10 左右：$\text{NH}_3$-$\text{NH}_4\text{Cl}$ 碱性缓冲液
缓冲溶液的选择不仅要考虑 pH 值，还需有足够的缓冲容量来应对滴定过程中释放的 $\text{H}^+$。
例题（PPT）：$\text{Mn}^{2+}$ 滴定的适宜 pH 范围

用 0.020 mol/L EDTA 滴定等浓度 $\text{Mn}^{2+}$，已知 $\lg K_{\text{MnY}} = 13.87$，$pK_{sp,\text{Mn(OH)}_2} = 12.72$，若允许 $\Delta pM' = 0.2$，$E_t = 0.1\%$，求适宜的 pH 范围。

解：

最低 pH（最高酸度）：由 $E_t \leq 0.1\%$ 和 $\lg cK'_{MY} \geq 6$ 求得

将 $\Delta pM' = 0.2$ 代入林邦公式，$K'_{MY} = 10^{7.96}$

$\lg K'_{MY} = \lg K_{MY} - \lg\alpha_{Y(H)} = 7.96$

$\lg\alpha_{Y(H)} = 13.87 - 7.96 = 5.91$

查表：$\lg\alpha_{Y(H)} = 5.91$ 对应 pH $\approx$ 5.3

最高 pH（最低酸度）：由 $\text{Mn(OH)}_2$ 沉淀条件求得

$$pH_{max} = 14 - \frac{1}{2}(pK_{sp} + \lg c_{sp,M}) = 14 - \frac{1}{2}(12.72 + \lg 0.01) = 8.49$$

适宜 pH 范围：5.3 ~ 8.5

6.8 混合离子的选择性滴定

PPT: 提高配位滴定选择性的方法

分别滴定判据
$$\boxed{\Delta\lg cK' = \lg c_M K'_{MY} - \lg c_N K'_{NY} \geq 6}$$

当满足此条件时，可在同一溶液中先滴定 M 再滴定 N（或只滴定 M 不受 N 干扰）。

若 $\Delta\lg cK'$ 不够大，可通过以下方式提高选择性：

  • 控制酸度（在较低 pH 下只有高稳定常数的配合物形成）
  • 使用掩蔽剂降低干扰离子的有效浓度
  • 更换滴定剂（如对某些离子有更好选择性的配位剂）
例题：溶液中含 $\text{Fe}^{3+}$ 和 $\text{Zn}^{2+}$，各 0.01 mol/L，在 pH = 2.0 时（$\lg\alpha_{Y(H)} = 13.51$），能否选择性滴定 $\text{Fe}^{3+}$？
解：

$\lg K'_{FeY} = 25.1 - 13.51 = 11.59$，$\lg c \cdot K'_{FeY} = -2 + 11.59 = 9.59 \geq 6$ ✓

$\lg K'_{ZnY} = 16.5 - 13.51 = 2.99$，$\lg c \cdot K'_{ZnY} = -2 + 2.99 = 0.99 < 6$ ✗

$\Delta\lg cK' = 9.59 - 0.99 = 8.60 \geq 6$ ✓

在 pH = 2.0 时，可以选择性滴定 $\text{Fe}^{3+}$ 而不受 $\text{Zn}^{2+}$ 干扰。

例题 (2022 Final Q13 改编)：$\text{Bi}^{3+}$ 和 $\text{Fe}^{3+}$ 能否分别滴定？

溶液含 $\text{Bi}^{3+}$ 和 $\text{Fe}^{3+}$，各 0.01 mol/L。$\lg K_{\text{BiY}} = 27.9$，$\lg K_{\text{FeY}} = 25.1$。

解：

两者的 $\lg K$ 差值：$\Delta\lg K = 27.9 - 25.1 = 2.8$

浓度相同，因此 $\Delta\lg cK' = \Delta\lg K' = \Delta\lg K = 2.8 < 6$

不能通过单纯控制酸度分别滴定 $\text{Bi}^{3+}$ 和 $\text{Fe}^{3+}$。

解决方案：需要使用掩蔽剂。例如加入抗坏血酸将 $\text{Fe}^{3+}$ 还原为 $\text{Fe}^{2+}$（$\lg K_{\text{Fe(II)Y}} = 14.3$），此时 $\Delta\lg K = 27.9 - 14.3 = 13.6 \gg 6$，可以在酸性条件下选择性滴定 $\text{Bi}^{3+}$。

  • 配位掩蔽：加入辅助配位剂使干扰离子形成更稳定的配合物（$\text{CN}^-$、$\text{F}^-$、TEA 等）
  • 沉淀掩蔽：使干扰离子生成沉淀而分离（如 NaOH 沉淀 $\text{Fe(OH)}_3$）

$\text{NH}_4\text{F}$ 掩蔽 $\text{Al}^{3+}$ 的定量分析

Quiz 2 考点：$\text{F}^-$ 对 $\text{Al}^{3+}$ 的掩蔽效果

$\text{Al}^{3+}$ 与 $\text{F}^-$ 形成极稳定的 $\text{AlF}_6^{3-}$ 配合物。在足量 $\text{F}^-$ 存在下：

$$\alpha_{\text{Al}(F)} = 1 + \beta_1[\text{F}^-] + \beta_2[\text{F}^-]^2 + \cdots + \beta_6[\text{F}^-]^6$$

典型条件下 $\lg\alpha_{\text{Al}(F)} \approx 10$，即 $\alpha_{\text{Al}(F)} \approx 10^{10}$。

此时 $\text{Al}^{3+}$ 的条件稳定常数：

$$\lg K'_{\text{AlY}} = \lg K_{\text{AlY}} - \lg\alpha_{\text{Al}(F)} - \lg\alpha_{Y(H)} = 16.1 - 10 - \lg\alpha_{Y(H)}$$

即使 pH 较高（$\lg\alpha_{Y(H)}$ 很小），$\lg K'_{\text{AlY}}$ 也只有 ~6，远不满足准确滴定条件。$\text{Al}^{3+}$ 被完全掩蔽。

6.10 滴定方式

四种基本方式

1. 直接滴定法

用 EDTA 标准溶液直接滴定含金属离子的待测溶液。
条件：反应快速、定量，有合适指示剂，无干扰。

2. 返滴定法（回滴法）

先加入过量已知的 EDTA，再用另一种金属标准溶液（如 $\text{Zn}^{2+}$）回滴过量的 EDTA。

适用：反应慢（如 $\text{Al}^{3+}$, $\text{Cr}^{3+}$）、无合适指示剂、或金属离子在滴定 pH 下会水解沉淀。

3. 置换滴定法

先用另一金属的 EDTA 配合物（如 $\text{MgY}$）与待测金属反应：$\text{M} + \text{MgY} \rightarrow \text{MY} + \text{Mg}$，然后滴定释放出的 $\text{Mg}^{2+}$。

适用：待测金属无合适指示剂，但置换出的金属有合适指示剂。

4. 间接滴定法

对不能直接与 EDTA 配位的物质（如 $\text{SO}_4^{2-}$, $\text{PO}_4^{3-}$），先与已知过量的金属离子沉淀或反应，再滴定剩余金属离子。

例：测 $\text{SO}_4^{2-}$：加过量 $\text{Ba}^{2+}$ 沉淀 $\text{BaSO}_4$，再滴定剩余 $\text{Ba}^{2+}$。

6.X 配位滴定的应用实例

应用一：水的总硬度测定

实验方案

原理：水中 $\text{Ca}^{2+}$ 和 $\text{Mg}^{2+}$ 的总量即为水的总硬度，以 $\text{CaCO}_3$ 的 mg/L 表示。在 pH = 10 的 $\text{NH}_3$-$\text{NH}_4\text{Cl}$ 缓冲溶液中，用 EDTA 标准溶液同时滴定 $\text{Ca}^{2+}$ 和 $\text{Mg}^{2+}$，以 EBT 为指示剂。

操作步骤：

  1. 移取 100.0 mL 水样于锥形瓶中
  2. 加入 5 mL $\text{NH}_3$-$\text{NH}_4\text{Cl}$ 缓冲溶液（pH = 10）
  3. 加入少量 EBT 指示剂（溶液呈紫红色）
  4. 用 $c$ mol/L EDTA 标准溶液滴定至溶液由紫红色变为纯蓝色，记录消耗体积 $V$ mL

计算公式：

$$\text{总硬度 (mg/L)} = \frac{c \times V \times M_{\text{CaCO}_3}}{V_{\text{水样}}} \times 1000 = \frac{c \times V \times 100.08}{V_{\text{水样}}/1000}$$
为什么用 EBT 而不用 NN？EBT 在 pH = 10 可同时指示 $\text{Ca}^{2+}$ 和 $\text{Mg}^{2+}$ 的滴定终点。但如果水样中 $\text{Mg}^{2+}$ 含量很低，EBT 变色不灵敏（因为 $K_{\text{MgIn}} > K_{\text{CaIn}}$，EBT 实际上主要依赖 $\text{Mg}^{2+}$ 的颜色变化）。此时可在 EDTA 标准溶液中预先加入少量 $\text{MgY}$，使滴定终点有 $\text{Mg}^{2+}$ 参与指示。
例题：水的总硬度测定计算

取 100.0 mL 水样，加入 pH = 10 的 $\text{NH}_3$-$\text{NH}_4\text{Cl}$ 缓冲溶液，以 EBT 为指示剂，用 0.01000 mol/L EDTA 滴定至蓝色，消耗 15.20 mL。求水的总硬度（以 $\text{CaCO}_3$ 的 mg/L 表示）。

解：

$n_{\text{EDTA}} = 0.01000 \times 15.20 = 0.1520\;\text{mmol}$

$n_{\text{Ca+Mg}} = n_{\text{EDTA}} = 0.1520\;\text{mmol}$（1:1 配位）

总硬度 $= \frac{0.1520 \times 100.08}{100.0} \times 1000 = 152.1\;\text{mg/L}$（以 $\text{CaCO}_3$ 计）

例题（进阶）：分别测 $\text{Ca}^{2+}$ 和 $\text{Mg}^{2+}$

同一水样 100.0 mL：(1) 在 pH = 10 用 EBT 指示剂，消耗 0.01000 mol/L EDTA 15.20 mL（测总硬度）；(2) 另取 100.0 mL 在 pH = 12~13 用钙指示剂 NN，消耗同浓度 EDTA 9.60 mL（仅测 $\text{Ca}^{2+}$）。求 $\text{Ca}^{2+}$ 和 $\text{Mg}^{2+}$ 各自的含量。

解：

$\text{Ca}^{2+}$ 含量：

$n_{\text{Ca}} = 0.01000 \times 9.60 = 0.0960\;\text{mmol}$

$[\text{Ca}^{2+}] = \frac{0.0960}{100.0} \times 1000 = 0.960\;\text{mmol/L}$

以 $\text{CaCO}_3$ 计：$0.960 \times 100.08 = 96.1\;\text{mg/L}$

$\text{Mg}^{2+}$ 含量：

$n_{\text{Mg}} = n_{\text{total}} - n_{\text{Ca}} = 0.1520 - 0.0960 = 0.0560\;\text{mmol}$

$[\text{Mg}^{2+}] = \frac{0.0560}{100.0} \times 1000 = 0.560\;\text{mmol/L}$

以 $\text{CaCO}_3$ 计：$0.560 \times 100.08 = 56.0\;\text{mg/L}$

为什么 pH = 12~13 只测 $\text{Ca}^{2+}$？在强碱性条件下，$\text{Mg}^{2+}$ 以 $\text{Mg(OH)}_2$ 沉淀析出，不参与配位反应。钙指示剂 NN 仅对 $\text{Ca}^{2+}$ 显色。

应用二：$\text{Al}^{3+}$ 的返滴定法测定

实验方案

问题：$\text{Al}^{3+}$ 与 EDTA 的反应速度很慢，且在 pH 4~5 下有水解倾向，不能直接滴定。

解决方案（返滴定法）：

  1. 向含 $\text{Al}^{3+}$ 的溶液中加入已知过量的 EDTA 标准溶液
  2. 在 pH = 3~4 的酸性条件下加热煮沸（促进 $\text{Al}^{3+}$ 与 EDTA 充分反应）
  3. 冷却后，调节 pH 至 5~6（HAc-NaAc 缓冲液）
  4. 以 XO（二甲酚橙）为指示剂，用 $\text{Zn}^{2+}$（或 $\text{Cu}^{2+}$）标准溶液回滴过量的 EDTA，至溶液由黄色变为紫红色

计算：

$$n_{\text{Al}} = n_{\text{EDTA(加入)}} - n_{\text{Zn(回滴)}}$$
Quiz 2 Q1 考点：返滴定法（back titration）的概念。关键要素：(1) 加入过量 EDTA；(2) 加热促进慢反应完全；(3) 用另一种金属标准溶液回滴剩余 EDTA。

应用三：$\text{Bi}^{3+}$/$\text{Pb}^{2+}$ 连续滴定

利用酸度控制的分步滴定

溶液中同时含 $\text{Bi}^{3+}$ 和 $\text{Pb}^{2+}$。$\lg K_{\text{BiY}} = 27.9$，$\lg K_{\text{PbY}} = 18.0$，差值 $\Delta\lg K = 9.9$。

步骤：

  1. 第一步：pH = 1（$\lg\alpha_{Y(H)} \approx 18$）
    • $\lg K'_{\text{BiY}} = 27.9 - 18 = 9.9$，$\lg cK' \approx 7.9 \geq 6$ ✓
    • $\lg K'_{\text{PbY}} = 18.0 - 18 = 0$，$\lg cK' \approx -2 \ll 6$ ✗
    • 以 XO 为指示剂，用 EDTA 滴定 $\text{Bi}^{3+}$，$\text{Pb}^{2+}$ 不干扰。记录消耗 $V_1$ mL
  2. 第二步：向同一溶液中加入六次甲基四胺调节 pH 至 5~6
    • 此时 $\text{Pb}^{2+}$ 可被滴定（$\lg K'_{\text{PbY}} = 18.0 - 6.6 = 11.4$）
    • 继续用 EDTA 滴定 $\text{Pb}^{2+}$，记录消耗 $V_2$ mL

$n_{\text{Bi}} = c \times V_1$，$n_{\text{Pb}} = c \times V_2$

例题 (2022 Final Q5 改编)：$\text{Pb}^{2+}$/$\text{Bi}^{3+}$ 混合溶液的 EDTA 滴定

取合金溶解后的试液 25.00 mL（含 $\text{Bi}^{3+}$ 和 $\text{Pb}^{2+}$），在 pH = 1 下用 0.02000 mol/L EDTA 滴定 $\text{Bi}^{3+}$，消耗 12.50 mL。然后调 pH = 5.5，继续滴定 $\text{Pb}^{2+}$，又消耗 18.00 mL。求溶液中 $\text{Bi}^{3+}$ 和 $\text{Pb}^{2+}$ 的浓度。

解：

$\text{Bi}^{3+}$：

$n_{\text{Bi}} = 0.02000 \times 12.50 = 0.2500\;\text{mmol}$

$[\text{Bi}^{3+}] = \frac{0.2500}{25.00} = 0.01000\;\text{mol/L}$

$\text{Pb}^{2+}$：

$n_{\text{Pb}} = 0.02000 \times 18.00 = 0.3600\;\text{mmol}$

$[\text{Pb}^{2+}] = \frac{0.3600}{25.00} = 0.01440\;\text{mol/L}$

应用四：$\text{Zn}^{2+}$/$\text{Al}^{3+}$ 混合体系 -- $\text{NH}_4\text{F}$ 掩蔽法

Quiz 2 考点方案

$\lg K_{\text{ZnY}} = 16.5$，$\lg K_{\text{AlY}} = 16.1$，两者 $\lg K$ 非常接近，无法通过酸度控制分别滴定。

方案：加入 $\text{NH}_4\text{F}$，利用 $\text{F}^-$ 对 $\text{Al}^{3+}$ 的强掩蔽作用（$\lg\alpha_{\text{Al}(F)} \approx 10$）。

  1. 在 pH 5~6 的 HAc-NaAc 缓冲液中，加入 $\text{NH}_4\text{F}$
  2. $\text{Al}^{3+}$ 被完全掩蔽为 $\text{AlF}_6^{3-}$，不与 EDTA 反应
  3. 用 EDTA 滴定 $\text{Zn}^{2+}$，XO 为指示剂（黄 $\rightarrow$ 紫红为终点前，紫红 $\rightarrow$ 黄为终点）
  4. $\text{F}^-$ 对 $\text{Zn}^{2+}$ 的掩蔽作用很弱（$\lg\alpha_{\text{Zn}(F)} \approx 1$），不影响 $\text{Zn}^{2+}$ 的滴定

应用五：$\text{Cu}^{2+}$/$\text{Pb}^{2+}$/$\text{Zn}^{2+}$ 合金分析 -- KCN 掩蔽 + 甲醛解蔽

三步分析方案

合金溶解后含 $\text{Cu}^{2+}$、$\text{Pb}^{2+}$、$\text{Zn}^{2+}$。

  1. 第一步：直接滴定总量

    在 pH = 5~6 下，用 EDTA 滴定三种离子总量，消耗 $V_1$ mL。

    $$n_{\text{Cu}} + n_{\text{Pb}} + n_{\text{Zn}} = c \times V_1$$
  2. 第二步：KCN 掩蔽后滴定 $\text{Pb}^{2+}$

    另取一份等量试液，加入过量 KCN（碱性条件）。$\text{CN}^-$ 将 $\text{Cu}^{2+}$、$\text{Zn}^{2+}$ 掩蔽为 $\text{Cu(CN)}_4^{3-}$、$\text{Zn(CN)}_4^{2-}$，但不掩蔽 $\text{Pb}^{2+}$（$\text{Pb}^{2+}$ 不与 $\text{CN}^-$ 形成稳定配合物）。

    用 EDTA 单独滴定 $\text{Pb}^{2+}$，消耗 $V_2$ mL。

    $$n_{\text{Pb}} = c \times V_2$$
  3. 第三步：甲醛解蔽 $\text{Zn}^{2+}$

    在上述含 KCN 的溶液中，加入甲醛（$\text{HCHO}$）。甲醛与 $\text{CN}^-$ 反应生成乙醇腈：

    $$\text{HCHO} + \text{CN}^- \rightarrow \text{HOCH}_2\text{CN}$$

    甲醛能解除 $\text{CN}^-$ 对 $\text{Zn}^{2+}$ 的掩蔽（释放 $\text{Zn}^{2+}$），但不能解除对 $\text{Cu}^{2+}$ 的掩蔽（$\text{Cu(CN)}_4^{3-}$ 更稳定）。

    继续用 EDTA 滴定释放出的 $\text{Zn}^{2+}$，消耗 $V_3$ mL。

    $$n_{\text{Zn}} = c \times V_3$$ $$n_{\text{Cu}} = c \times V_1 - c \times V_2 - c \times V_3 = c(V_1 - V_2 - V_3)$$
记忆要点：KCN 掩蔽 Cu、Zn、Cd、Ni、Co，不掩蔽 Pb、Ca、Mg。甲醛解蔽 Zn、Cd，不解蔽 Cu、Ni、Co。

真题考点精讲

Quiz 2 Q1：返滴定法的概念

考点解析

返滴定法（Back Titration）的核心操作：

  1. 向待测溶液中加入过量、已知量的 EDTA 标准溶液
  2. 加热使反应完全（适用于慢反应金属如 $\text{Al}^{3+}$、$\text{Cr}^{3+}$）
  3. 冷却后调节 pH
  4. 用另一种金属标准溶液（通常为 $\text{Zn}^{2+}$）回滴剩余的 EDTA

计算公式：$n_M = n_{\text{EDTA(加入)}} - n_{\text{Zn(回滴)}}$

易错点：返滴定中回滴用的金属离子（如 $\text{Zn}^{2+}$）与 EDTA 的 $\lg K$ 不能大于待测金属的 $\lg K$，否则会将 MY 中的 M 置换出来。通常要求 $K_{\text{ZnY}} < K_{\text{MY}}$。

Quiz 2 Q2：EDTA 在不同 pH 下的存在形态

关键数据

EDTA 的 $\text{p}K_a$ 值：0.9, 1.6, 2.07, 2.75, 6.24, 10.34

在 $\text{p}K_{a5} = 6.24$ 到 $\text{p}K_{a6} = 10.34$ 之间，$\text{HY}^{3-}$ 是主要形态。

在 pH = 2.75 ~ 6.24 之间，$\text{H}_2\text{Y}^{2-}$ 是主要形态（这也是二钠盐溶解在中性水中的形态）。

快速判断法：pH 处于 $\text{p}K_{a,n}$ 和 $\text{p}K_{a,n+1}$ 之间时，失去 $n$ 个质子的形态为主要形态。

2022 Final Q11：$\text{CaCO}_3$ 作为 EDTA 标定的基准物

考点解析

标定 EDTA 浓度常用的基准物质：

  • $\text{CaCO}_3$（纯度高、不吸湿、分子量大、易得）
  • $\text{Zn}$（金属锌粒）
  • $\text{ZnO}$
  • $\text{Bi}$（金属铋）
  • $\text{MgSO}_4 \cdot 7\text{H}_2\text{O}$（需注意含水量）

$\text{CaCO}_3$ 标定操作：

  1. 准确称取 $\text{CaCO}_3$ 基准物质
  2. 用少量 HCl 溶解：$\text{CaCO}_3 + 2\text{HCl} \rightarrow \text{CaCl}_2 + \text{H}_2\text{O} + \text{CO}_2\uparrow$
  3. 煮沸除去 $\text{CO}_2$
  4. 调 pH 至 12~13（加 NaOH），用 NN 指示剂
  5. 用待标定的 EDTA 滴定 $\text{Ca}^{2+}$
$$c_{\text{EDTA}} = \frac{m_{\text{CaCO}_3} / M_{\text{CaCO}_3}}{V_{\text{EDTA}}}$$
例题：EDTA 标准溶液的标定

称取 0.2504 g $\text{CaCO}_3$ 基准物质（$M = 100.09$），溶于 HCl，调 pH 至 12，用 EDTA 滴定，消耗 24.98 mL。求 EDTA 的浓度。

解：

$n_{\text{Ca}} = \frac{0.2504}{100.09} = 2.502 \times 10^{-3}\;\text{mol} = 2.502\;\text{mmol}$

$c_{\text{EDTA}} = \frac{2.502}{24.98} = 0.1002\;\text{mmol/mL} = 0.1002\;\text{mol/L}$

2022 Final Q13：$\text{Bi}^{3+}$/$\text{Fe}^{3+}$ 的分离

考点解析

$\lg K_{\text{BiY}} = 27.9$，$\lg K_{\text{FeY}} = 25.1$，$\Delta\lg K = 2.84 < 6$。

结论：单纯控制酸度不能分别滴定。$\Delta\lg K$ 至少需要 $\geq 6$ 才能实现分步滴定。

解决方案：

  • 加入抗坏血酸将 $\text{Fe}^{3+}$ 还原为 $\text{Fe}^{2+}$（$\lg K_{\text{Fe(II)Y}} = 14.3$），$\Delta\lg K = 27.9 - 14.3 = 13.6 \geq 6$ ✓
  • 或加入巯基乙酸掩蔽 $\text{Fe}^{3+}$
$\Delta\lg K = 2.84$ 这个数值是常考点。牢记：两种金属 $\lg K$ 差值不足 6 时，不能通过 pH 控制分步滴定，必须借助掩蔽剂。

综合计算考点：从条件稳定常数到最低 pH 的完整求解

例题：求 $\text{Cu}^{2+}$ 准确滴定的最低 pH（$c = 0.010$ mol/L，忽略金属副反应）
解：

$\lg K_{\text{CuY}} = 18.8$

由判据 $\lg cK' \geq 6$：

$$\lg c + \lg K_{\text{CuY}} - \lg\alpha_{Y(H)} \geq 6$$ $$-2 + 18.8 - \lg\alpha_{Y(H)} \geq 6$$ $$\lg\alpha_{Y(H)} \leq 10.8$$

查表：$\lg\alpha_{Y(H)} = 10.8$ 对应 pH $\approx$ 3

因此 $\text{Cu}^{2+}$ 的最低滴定 pH $\approx$ 3。

实际操作中选择 pH = 3~5（HAc-NaAc 缓冲），使用 PAN 或 XO 指示剂。

配位滴定法核心知识框架

一、基本概念线

EDTA 性质 $\rightarrow$ 稳定常数 $K_{MY}$ $\rightarrow$ 副反应系数 $\alpha$ $\rightarrow$ 条件稳定常数 $K'_{MY}$ $\rightarrow$ 准确滴定判据 $\lg cK' \geq 6$

二、应用技术线

酸度控制 $\rightarrow$ 选择性滴定 $\rightarrow$ 掩蔽与解蔽 $\rightarrow$ 四种滴定方式 $\rightarrow$ 实际分析方案

三、计算能力线

查 $\lg\alpha_{Y(H)}$ 表 $\rightarrow$ 求 $\lg K'$ $\rightarrow$ 判断能否滴定 $\rightarrow$ 求最低/最高 pH $\rightarrow$ 滴定曲线 $\rightarrow$ 终点误差

易错点总结

  1. 副反应系数的加法与乘法：严格公式是 $\alpha_Y = \alpha_{Y(H)} + \alpha_{Y(N)} - 1$（加法减一），不是简单的乘法。乘法近似只在各项 $\gg 1$ 时成立。
  2. $\lg cK'$ 中的 $c$ 是 $c_{sp}$（化学计量点浓度），不是原始浓度。等体积滴定时 $c_{sp} = c_0 / 2$。
  3. $pM'_{sp}$ 公式中的 $pc_{sp}$：$pc_{sp} = -\lg c_{sp}$，注意 $c_{sp}$ 是稀释后的浓度。
  4. 分别滴定判据 $\Delta\lg cK' \geq 6$：是两个 $\lg cK'$ 之差，不是两个 $\lg K$ 之差（虽然浓度相同时效果相同）。
  5. 封闭 vs 僵化：封闭 = MIn 太稳定，EDTA 夺不走；僵化 = 置换反应速度太慢。两者解决方案不同。
  6. EBT 适用 pH 范围：pH 8~11。pH $< 6.3$ 时 EBT 本身变色（红色），无法指示终点。此时应选 XO。
  7. 返滴定中回滴金属的选择：$K_{\text{回滴MY}}$ 必须 $< K_{\text{待测MY}}$，否则发生置换反应。
