第七章 氧化还原滴定法

7.1 条件电位与Nernst方程

PPT: 可逆电对与不可逆电对

PPT: 条件电位的概念

可逆电对与不可逆电对

根据电极反应能否在任一瞬间迅速建立氧化还原平衡，电对分为：

可逆电对

在反应的任一瞬间能迅速建立平衡，实测电位与 Nernst 方程计算值一致。

例：$\text{Fe}^{3+}/\text{Fe}^{2+}$、$\text{Ce}^{4+}/\text{Ce}^{3+}$、$\text{I}_2/\text{I}^-$

不可逆电对

在反应的任一瞬间不能迅速建立平衡，实测电位与 Nernst 方程计算值不一致或相差很大。

例：$\text{MnO}_4^-/\text{Mn}^{2+}$、$\text{Cr}_2\text{O}_7^{2-}/\text{Cr}^{3+}$、$\text{CO}_2/\text{C}_2\text{O}_4^{2-}$、$\text{S}_4\text{O}_6^{2-}/\text{S}_2\text{O}_3^{2-}$

判断方法：能否在氧化还原反应的任一瞬间建立真正的平衡，或实测电位与理论电位是否相等。不可逆电对在滴定曲线计算中只能在化学计量点前后用另一电对（可逆的）进行计算。

Nernst方程

标准 Nernst 方程
$$\varphi = \varphi^0 + \frac{0.0592}{n}\lg\frac{[\text{Ox}]}{[\text{Red}]}$$

其中 $\varphi^0$ 为标准电极电位，$n$ 为电子转移数。严格地说，方程中的浓度应替换为活度 $a$。在实际分析中，由于活度系数和副反应的影响，需引入条件电位。

条件电位的推导

考虑电对 $\text{Ox} + ne^- \rightleftharpoons \text{Red}$，若 Ox 和 Red 都存在副反应（配位、沉淀、酸碱等），令：

  • $\alpha_\text{Ox}$：Ox 的副反应系数（$\alpha_\text{Ox} = c_\text{Ox}/[\text{Ox}]$，$c_\text{Ox}$ 为总分析浓度）
  • $\alpha_\text{Red}$：Red 的副反应系数

则 $[\text{Ox}] = c_\text{Ox}/\alpha_\text{Ox}$，$[\text{Red}] = c_\text{Red}/\alpha_\text{Red}$，代入 Nernst 方程：

$$\varphi = \varphi^0 + \frac{0.0592}{n}\lg\frac{c_\text{Ox}/\alpha_\text{Ox}}{c_\text{Red}/\alpha_\text{Red}} = \varphi^0 + \frac{0.0592}{n}\lg\frac{\alpha_\text{Red}}{\alpha_\text{Ox}} + \frac{0.0592}{n}\lg\frac{c_\text{Ox}}{c_\text{Red}}$$
条件电位 $\varphi^{0'}$
$$\boxed{\varphi^{0'} = \varphi^0 + \frac{0.0592}{n}\lg\frac{\alpha_{\text{Red}}}{\alpha_{\text{Ox}}}}$$

条件电位考虑了离子强度、副反应（配位、酸碱等）对电对实际氧化还原能力的影响。

使用条件电位后 Nernst 方程变为：

$$\varphi = \varphi^{0'} + \frac{0.0592}{n}\lg\frac{c_{\text{Ox}}}{c_{\text{Red}}}$$
条件电位比标准电位更能反映实际溶液条件下电对的氧化还原能力。在分析计算中，应优先使用条件电位。

影响条件电位的因素

  • 离子强度：通过活度系数影响
  • 酸度：含 $\text{H}^+$ 参与的电对（如 $\text{MnO}_4^-/\text{Mn}^{2+}$）受 pH 影响显著
  • 配位效应：配位剂优先稳定某一氧化态时改变电位。如 $\text{Fe}^{3+}/\text{Fe}^{2+}$ 加 $\text{F}^-$ 后电位降低（$\text{F}^-$ 主要配位 $\text{Fe}^{3+}$）；加邻菲罗啉后电位升高（主要配位 $\text{Fe}^{2+}$）
  • 沉淀效应：某一氧化态生成沉淀使其浓度降低
规律总结：
  • 配位剂（或沉淀剂）主要作用于氧化态（$\alpha_\text{Ox}$ 增大）→ 条件电位降低，氧化态的氧化能力减弱
  • 配位剂（或沉淀剂）主要作用于还原态（$\alpha_\text{Red}$ 增大）→ 条件电位升高，还原态更容易被氧化

酸度对条件电位的影响 -- 含 $\text{H}^+$ 参与的电对

PPT: 酸度对条件电位的影响 -- As(V)/As(III)

对于 $\text{H}^+$ 直接参与的电对，如 $\text{H}_3\text{AsO}_4/\text{H}_3\text{AsO}_3$（$\varphi^0 = 0.559$ V），条件电位与 pH 密切相关。

精确处理方法

将 Nernst 方程中的 $[\text{Ox}]$ 和 $[\text{Red}]$ 展开为总浓度和分布系数：

$$\varphi = \varphi^0_{\text{H}_3\text{AsO}_4/\text{H}_3\text{AsO}_3} + \frac{0.0592}{2}\lg\frac{\delta_{\text{H}_3\text{AsO}_4}}{\delta_{\text{H}_3\text{AsO}_3}} + \frac{0.0592}{2}\lg\frac{c_{\text{As(V)}}}{c_{\text{As(III)}}}$$

其中 $\delta$ 为各存在形态的分布系数，与 pH 和酸的 $K_a$ 有关。因此：

$$\varphi^{0'} = \varphi^0 + \frac{0.0592}{2}\lg\frac{\delta_{\text{H}_3\text{AsO}_4}}{\delta_{\text{H}_3\text{AsO}_3}}$$

当 pH 变化时，$\delta$ 值变化，条件电位随之改变。如 $\text{H}_3\text{AsO}_4$ 的 $pK_a$ 为 2.2, 7.0, 11.5，$\text{H}_3\text{AsO}_3$ 的 $pK_a$ 为 9.2，不同 pH 下分布分数差异很大。

简化结论：对含 $\text{H}^+$ 参与的电对，酸度增大通常使条件电位升高（氧化能力增强）。例如 $\text{MnO}_4^-$ 在强酸性介质中氧化能力最强（$\varphi^{0'} = 1.51$ V），而在中性或碱性介质中氧化能力减弱。
例题1：Fe(III)-F 配位体系的条件电位计算

已知 $\text{Fe}^{3+}/\text{Fe}^{2+}$ 的 $\varphi^0 = 0.771$ V。$\text{Fe}^{3+}$ 与 $\text{F}^-$ 的逐级稳定常数为 $\lg\beta_1=5.2$, $\lg\beta_2=9.2$, $\lg\beta_3=11.9$。$\text{Fe}^{2+}$ 与 $\text{F}^-$ 基本不配位（$\alpha_{\text{Fe(II)}} \approx 1$）。若 $[\text{F}^-] = 10^{-1.5}$ mol/L，求条件电位 $\varphi^{0'}$。

解：

第一步：计算 $\text{Fe}^{3+}$ 的副反应系数

$$\alpha_{\text{Fe(III)}(\text{F})} = 1 + \beta_1[\text{F}^-] + \beta_2[\text{F}^-]^2 + \beta_3[\text{F}^-]^3$$ $$= 1 + 10^{5.2}\times 10^{-1.5} + 10^{9.2}\times 10^{-3.0} + 10^{11.9}\times 10^{-4.5}$$ $$= 1 + 10^{3.7} + 10^{6.2} + 10^{7.4}$$

取最大项：$\alpha_{\text{Fe(III)}(\text{F})} \approx 10^{7.4}$

第二步：由于 $\text{Fe}^{2+}$ 与 $\text{F}^-$ 几乎不配位，$\alpha_{\text{Fe(II)}} \approx 1$

第三步：计算条件电位

$$\varphi^{0'} = \varphi^0 + \frac{0.0592}{1}\lg\frac{\alpha_\text{Red}}{\alpha_\text{Ox}} = 0.771 + 0.0592\times\lg\frac{1}{10^{7.4}}$$ $$= 0.771 + 0.0592\times(-7.4) = 0.771 - 0.438 = \boxed{0.333\;\text{V}}$$

条件电位从 0.771 V 大幅降至 0.333 V，说明 $\text{F}^-$ 配位使 $\text{Fe}^{3+}$ 的氧化能力显著减弱。

例题2：邻菲罗啉（phen）对 Fe 电对的影响

邻菲罗啉（1,10-phenanthroline, phen）主要与 $\text{Fe}^{2+}$ 形成极稳定的 $[\text{Fe(phen)}_3]^{2+}$ 配合物（红色，ferroin），而与 $\text{Fe}^{3+}$ 的配合物较弱。定性分析条件电位如何变化。

解：

phen 主要稳定 $\text{Fe}^{2+}$（还原态），故 $\alpha_\text{Red} \gg \alpha_\text{Ox}$。

$$\varphi^{0'} = \varphi^0 + \frac{0.0592}{1}\lg\frac{\alpha_\text{Red}}{\alpha_\text{Ox}} > \varphi^0$$

条件电位升高（实际测定 $\varphi^{0'} \approx 1.06$ V）。这意味着 $\text{Fe}^{2+}$ 被 phen 稳定后，更难被氧化，相当于该电对的氧化能力增强。

这正是 ferroin 可用作 $\text{Ce}^{4+}/\text{Fe}^{2+}$ 滴定指示剂的原因（$\varphi^{0'}_\text{In} = 1.06$ V 恰好接近 $\varphi_{sp}$）。

例题3：H3PO4 在 K2Cr2O7 法中的作用

在 $\text{K}_2\text{Cr}_2\text{O}_7$ 法测铁中，滴定前加入 $\text{H}_3\text{PO}_4$。已知 $\text{H}_3\text{PO}_4$ 与 $\text{Fe}^{3+}$ 形成 $\text{Fe(HPO}_4)^+$ 等无色配合物（$\lg\beta \approx 9.4$），而与 $\text{Fe}^{2+}$ 配合能力很弱。分析其作用。

解：

加入 $\text{H}_3\text{PO}_4$ 后：

  1. 降低条件电位：$\text{H}_3\text{PO}_4$ 主要配位氧化态 $\text{Fe}^{3+}$，$\alpha_\text{Ox}$ 增大，$\varphi^{0'}$ 降低（从 0.68 V 降至约 0.44 V）
  2. 消除颜色干扰：$\text{Fe}^{3+}$ 在 HCl 溶液中呈黄色，与 $\text{Fe(HPO}_4)^+$ 配合后变无色，有利于终点观察
  3. 增大滴定突跃：$\varphi^{0'}_{\text{Fe}}$ 降低使 $\Delta\varphi^{0'}$ 增大（从 $1.33 - 0.68 = 0.65$ V 增至 $1.33 - 0.44 = 0.89$ V），突跃范围增大，终点更敏锐

这就是为什么 $\text{K}_2\text{Cr}_2\text{O}_7$ 法测铁要使用 $\text{H}_2\text{SO}_4 + \text{H}_3\text{PO}_4$ 混合酸。

7.2 氧化还原反应的平衡常数与定量判据

平衡常数的推导

考虑反应 $n_2\text{Ox}_1 + n_1\text{Red}_2 \rightleftharpoons n_2\text{Red}_1 + n_1\text{Ox}_2$

其中电对1: $\text{Ox}_1 + n_1 e^- \rightleftharpoons \text{Red}_1$，$\varphi_1^{0'}$；
电对2: $\text{Ox}_2 + n_2 e^- \rightleftharpoons \text{Red}_2$，$\varphi_2^{0'}$。

在平衡时，两个电对的电位相等（$\varphi_1 = \varphi_2$）：

$$\varphi_1^{0'} + \frac{0.0592}{n_1}\lg\frac{c_{\text{Ox}_1}}{c_{\text{Red}_1}} = \varphi_2^{0'} + \frac{0.0592}{n_2}\lg\frac{c_{\text{Ox}_2}}{c_{\text{Red}_2}}$$

两边同乘 $n_1 n_2 / 0.0592$，经整理可得：

平衡常数
$$\boxed{\lg K = \frac{n_1 n_2 (\varphi_1^{0'} - \varphi_2^{0'})}{0.0592} = \frac{n(\varphi_1^{0'} - \varphi_2^{0'})}{0.0592}}$$

其中 $n = n_1 \cdot n_2$（当电子转移数互质时），或更一般地 $n$ 为反应中实际电子转移总数（即 $n_1$ 和 $n_2$ 的最小公倍数）。

定量反应判据
$$\boxed{\Delta\varphi^{0'} = \varphi_1^{0'} - \varphi_2^{0'} \geq 0.4\;\text{V}}$$

（对 $n_1 = n_2 = 1$ 的情况，此时 $\lg K \geq 6.8$）

一般来说，$\lg K \geq 6$ 即可认为反应定量完成（反应完成度 $\geq 99.9\%$）。

注意：$\Delta\varphi^{0'} \geq 0.4$ V 仅是经验判据，具体还需考虑 $n$ 值。当 $n$ 较大时，所需 $\Delta\varphi^{0'}$ 可以更小。例如 $n = 2$ 时 $\Delta\varphi^{0'} \geq 0.2$ V 即可；$n = 3$ 时 $\Delta\varphi^{0'} \geq 0.12$ V。
反应完全度

反应完全度是衡量化学计量点时反应完成程度的指标。对于 $n_1 = n_2 = 1$ 的对称反应（等浓度）：

$$\text{完成度} = 1 - \frac{c_{sp,\text{Red}_2}}{c_0} \approx 1 - \frac{1}{\sqrt{K}}$$

当 $\lg K \geq 6$ 时，完成度 $\geq 99.9\%$，满足定量分析要求。

例题4：判断 Ce4+ 能否定量滴定 Fe2+

在 1 mol/L $\text{H}_2\text{SO}_4$ 中，$\varphi^{0'}_{\text{Ce}^{4+}/\text{Ce}^{3+}} = 1.44$ V，$\varphi^{0'}_{\text{Fe}^{3+}/\text{Fe}^{2+}} = 0.68$ V。判断能否定量滴定。

解：

$n_1 = n_2 = 1$，$n = 1$

$$\lg K = \frac{1 \times (1.44 - 0.68)}{0.0592} = \frac{0.76}{0.0592} = 12.8$$

$\lg K = 12.8 \gg 6$，因此 $\text{Ce}^{4+}$ 完全可以定量滴定 $\text{Fe}^{2+}$。

同时 $\Delta\varphi^{0'} = 0.76 \;\text{V} > 0.4\;\text{V}$，满足判据。

例题5：I2 能否滴定 S2O32-？

已知 $\varphi^0_{\text{I}_2/\text{I}^-} = 0.545$ V，$\varphi^0_{\text{S}_4\text{O}_6^{2-}/\text{S}_2\text{O}_3^{2-}} = 0.08$ V。判断 $\text{I}_2$ 能否定量滴定 $\text{S}_2\text{O}_3^{2-}$。

解：

反应：$\text{I}_2 + 2\text{S}_2\text{O}_3^{2-} \rightarrow 2\text{I}^- + \text{S}_4\text{O}_6^{2-}$

$n_1 = 2$（I2 得 2 个电子），$n_2 = 2$（2 个 S2O32- 共失 2 个电子），$n = 2$

$$\lg K = \frac{2 \times (0.545 - 0.08)}{0.0592} = \frac{2 \times 0.465}{0.0592} = 15.7$$

$\lg K = 15.7 \gg 6$，反应非常完全，$\text{I}_2$ 可以定量滴定 $\text{S}_2\text{O}_3^{2-}$。

这就是碘量法中用 $\text{Na}_2\text{S}_2\text{O}_3$ 标准溶液滴定 $\text{I}_2$ 的理论基础。

$$\boxed{\varphi_{sp} = \frac{n_1\varphi_1^{0'} + n_2\varphi_2^{0'}}{n_1 + n_2}}$$

对 $\text{Ce}^{4+}/\text{Fe}^{2+}$ 体系（$n_1 = n_2 = 1$）：

$$\varphi_{sp} = \frac{0.68 + 1.44}{2} = 1.06\;\text{V}$$
对称电对：当 $n_1 = n_2$ 时，化学计量点在突跃中央（对称曲线）。当 $n_1 \neq n_2$ 时（如 $\text{MnO}_4^-/\text{Fe}^{2+}$，$n_1=5, n_2=1$），化学计量点偏向高电位一侧。

不对称曲线：MnO4- 滴定 Fe2+

$n_1 = 5$（MnO4- 得 5e），$n_2 = 1$（Fe2+ 失 1e）

$$\varphi_{sp} = \frac{5 \times 1.51 + 1 \times 0.68}{5 + 1} = \frac{7.55 + 0.68}{6} = \frac{8.23}{6} = 1.37\;\text{V}$$

$\varphi_{sp} = 1.37$ V 明显偏向高电位端（$\text{MnO}_4^-$ 一侧），这就是不对称曲线的特征。

突跃范围的计算方法

突跃范围由 $\pm 0.1\%$ 的滴定误差所对应的电位变化定义：

  • 突跃下限（99.9% 处）：用被滴定物的电对计算
$$\varphi_{99.9\%} = \varphi_2^{0'} + \frac{0.0592}{n_2}\lg\frac{99.9}{0.1} = \varphi_2^{0'} + \frac{3 \times 0.0592}{n_2}$$
  • 突跃上限（100.1% 处）：用滴定剂的电对计算
$$\varphi_{100.1\%} = \varphi_1^{0'} + \frac{0.0592}{n_1}\lg\frac{0.1}{99.9} = \varphi_1^{0'} - \frac{3 \times 0.0592}{n_1}$$
影响突跃大小的因素：(1) $\Delta\varphi^{0'}$ 越大，突跃越大；(2) $n_1, n_2$ 越大，突跃越大（因为 $\frac{0.0592}{n}$ 更小，突跃上下限更接近各自的 $\varphi^{0'}$）。
例题6：在 1 mol/L $\text{H}_2\text{SO}_4$ 中，用 $\text{Ce}^{4+}$（$\varphi^{0'}=1.44$ V）滴定 $\text{Fe}^{2+}$（$\varphi^{0'}=0.68$ V），求化学计量点电位、突跃范围及 $\lg K$。
解：

$\varphi_{sp} = \frac{1 \times 1.44 + 1 \times 0.68}{1+1} = 1.06\;\text{V}$

$\lg K = \frac{1 \times (1.44 - 0.68)}{0.0592} = \frac{0.76}{0.0592} = 12.8$

$K$ 非常大，反应完全定量。

突跃下限：$\varphi_{99.9\%} = 0.68 + 3 \times 0.0592 = 0.86\;\text{V}$

突跃上限：$\varphi_{100.1\%} = 1.44 - 3 \times 0.0592 = 1.26\;\text{V}$

突跃范围：$0.86 \sim 1.26\;\text{V}$，可选用 ferroin（$\varphi^{0'}_\text{In} = 1.06$ V）作指示剂。

氧化还原滴定的终点误差公式

PPT: 氧化还原滴定终点误差

PPT: 终点误差的详细推导

一般终点误差公式（可逆对称电对，$n_1 = n_2 = 1$）
$$\boxed{E_t = \frac{10^{\Delta\varphi / 0.0592} - 10^{-\Delta\varphi / 0.0592}}{10^{\Delta\varphi^{0'} / (2\times 0.0592)}} \times 100\%}$$

其中 $\Delta\varphi = \varphi_{ep} - \varphi_{sp}$（指示剂变色点电位与化学计量点电位之差）。

一般形式（$n_1 \neq n_2$ 时）
$$E_t = \frac{10^{n_2\Delta\varphi / 0.0592} - 10^{-n_1\Delta\varphi / 0.0592}}{10^{n_1 n_2 \Delta\varphi^{0'} / [0.0592(n_1+n_2)]}} \times 100\%$$

分析：

  • 终点误差不仅与电位差 $\Delta\varphi$ 有关，还与电对半反应的得失电子数 $n$ 有关
  • 条件电位差 $\Delta\varphi^{0'}$ 越大，终点误差越小（分母越大）
  • 终点离化学计量点越近（$\Delta\varphi$ 越小），终点误差越小

7.4 氧化还原指示剂

指示剂类型

1. 自身指示剂

滴定剂本身有颜色变化。最典型：$\text{KMnO}_4$（$\text{MnO}_4^-$ 紫红色，$\text{Mn}^{2+}$ 几乎无色）。滴定终点多半滴 $\text{KMnO}_4$ 即呈粉红色。

2. 特殊指示剂

淀粉指示剂：用于碘量法，淀粉与 $\text{I}_2$ 形成深蓝色吸附化合物。

淀粉指示剂应在近终点时加入（滴定碘法中溶液变浅黄后再加），否则大量 $\text{I}_2$ 被淀粉吸附不易释放，造成终点拖后。

$\text{Fe(SCN)}^{2+}$ 指示剂：$\text{Sn}^{2+} + 2\text{Fe}^{3+} = 2\text{Fe}^{2+} + \text{Sn}^{4+}$ 反应中，加入 $\text{SCN}^-$，当有 $\text{Fe}^{3+}$ 存在时显红色（$\text{Fe(SCN)}^{2+}$），还原完全后红色消失。这是一种利用特定反应产物颜色变化的特殊指示剂。

3. 氧化还原指示剂

其氧化态和还原态颜色不同，变色取决于溶液电位：

$$\varphi = \varphi^{0'}_{In} + \frac{0.0592}{n}\lg\frac{[\text{In}_{\text{Ox}}]}{[\text{In}_{\text{Red}}]}$$

变色范围：$\varphi^{0'}_{In} \pm \frac{0.0592}{n}$

指示剂$\varphi^{0'}$/V颜色（氧化态/还原态）
二苯胺磺酸钠0.84紫色/无色
邻二氮菲亚铁（ferroin）1.06淡蓝/红色
次甲基蓝0.53蓝色/无色
选择指示剂的原则：指示剂的 $\varphi^{0'}_{In}$ 应尽量接近 $\varphi_{sp}$，且落在滴定突跃范围内。

7.5 影响氧化还原反应速率的因素

  • 反应物浓度：浓度越大，反应速率越快
  • 温度：升温加速反应（$\text{KMnO}_4$ 法用 $\text{Na}_2\text{C}_2\text{O}_4$ 标定时需加热至 75~85 °C）
  • 催化剂：$\text{KMnO}_4$ 与 $\text{C}_2\text{O}_4^{2-}$ 反应中，$\text{Mn}^{2+}$ 为自催化剂；加入 $\text{OsO}_4$ 或 $\text{ICl}$ 催化 $\text{Ce}^{4+}$-$\text{As}(\text{III})$ 反应
  • 诱导反应：一个反应诱导另一个热力学上可行但动力学上缓慢的反应发生。诱导反应会引起诱导误差（如 $\text{KMnO}_4$ 法中 $\text{Cl}^-$ 的干扰部分源于诱导效应）

7.6 预处理：预氧化与预还原

滴定前需将待测物定量转化为某一特定氧化态，称为预处理。

预氧化剂

氧化剂用途除去过量的方法
$(\text{NH}_4)_2\text{S}_2\text{O}_8$（过硫酸铵）氧化 $\text{Mn}^{2+}$ 为 $\text{MnO}_4^-$, $\text{Cr}^{3+}$ 为 $\text{Cr}_2\text{O}_7^{2-}$煮沸分解
$\text{NaBiO}_3$（铋酸钠）氧化 $\text{Mn}^{2+}$ 为 $\text{MnO}_4^-$过滤除去（不溶性固体）
$\text{HClO}_4$（高氯酸）高温下的强氧化剂稀释冷却后失去氧化性
$\text{H}_2\text{O}_2$氧化多种还原态离子煮沸分解

预还原剂

还原剂用途除去过量的方法
$\text{SnCl}_2$还原 $\text{Fe}^{3+}$ 为 $\text{Fe}^{2+}$加 $\text{HgCl}_2$（见铁的测定）
Jones 还原器（锌汞齐）还原多种金属离子通过分离
银还原器选择性还原 $\text{Fe}^{3+}$通过分离
关键原则：过量的预氧化/还原剂必须能够方便地除去，否则会干扰后续滴定。

SnCl2 + HgCl2 预还原铁的详细步骤

在 $\text{K}_2\text{Cr}_2\text{O}_7$ 法测铁中，需将 $\text{Fe}^{3+}$ 预还原为 $\text{Fe}^{2+}$：

  1. 热 HCl 溶液中逐滴加入 $\text{SnCl}_2$：$\text{Sn}^{2+} + 2\text{Fe}^{3+} \rightarrow \text{Sn}^{4+} + 2\text{Fe}^{2+}$

    边加边摇匀，观察溶液颜色由深黄色逐渐变浅。当溶液变为浅黄色或接近无色时，再多加 1~2 滴（稍过量），停止加入。

  2. 冷却：将溶液冷至室温（热溶液加 $\text{HgCl}_2$ 反应过快，不易控制）。
  3. 快速加入 20 mL $\text{HgCl}_2$ 溶液（过量）：

    $\text{Sn}^{2+} + 2\text{HgCl}_2 \rightarrow \text{Sn}^{4+} + \text{Hg}_2\text{Cl}_2\downarrow$（白色丝状沉淀）

    $\text{Hg}_2\text{Cl}_2$（甘汞）不与 $\text{K}_2\text{Cr}_2\text{O}_7$ 反应，不干扰滴定。

  4. 静置 2~3 分钟后，观察沉淀颜色。
三种情况的判断：
  • 正常：白色丝状 $\text{Hg}_2\text{Cl}_2$ 沉淀 → 操作成功，可继续滴定
  • 无白色沉淀 → $\text{SnCl}_2$ 加入不足，$\text{Fe}^{3+}$ 未被完全还原，必须弃去重做
  • 灰黑色沉淀 → $\text{SnCl}_2$ 过量太多，过量的 $\text{Sn}^{2+}$ 将 $\text{Hg}_2\text{Cl}_2$ 进一步还原为金属 Hg：

    $\text{Sn}^{2+} + \text{Hg}_2\text{Cl}_2 \rightarrow \text{Sn}^{4+} + 2\text{Hg}\downarrow(\text{灰黑色}) + 2\text{Cl}^-$

    金属 Hg 能还原 $\text{K}_2\text{Cr}_2\text{O}_7$，导致结果偏高。必须弃去重做。

操作要诀：$\text{SnCl}_2$ 的用量是关键。宁可多做一次也不可冒险。实际操作中，当溶液变浅黄近无色后，仅多加 1~2 滴即可。加 $\text{HgCl}_2$ 时应一次快速加入，不可缓慢滴加。

7.7 高锰酸钾法

PPT: 高锰酸钾法

基本特性

$\text{KMnO}_4$ 在强酸性介质中：$\text{MnO}_4^- + 8\text{H}^+ + 5e^- \rightarrow \text{Mn}^{2+} + 4\text{H}_2\text{O}$，$\varphi^0 = 1.51$ V

自身指示剂：终点溶液呈粉红色（0.02 mL 0.02 mol/L $\text{KMnO}_4$ 可使 50 mL 溶液显色）。

$\text{KMnO}_4$ 不是基准物质（含 $\text{MnO}_2$ 等杂质），需标定。配制时需煮沸、静置、过滤除去 $\text{MnO}_2$，保存于棕色瓶中。

用 $\text{Na}_2\text{C}_2\text{O}_4$ 标定 $\text{KMnO}_4$ 的详细操作

反应：$2\text{MnO}_4^- + 5\text{C}_2\text{O}_4^{2-} + 16\text{H}^+ \rightarrow 2\text{Mn}^{2+} + 10\text{CO}_2 + 8\text{H}_2\text{O}$

标准操作步骤

  1. 准确称取 $\text{Na}_2\text{C}_2\text{O}_4$ 基准物（已于 105~110 °C 干燥 2 h），溶于水
  2. 加入 $\text{H}_2\text{SO}_4$ 使酸度为 0.5~1 mol/L
  3. 加热至 75~85 °C（不可煮沸！）
  4. 开始滴入 $\text{KMnO}_4$：第一滴加入后，紫红色褪色很慢（此时无 $\text{Mn}^{2+}$ 催化），须充分搅拌，等紫色完全褪去后再加第二滴
  5. 随着 $\text{Mn}^{2+}$ 积累（自催化），反应加速，可适当加快滴定速度
  6. 临近终点时减慢速度，最后半滴 $\text{KMnO}_4$ 使溶液呈粉红色，30 s 不褪即为终点
"三度一点"操作要点
  • 酸度：0.5~1 mol/L $\text{H}_2\text{SO}_4$
    • 酸度过低：生成 $\text{MnO}_2$（棕色），而非 $\text{Mn}^{2+}$，反应不按计量关系进行
    • 酸度过高：$\text{H}_2\text{C}_2\text{O}_4$ 分解加快
  • 温度：加热至 75~85 °C
    • 温度过低：反应太慢
    • 煮沸：$\text{H}_2\text{C}_2\text{O}_4$ 分解（$\text{H}_2\text{C}_2\text{O}_4 \rightarrow \text{CO}_2 + \text{CO} + \text{H}_2\text{O}$），使结果偏高
  • 速度：开始慢（等待自催化），中间快，近终点慢
  • 终点：最后半滴 $\text{KMnO}_4$ 使溶液呈粉红色，30 s 不褪

为什么不能用 HCl 或 HNO3？

不能用 HCl 的原因：
  1. $\text{Cl}^-$ 被氧化：$2\text{MnO}_4^- + 10\text{Cl}^- + 16\text{H}^+ \rightarrow 2\text{Mn}^{2+} + 5\text{Cl}_2 + 8\text{H}_2\text{O}$，消耗 $\text{KMnO}_4$，结果偏高
  2. 诱导反应：$\text{MnO}_4^-$ 与 $\text{C}_2\text{O}_4^{2-}$ 的反应会诱导 $\text{Cl}^-$ 被氧化（即使从热力学上看 $\text{Cl}^-$ 的氧化速度很慢，在 $\text{MnO}_4^-$-$\text{C}_2\text{O}_4^{2-}$ 反应的诱导下也会发生），引起显著的正误差
不能用 HNO3 的原因：

$\text{HNO}_3$ 本身是氧化性酸，在加热条件下会氧化 $\text{C}_2\text{O}_4^{2-}$（或其他还原性被测物），导致 $\text{KMnO}_4$ 用量偏少，结果偏低。

因此 $\text{KMnO}_4$ 法只能使用 $\text{H}_2\text{SO}_4$ 作为酸性介质（$\text{SO}_4^{2-}$ 不被氧化，$\text{H}_2\text{SO}_4$ 也不是氧化性酸）。

$\text{KMnO}_4$ 法的应用

  • 测定 $\text{Fe}^{2+}$（直接滴定）
  • 测定 $\text{Ca}^{2+}$（间接法：沉淀为 $\text{CaC}_2\text{O}_4$，酸溶后滴定 $\text{C}_2\text{O}_4^{2-}$）
  • 测定 $\text{H}_2\text{O}_2$（直接滴定：$2\text{MnO}_4^- + 5\text{H}_2\text{O}_2 + 6\text{H}^+ \rightarrow 5\text{O}_2 + 2\text{Mn}^{2+} + 8\text{H}_2\text{O}$）
  • 测定软锰矿中 $\text{MnO}_2$ 含量
例题7：KMnO4 法间接测定 Ca2+

将含 $\text{Ca}^{2+}$ 的溶液用 $(\text{NH}_4)_2\text{C}_2\text{O}_4$ 沉淀为 $\text{CaC}_2\text{O}_4$，过滤洗涤后用 $\text{H}_2\text{SO}_4$ 溶解，加热至 75~85 °C，用 0.02000 mol/L $\text{KMnO}_4$ 标准溶液滴定，消耗 25.00 mL。求 $\text{Ca}^{2+}$ 的物质的量。

解：

$\text{CaC}_2\text{O}_4 \xrightarrow{\text{H}_2\text{SO}_4} \text{Ca}^{2+} + \text{C}_2\text{O}_4^{2-}$（酸溶）

$2\text{MnO}_4^- + 5\text{C}_2\text{O}_4^{2-} + 16\text{H}^+ \rightarrow 2\text{Mn}^{2+} + 10\text{CO}_2 + 8\text{H}_2\text{O}$

$n_{\text{MnO}_4^-} = 0.02000 \times 25.00 \times 10^{-3} = 5.000 \times 10^{-4}\;\text{mol}$

$n_{\text{C}_2\text{O}_4^{2-}} = \frac{5}{2} \times n_{\text{MnO}_4^-} = \frac{5}{2} \times 5.000 \times 10^{-4} = 1.250 \times 10^{-3}\;\text{mol}$

因 $\text{Ca}^{2+} : \text{C}_2\text{O}_4^{2-} = 1:1$（在 $\text{CaC}_2\text{O}_4$ 中），故：

$$n_{\text{Ca}^{2+}} = n_{\text{C}_2\text{O}_4^{2-}} = 1.250 \times 10^{-3}\;\text{mol}$$
例题（PPT）：$\text{KMnO}_4$ 法测定软锰矿中 $\text{MnO}_2$ 含量

PPT: KMnO4法测软锰矿

称取软锰矿 0.3216 g，分析纯 $\text{Na}_2\text{C}_2\text{O}_4$ 0.3685 g，共置于同一烧杯中，加入 $\text{H}_2\text{SO}_4$，加热使反应完全，用 0.02400 mol/L $\text{KMnO}_4$ 溶液滴定剩余 $\text{Na}_2\text{C}_2\text{O}_4$，消耗 11.26 mL。求软锰矿中 $\text{MnO}_2$ 的质量分数。

解：

原理：$\text{MnO}_2$ 先与过量 $\text{Na}_2\text{C}_2\text{O}_4$ 反应：

$\text{MnO}_2 + \text{C}_2\text{O}_4^{2-} + 4\text{H}^+ \rightarrow \text{Mn}^{2+} + 2\text{CO}_2 + 2\text{H}_2\text{O}$（剩余 $\text{C}_2\text{O}_4^{2-}$）

再用 $\text{KMnO}_4$ 滴定剩余的 $\text{C}_2\text{O}_4^{2-}$：

$2\text{MnO}_4^- + 5\text{C}_2\text{O}_4^{2-} + 16\text{H}^+ \rightarrow 2\text{Mn}^{2+} + 10\text{CO}_2 + 8\text{H}_2\text{O}$

计算：基本单元关系 $\text{MnO}_2 \sim \text{Na}_2\text{C}_2\text{O}_4$（1:1），$\text{KMnO}_4 \sim \frac{5}{2}\text{C}_2\text{O}_4^{2-}$

$n_{\text{C}_2\text{O}_4^{2-},\text{total}} = \frac{0.3685}{134.0} = 2.750 \times 10^{-3}\;\text{mol}$

$n_{\text{C}_2\text{O}_4^{2-},\text{excess}} = \frac{5}{2} \times n_{\text{KMnO}_4} = \frac{5}{2} \times 0.02400 \times 11.26 \times 10^{-3} = 6.756 \times 10^{-4}\;\text{mol}$

$n_{\text{MnO}_2} = n_{\text{C}_2\text{O}_4^{2-},\text{total}} - n_{\text{C}_2\text{O}_4^{2-},\text{excess}} = 2.750 \times 10^{-3} - 6.756 \times 10^{-4} = 2.074 \times 10^{-3}\;\text{mol}$

$$w(\text{MnO}_2) = \frac{2.074 \times 10^{-3} \times 86.94}{0.3216} \times 100\% = 56.0\%$$

7.8 重铬酸钾法

PPT: 重铬酸钾法

基本特性

$\text{Cr}_2\text{O}_7^{2-} + 14\text{H}^+ + 6e^- \rightarrow 2\text{Cr}^{3+} + 7\text{H}_2\text{O}$，$\varphi^0 = 1.33$ V

$\text{K}_2\text{Cr}_2\text{O}_7$ 法的优点

  • $\text{K}_2\text{Cr}_2\text{O}_7$ 可作基准物质（高纯、稳定、$M$ 大）直接配制标准溶液
  • 在 HCl 介质中也可使用（$\text{KMnO}_4$ 不行）
  • 标准溶液极其稳定

铁矿石中铁含量的测定（经典方法）

  1. 溶样：用 HCl 溶解铁矿石
  2. 预还原：滴加 $\text{SnCl}_2$ 还原 $\text{Fe}^{3+}$ 至 $\text{Fe}^{2+}$

    $\text{Sn}^{2+} + 2\text{Fe}^{3+} \rightarrow \text{Sn}^{4+} + 2\text{Fe}^{2+}$

    加 $\text{SnCl}_2$ 至溶液恰好变为浅黄色后，再多加 1~2 滴（不可过量太多！）
  3. 除去过量 $\text{SnCl}_2$：立即加入过量 $\text{HgCl}_2$

    $\text{Sn}^{2+} + 2\text{HgCl}_2 \rightarrow \text{Sn}^{4+} + \text{Hg}_2\text{Cl}_2\downarrow(\text{白色})$

    生成白色丝状 $\text{Hg}_2\text{Cl}_2$ 沉淀为正常。
    若无白色沉淀 → $\text{SnCl}_2$ 不足，$\text{Fe}^{3+}$ 未完全还原，弃去重做。
    若生成灰黑色沉淀 → $\text{SnCl}_2$ 过量太多，$\text{Hg}_2\text{Cl}_2$ 被还原为 Hg，会消耗 $\text{K}_2\text{Cr}_2\text{O}_7$，弃去重做。
  4. 加入 S-P 混酸（$\text{H}_2\text{SO}_4 + \text{H}_3\text{PO}_4$）
    • $\text{H}_2\text{SO}_4$：提供酸性介质
    • $\text{H}_3\text{PO}_4$：与 $\text{Fe}^{3+}$ 形成无色 $\text{Fe(HPO}_4)^+$ 配合物，消除 $\text{Fe}^{3+}$ 黄色对终点的干扰，同时降低 $\text{Fe}^{3+}/\text{Fe}^{2+}$ 的条件电位，增大突跃范围
  5. 滴定：以二苯胺磺酸钠为指示剂，用 $\text{K}_2\text{Cr}_2\text{O}_7$ 标准溶液滴定至紫色
例题8：称取铁矿石 0.5000 g，溶解后用 0.01667 mol/L $\text{K}_2\text{Cr}_2\text{O}_7$ 标准溶液滴定，消耗 35.20 mL，求铁含量（$w_{Fe}$）。
解：

$n_{Cr_2O_7^{2-}} = 0.01667 \times 35.20 \times 10^{-3} = 5.868 \times 10^{-4}\;\text{mol}$

$\text{Cr}_2\text{O}_7^{2-} + 6\text{Fe}^{2+} + 14\text{H}^+ \rightarrow 2\text{Cr}^{3+} + 6\text{Fe}^{3+} + 7\text{H}_2\text{O}$

$n_{Fe} = 6 \times n_{Cr_2O_7^{2-}} = 6 \times 5.868 \times 10^{-4} = 3.521 \times 10^{-3}\;\text{mol}$

$w_{Fe} = \frac{3.521 \times 10^{-3} \times 55.85}{0.5000} \times 100\% = 39.34\%$

例题（PPT）：$\text{K}_2\text{Cr}_2\text{O}_7$ 法测白兰地酒精含量

PPT: K2Cr2O7法应用实例

取白兰地样品 5.00 mL，稀释至 1.000 L。取稀释液 25.00 mL 蒸馏至 50.00 mL $\text{K}_2\text{Cr}_2\text{O}_7$ 标准溶液（0.02000 mol/L）中。加热氧化乙醇为乙酸：

$$3\text{C}_2\text{H}_5\text{OH} + 2\text{Cr}_2\text{O}_7^{2-} + 16\text{H}^+ \rightarrow 4\text{Cr}^{3+} + 3\text{CH}_3\text{COOH} + 11\text{H}_2\text{O}$$

冷却后加入 0.1253 mol/L $\text{FeSO}_4$ 标准溶液 20.00 mL（过量），用 7.46 mL 0.02000 mol/L $\text{K}_2\text{Cr}_2\text{O}_7$ 标准溶液返滴定至二苯胺磺酸钠终点。

解：

计量关系：$3\text{C}_2\text{H}_5\text{OH} \sim 2\text{Cr}_2\text{O}_7^{2-} \sim 12\text{Fe}^{2+} \sim 12e$

与乙醇反应的 $\text{Cr}_2\text{O}_7^{2-}$：

利用电子守恒：$n_e(\text{Cr}_2\text{O}_7^{2-},\text{total}) = n_e(\text{C}_2\text{H}_5\text{OH}) + n_e(\text{Fe}^{2+}) - n_e(\text{Cr}_2\text{O}_7^{2-},\text{back})$

$$n_{\text{C}_2\text{H}_5\text{OH}} = \frac{(6\times 0.02000\times(50.00+7.46) - 0.1253\times 20.00) \times \frac{46.07}{4}}{25.00 \times 1000 \times 5.00} \times 1000$$

最终 $w(\text{C}_2\text{H}_5\text{OH}) = 40.43\%$（v/v）。

此题综合运用了返滴定法和电子守恒。注意总反应中 1 mol 乙醇失 4 mol 电子（从 $-1$ 价碳氧化到 $0$ 价碳），所以 $3\text{C}_2\text{H}_5\text{OH} \sim 12e$。

7.9 碘量法

PPT: 碘量法的原理与分类

基本原理

$\text{I}_2 + 2e^- \rightleftharpoons 2\text{I}^-$，$\varphi^0 = 0.545$ V

碘量法利用 $\text{I}_2/\text{I}^-$ 电对的中等电位，可进行两类滴定：

直接碘滴定法（碘滴定法）

用 $\text{I}_2$ 标准溶液直接滴定还原性物质（$\varphi^{0'} < 0.545$ V 的电对）。

例：测 $\text{As}(\text{III})$, $\text{Sn}^{2+}$, $\text{SO}_3^{2-}$, 抗坏血酸等。

淀粉指示剂在开始时即可加入，终点为溶液由无色变为蓝色。

间接碘量法（滴定碘法）

用还原性物质 $\text{I}^-$（过量 KI）与氧化性物质反应，析出定量 $\text{I}_2$，再用 $\text{Na}_2\text{S}_2\text{O}_3$ 标准溶液滴定。

$\text{I}_2 + 2\text{S}_2\text{O}_3^{2-} \rightarrow 2\text{I}^- + \text{S}_4\text{O}_6^{2-}$

例：测 $\text{Cu}^{2+}$, $\text{Cr}_2\text{O}_7^{2-}$, 溶解氧, $\text{H}_2\text{O}_2$ 等。

淀粉指示剂在近终点时加入（溶液变浅黄色时），终点为蓝色褪去变无色。

$\text{Na}_2\text{S}_2\text{O}_3$ 标准溶液的配制

$\text{Na}_2\text{S}_2\text{O}_3 \cdot 5\text{H}_2\text{O}$ 不是基准物质（含结晶水量不确定，且可被细菌分解、空气氧化），需标定。

配制步骤及原因
步骤原因
使用新煮沸并冷却的蒸馏水煮沸除去水中溶解的 $\text{CO}_2$ 和 $\text{O}_2$，杀灭细菌。$\text{CO}_2$ 使溶液偏酸导致 $\text{S}_2\text{O}_3^{2-}$ 分解；$\text{O}_2$ 会氧化 $\text{S}_2\text{O}_3^{2-}$；细菌（硫杆菌）能分解 $\text{S}_2\text{O}_3^{2-}$

保存于棕色瓶中避免光照加速分解
放置 1~2 周后标定等待浓度稳定（初配制后浓度会缓慢变化）

用 $\text{K}_2\text{Cr}_2\text{O}_7$ 标定 $\text{Na}_2\text{S}_2\text{O}_3$ 的完整步骤

  1. 准确称取 $\text{K}_2\text{Cr}_2\text{O}_7$ 基准物（已于 150 °C 干燥），溶于水，移入碘量瓶
  2. 加入过量 KI（约 2 g）和 6 mol/L $\text{HCl}$ 5 mL
  3. 盖好瓶盖，在暗处放置 5 min（使反应完全）：

    $\text{Cr}_2\text{O}_7^{2-} + 6\text{I}^- + 14\text{H}^+ \rightarrow 2\text{Cr}^{3+} + 3\text{I}_2 + 7\text{H}_2\text{O}$

  4. 加水 100 mL 稀释
  5. 用 $\text{Na}_2\text{S}_2\text{O}_3$ 标准溶液滴定至溶液由棕色变为浅黄色
  6. 加入 2 mL 淀粉指示剂，继续滴定至蓝色恰好褪去即为终点

计量关系：$\text{Cr}_2\text{O}_7^{2-} \sim 3\text{I}_2 \sim 6\text{S}_2\text{O}_3^{2-}$

标定时要注意：(1) 暗处反应防止 $\text{I}^-$ 被光催化氧化；(2) 碘量瓶的水封防止 $\text{I}_2$ 挥发；(3) 淀粉在近终点时加入。

碘量法的误差来源

碘量法注意事项（系统梳理）：
误差来源影响防止措施
$\text{I}_2$ 挥发结果偏低加过量 KI（形成 $\text{I}_3^-$）；密塞；快速滴定
$\text{I}^-$ 被空气氧化
$4\text{I}^- + \text{O}_2 + 4\text{H}^+ \rightarrow 2\text{I}_2 + 2\text{H}_2\text{O}$		结果偏高避免强酸性；避免阳光直射；反应后立即滴定；不要剧烈摇动
淀粉加入过早（间接碘量法）终点拖后溶液变浅黄色时再加

碘量法测铜 -- 完整操作

反应：$2\text{Cu}^{2+} + 4\text{I}^- \rightarrow 2\text{CuI}\downarrow + \text{I}_2$

生成的 $\text{I}_2$ 用 $\text{Na}_2\text{S}_2\text{O}_3$ 滴定。

详细步骤

  1. 取含 $\text{Cu}^{2+}$ 的弱酸性溶液（pH 3~4），加入过量 KI（约 3~4 g）
  2. 加盖，在暗处放置 3~5 min
  3. 用 $\text{Na}_2\text{S}_2\text{O}_3$ 滴定至浅黄色
  4. 加 2 mL 淀粉指示剂，溶液变蓝
  5. 加入约 5 mL 10% KSCN 溶液：

    $\text{CuI} + \text{SCN}^- \rightarrow \text{CuSCN}\downarrow + \text{I}^-$

    CuSCN 比 CuI 更不溶（$K_{sp}$ 更小），从 CuI 沉淀表面释放被吸附的 $\text{I}_2$。加入 KSCN 后蓝色可能重新加深。

  6. 继续用 $\text{Na}_2\text{S}_2\text{O}_3$ 滴定至蓝色褪去即为终点
计量关系：$2\text{Cu}^{2+} \sim \text{I}_2 \sim 2\text{S}_2\text{O}_3^{2-}$，即 $n_{\text{Cu}^{2+}} = n_{\text{S}_2\text{O}_3^{2-}}$

碘量法中的放大（倍量）反应

利用某些反应中电子转移倍数不同实现信号放大。例如：

$\text{IO}_3^- + 5\text{I}^- + 6\text{H}^+ \rightarrow 3\text{I}_2 + 3\text{H}_2\text{O}$

1 mol $\text{IO}_3^-$ 产生 3 mol $\text{I}_2$，消耗 6 mol $\text{Na}_2\text{S}_2\text{O}_3$，实现 6 倍放大。

Quiz 2 Q6：K2Cr2O7 法测铁中混合酸的作用

在 $\text{K}_2\text{Cr}_2\text{O}_7$ 法测定铁含量时，滴定前加入 $\text{H}_2\text{SO}_4$-$\text{H}_3\text{PO}_4$ 混合酸，下列说法中正确的有？

A. $\text{H}_2\text{SO}_4$ 提供酸性介质   B. $\text{H}_3\text{PO}_4$ 与 $\text{Fe}^{3+}$ 配合消除黄色干扰   C. $\text{H}_3\text{PO}_4$ 降低 $\text{Fe}^{3+}/\text{Fe}^{2+}$ 条件电位   D. 使突跃范围增大

解：ABCD 全部正确
  • A. $\text{H}_2\text{SO}_4$ 提供 $\text{H}^+$，维持反应所需酸度
  • B. $\text{Fe}^{3+}$ 黄色 → $\text{Fe(HPO}_4)^+$ 无色，有利于终点判断
  • C. $\text{H}_3\text{PO}_4$ 配位氧化态 $\text{Fe}^{3+}$，$\alpha_\text{Ox}$ 增大，$\varphi^{0'}$ 降低
  • D. $\varphi^{0'}_\text{Fe}$ 降低 → $\Delta\varphi^{0'}$ 增大 → 突跃范围增大
Quiz 2 Q8：间接碘量法中淀粉指示剂的加入时机

间接碘量法（滴定碘法）中，淀粉指示剂应何时加入？

解：在近终点时（溶液变为浅黄色时）加入

原因：若过早加入，大量 $\text{I}_2$ 与淀粉形成深蓝色包合物，$\text{I}_2$ 被紧密吸附在淀粉螺旋结构内，不易被 $\text{Na}_2\text{S}_2\text{O}_3$ 释放出来，导致终点拖后，滴定误差增大。

注意：在直接碘滴定法中，淀粉指示剂可在开始时加入（因为开始时溶液中没有 $\text{I}_2$，不存在吸附问题）。

Quiz 2 Q9：Ce4+ 滴定 Fe2+ 为对称曲线的原因

$\text{Ce}^{4+}$ 滴定 $\text{Fe}^{2+}$ 的滴定曲线为对称曲线，原因是什么？

解：

因为两个电对的电子转移数相同：$n_1 = n_2 = 1$

$\text{Ce}^{4+} + e^- \rightarrow \text{Ce}^{3+}$ ($n_1 = 1$)

$\text{Fe}^{3+} + e^- \rightarrow \text{Fe}^{2+}$ ($n_2 = 1$)

当 $n_1 = n_2$ 时，$\varphi_{sp} = \frac{\varphi_1^{0'} + \varphi_2^{0'}}{2}$，化学计量点恰在突跃范围中央，曲线关于 SP 对称。

Quiz 2 计算题

Quiz 2 Calc Q3：钢中 Cr 含量的返滴定法

称取钢样 1.000 g，用酸溶解后将 Cr 氧化为 $\text{Cr}_2\text{O}_7^{2-}$，恰好转入 250 mL 容量瓶中。取 25.00 mL 试液，加入 0.1000 mol/L $\text{FeSO}_4$ 标准溶液 25.00 mL（过量），用 0.01800 mol/L $\text{KMnO}_4$ 标准溶液返滴定剩余 $\text{Fe}^{2+}$，消耗 7.00 mL。求钢中 Cr 的质量分数 $w(\text{Cr})$。

解：

第一步：写出反应方程式

$\text{Cr}_2\text{O}_7^{2-} + 6\text{Fe}^{2+} + 14\text{H}^+ \rightarrow 2\text{Cr}^{3+} + 6\text{Fe}^{3+} + 7\text{H}_2\text{O}$   ...(1)

$\text{MnO}_4^- + 5\text{Fe}^{2+} + 8\text{H}^+ \rightarrow \text{Mn}^{2+} + 5\text{Fe}^{3+} + 4\text{H}_2\text{O}$   ...(2)

第二步：计算各物质的量

$n_{\text{Fe}^{2+},\text{total}} = 0.1000 \times 25.00 \times 10^{-3} = 2.500 \times 10^{-3}\;\text{mol}$

返滴定消耗：$n_{\text{MnO}_4^-} = 0.01800 \times 7.00 \times 10^{-3} = 1.260 \times 10^{-4}\;\text{mol}$

剩余 $\text{Fe}^{2+}$：$n_{\text{Fe}^{2+},\text{excess}} = 5 \times n_{\text{MnO}_4^-} = 5 \times 1.260 \times 10^{-4} = 6.300 \times 10^{-4}\;\text{mol}$

第三步：与 $\text{Cr}_2\text{O}_7^{2-}$ 反应的 $\text{Fe}^{2+}$

$n_{\text{Fe}^{2+},\text{reacted}} = 2.500 \times 10^{-3} - 6.300 \times 10^{-4} = 1.870 \times 10^{-3}\;\text{mol}$

第四步：计算 Cr 的量（注意 25.00 mL 是 250 mL 的 1/10）

由反应(1)：$n_{\text{Cr}_2\text{O}_7^{2-}} = \frac{1}{6} \times n_{\text{Fe}^{2+},\text{reacted}} = \frac{1.870 \times 10^{-3}}{6} = 3.117 \times 10^{-4}\;\text{mol}$

$n_{\text{Cr}} = 2 \times n_{\text{Cr}_2\text{O}_7^{2-}} = 2 \times 3.117 \times 10^{-4} = 6.233 \times 10^{-4}\;\text{mol}$（取 25.00 mL 中的）

总量（250 mL）：$n_{\text{Cr},\text{total}} = 6.233 \times 10^{-4} \times 10 = 6.233 \times 10^{-3}\;\text{mol}$

$$w(\text{Cr}) = \frac{6.233 \times 10^{-3} \times 52.00}{1.000} \times 100\% = \boxed{3.24\%}$$
简洁的等量关系法（用等物质的量的电子转移）：
与 Cr 反应的 $\text{Fe}^{2+}$ 的物质的量 = $0.1000 \times \frac{25.00}{1000} - 5 \times 0.01800 \times \frac{7.00}{1000}$
$= 2.500 \times 10^{-3} - 6.300 \times 10^{-4} = 1.870 \times 10^{-3}\;\text{mol}$
$n_\text{Cr} = \frac{1}{3} \times n_{\text{Fe}^{2+}} = \frac{1.870 \times 10^{-3}}{3} = 6.233 \times 10^{-4}\;\text{mol}$
（因为 1 个 Cr 从 +6 降到 +3，失去 3 个电子；1 个 Fe2+ 失去 1 个电子）
$w(\text{Cr}) = \frac{6.233 \times 10^{-4} \times 10 \times 52.00}{1.000} = 3.24\%$

2022 Final Q3：H2SO4-H3PO4 混合酸的作用

$\text{K}_2\text{Cr}_2\text{O}_7$ 法测铁时，加入 $\text{H}_2\text{SO}_4$-$\text{H}_3\text{PO}_4$ 混合酸的目的是什么？

解：
  1. $\text{H}_2\text{SO}_4$：提供反应所需酸性介质（$\text{Cr}_2\text{O}_7^{2-}$ 需在酸性条件下氧化 $\text{Fe}^{2+}$）
  2. $\text{H}_3\text{PO}_4$ 有三重作用：
    • (a) 与生成的 $\text{Fe}^{3+}$ 配位形成 $[\text{Fe(HPO}_4)]^+$ 无色配合物，消除 $\text{Fe}^{3+}$ 的黄色对终点观察的干扰
    • (b) 配位 $\text{Fe}^{3+}$（氧化态），使 $\alpha_{\text{Fe(III)}}$ 增大，降低 $\text{Fe}^{3+}/\text{Fe}^{2+}$ 电对的条件电位
    • (c) 条件电位降低使 $\Delta\varphi^{0'}$ 增大，增大滴定突跃范围，有利于准确判定终点，可使用 $\varphi^{0'} = 0.84$ V 的二苯胺磺酸钠作指示剂
2022 Final Q18：KMnO4 法为什么不能用 HCl？

用 $\text{KMnO}_4$ 标准溶液滴定 $\text{C}_2\text{O}_4^{2-}$ 时，为什么不能用 HCl 作酸性介质？

解：

两个原因：

  1. 直接氧化：$\text{KMnO}_4$ 可氧化 HCl 中的 $\text{Cl}^-$：$2\text{MnO}_4^- + 10\text{Cl}^- + 16\text{H}^+ \rightarrow 2\text{Mn}^{2+} + 5\text{Cl}_2 + 8\text{H}_2\text{O}$，消耗额外的 $\text{KMnO}_4$，使结果偏高
  2. 诱导反应：$\text{MnO}_4^-$ 氧化 $\text{C}_2\text{O}_4^{2-}$ 的反应会诱导 $\text{Cl}^-$ 被氧化（即所谓的诱导效应），即使在动力学上 $\text{Cl}^-$ 单独被 $\text{MnO}_4^-$ 氧化的速度较慢，在诱导反应的条件下也会显著发生，进一步消耗 $\text{KMnO}_4$

因此 $\text{KMnO}_4$ 法只能使用 $\text{H}_2\text{SO}_4$。

2022 Final Q19：Ce4+ 滴定 Fe2+ 至 50% 时的电位

在 1 mol/L $\text{H}_2\text{SO}_4$ 中，用 $\text{Ce}^{4+}$（$\varphi^{0'} = 1.44$ V）滴定 $\text{Fe}^{2+}$（$\varphi^{0'} = 0.68$ V），当滴定至 50% 时，溶液电位为多少？

解：

滴定 50% 时，$\text{Fe}^{2+}$ 被氧化了一半：$[\text{Fe}^{3+}] = [\text{Fe}^{2+}]$

$$\varphi = \varphi^{0'}_{\text{Fe}^{3+}/\text{Fe}^{2+}} + 0.0592\lg\frac{[\text{Fe}^{3+}]}{[\text{Fe}^{2+}]} = 0.68 + 0.0592\lg 1 = \boxed{0.68\;\text{V}}$$

化学计量点前的电位由被滴定物的电对控制。50% 时恰好 $[\text{Ox}] = [\text{Red}]$，$\lg$ 项为零，电位等于该电对的条件电位。

2022 Final Calc Q3：条件电位计算（含 F- 配位）

已知 $\text{Fe}^{3+}/\text{Fe}^{2+}$ 电对 $\varphi^0 = 0.771$ V。在含 $\text{F}^-$ 的溶液中，$\text{Fe}^{3+}$ 与 $\text{F}^-$ 的累积稳定常数 $\lg\beta_1=5.2$, $\lg\beta_2=9.2$, $\lg\beta_3=11.9$。$\text{Fe}^{2+}$ 基本不与 $\text{F}^-$ 配位。若 $[\text{F}^-] = 10^{-1.5}$ mol/L，求条件电位。

解：

$\alpha_{\text{Fe(III)}(\text{F})} = 1 + \beta_1[\text{F}^-] + \beta_2[\text{F}^-]^2 + \beta_3[\text{F}^-]^3$

$= 1 + 10^{5.2-1.5} + 10^{9.2-3.0} + 10^{11.9-4.5}$

$= 1 + 10^{3.7} + 10^{6.2} + 10^{7.4} \approx 10^{7.4}$

$\alpha_{\text{Fe(II)}} = 1$

$$\varphi^{0'} = 0.771 + 0.0592\lg\frac{1}{10^{7.4}} = 0.771 - 0.0592 \times 7.4 = 0.771 - 0.438 = \boxed{0.333\;\text{V}}$$

$\text{F}^-$ 配位使电位大幅降低约 0.44 V。

例题9：MnO4- 滴定 Fe2+ 的化学计量点电位

在 $\text{H}_2\text{SO}_4$ 介质中用 $\text{KMnO}_4$（$\varphi^{0'} = 1.51$ V）滴定 $\text{Fe}^{2+}$（$\varphi^{0'} = 0.68$ V），求 $\varphi_{sp}$ 并说明其特征。

解：

$\text{MnO}_4^- + 5\text{Fe}^{2+} + 8\text{H}^+ \rightarrow \text{Mn}^{2+} + 5\text{Fe}^{3+} + 4\text{H}_2\text{O}$

$n_1 = 5$（MnO4-），$n_2 = 1$（Fe2+）

$$\varphi_{sp} = \frac{5 \times 1.51 + 1 \times 0.68}{5 + 1} = \frac{8.23}{6} = 1.37\;\text{V}$$

$\varphi_{sp} = 1.37$ V 明显偏向高电位端（$\text{MnO}_4^-$ 一侧）。曲线不对称，$n_1 > n_2$ 故偏向 $\varphi_1^{0'}$。

注意：$\text{KMnO}_4$ 为自身指示剂，不需额外加指示剂。

例题10：Na2S2O3 浓度的标定计算

准确称取 $\text{K}_2\text{Cr}_2\text{O}_7$ 基准物 0.1500 g，溶解后加入过量 KI 和 HCl，暗处反应后用 $\text{Na}_2\text{S}_2\text{O}_3$ 溶液滴定，消耗 30.50 mL。求 $\text{Na}_2\text{S}_2\text{O}_3$ 的浓度。（$M_{\text{K}_2\text{Cr}_2\text{O}_7} = 294.2$ g/mol）

$$c_{\text{Na}_2\text{S}_2\text{O}_3} = \frac{3.059 \times 10^{-3}}{30.50 \times 10^{-3}} = \boxed{0.1003\;\text{mol/L}}$$
