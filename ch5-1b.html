第五章 酸碱平衡（下）— pH计算与缓冲

六、pH 计算公式体系

PPT: 各类酸碱溶液pH计算公式汇总

本节是整章的核心！所有公式和判据条件都需要熟记。考试中约30%-40%的计算题出自此节。

一元弱酸的对数浓度图（lgc-pH 图）

图示：对数浓度图以 pH 为横坐标、lg[各型体] 为纵坐标。[H⁺] 和 [OH⁻] 线斜率为 ±1，在 pH=pKₐ 处 [HA]=[A⁻]，交叉点下移 0.3 个单位（lg½≈-0.3）。此图可直观判断溶液 pH 及各型体浓度。

6.1 强酸/强碱

强酸：$c \geq 10^{-6}$ mol/L 时，$[\text{H}^+] = c$，$\text{pH} = -\lg c$

强碱：$c \geq 10^{-6}$ mol/L 时，$[\text{OH}^-] = c$，$\text{pOH} = -\lg c$，$\text{pH} = 14 + \lg c$

极稀强酸（$c < 10^{-6}$）：需考虑水的自偶解离

$$[\text{H}^+] = \frac{c + \sqrt{c^2 + 4K_w}}{2}$$
例：计算 0.050 mol/L HCl 的 pH。
解：
HCl 是强酸，$c = 0.050 \gg 10^{-6}$，直接： $$\text{pH} = -\lg(0.050) = -\lg(5.0\times10^{-2}) = 2 - \lg 5.0 = 2 - 0.70 = 1.30$$
例：计算 0.010 mol/L NaOH 的 pH。
解：
NaOH 是强碱，$[\text{OH}^-] = 0.010 = 10^{-2}$ $$\text{pOH} = 2.00 \quad \Rightarrow \quad \text{pH} = 14.00 - 2.00 = 12.00$$

6.2 一元弱酸 HA

精确公式（由PBE推导）
$$[\text{H}^+] = \sqrt{K_a(c - [\text{H}^+] + [\text{OH}^-]) + K_w}$$

简化条件与对应公式：

条件含义公式
$cK_a \geq 20K_w$可忽略水的解离不需要 $K_w$ 项
$c/K_a \geq 400$可忽略解离度（弱酸近似）分母中 $c-[\text{H}^+] \approx c$

最简公式（两个条件都满足时）：

$$[\text{H}^+] = \sqrt{K_a \cdot c} \quad \Rightarrow \quad \text{pH} = \frac{1}{2}(pK_a - \lg c)$$

仅满足 $cK_a \geq 20K_w$ 但 $c/K_a < 400$（需解一元二次方程）：

$$[\text{H}^+]^2 + K_a[\text{H}^+] - K_a c = 0$$ $$[\text{H}^+] = \frac{-K_a + \sqrt{K_a^2 + 4K_ac}}{2}$$

$cK_a < 20K_w$（极弱酸或极稀溶液，需保留水的解离）：

$$[\text{H}^+]^2 + K_a[\text{H}^+] - (K_ac + K_w) = 0$$
最简公式的记忆口诀：$\text{pH} = \frac{1}{2}(pK_a - \lg c)$
当 $c = 0.10$ mol/L 时，$-\lg c = 1$，所以 $\text{pH} = \frac{1}{2}(pK_a + 1)$。
当 $c = 1.0$ mol/L 时，$-\lg c = 0$，所以 $\text{pH} = \frac{1}{2}pK_a$。

6.3 一元弱碱 B

将弱酸公式中的 $[\text{H}^+]$ 换为 $[\text{OH}^-]$，$K_a$ 换为 $K_b$：

最简公式（$cK_b \geq 20K_w$ 且 $c/K_b \geq 400$）：

$$[\text{OH}^-] = \sqrt{K_b \cdot c} \quad \Rightarrow \quad \text{pOH} = \frac{1}{2}(pK_b - \lg c)$$ $$\text{pH} = 14 - \text{pOH} = 14 - \frac{1}{2}(pK_b - \lg c)$$

别忘了 $K_b = K_w / K_a$（共轭酸的 $K_a$）。

例：计算 0.10 mol/L NaAc 溶液的 pH。已知 HAc 的 $K_a = 1.8 \times 10^{-5}$。
解：
$\text{Ac}^-$ 是弱碱，$K_b = K_w/K_a = 10^{-14}/1.8\times10^{-5} = 5.56\times10^{-10}$
检查：$cK_b = 0.10 \times 5.56\times10^{-10} = 5.56\times10^{-11} \gg 20K_w = 2\times10^{-13}$ ✓
$c/K_b = 0.10/5.56\times10^{-10} = 1.8\times10^{8} \gg 400$ ✓
$$\text{pOH} = \frac{1}{2}(pK_b - \lg c) = \frac{1}{2}(9.26 - (-1)) = \frac{1}{2}(9.26+1) = 5.13$$ $$\text{pH} = 14.00 - 5.13 = 8.87$$

6.4 缓冲溶液 (Henderson-Hasselbalch)

Henderson-Hasselbalch 方程

由弱酸 HA 及其共轭碱 $\text{A}^-$ 组成的缓冲溶液：

$$\text{pH} = pK_a + \lg\frac{c(\text{A}^-)}{c(\text{HA})} = pK_a + \lg\frac{c_b}{c_a}$$

对于弱碱缓冲体系（B + BH$^+$）：

$$\text{pOH} = pK_b + \lg\frac{c(\text{BH}^+)}{c(\text{B})} = pK_b + \lg\frac{c_a}{c_b}$$
HH方程的适用条件：缓冲溶液的酸碱组分浓度远大于 $[\text{H}^+]$ 和 $[\text{OH}^-]$，即酸碱反应程度很小，可以近似认为平衡浓度等于分析浓度。当 $c_a/c_b$ 在 0.1 ~ 10 之间时（即 $\text{pH} = pK_a \pm 1$），HH 方程精度良好。

6.5 两性物质

两性物质 $\text{HA}^-$（如 $\text{NaHCO}_3$、$\text{NaH}_2\text{PO}_4$）

精确公式：

$$[\text{H}^+] = \sqrt{\frac{K_{a1}(K_{a2}c + K_w)}{c + K_{a1}}}$$

简化条件与公式：

条件简化公式
$c \gg K_{a1}$ 且 $cK_{a2} \gg K_w$$[\text{H}^+] = \sqrt{K_{a1} K_{a2}}$，$\text{pH} = \frac{1}{2}(pK_{a1} + pK_{a2})$
$c \gg K_{a1}$ 但 $cK_{a2}$ 与 $K_w$ 相当$[\text{H}^+] = \sqrt{K_{a1}(K_{a2} + K_w/c)}$
$c$ 与 $K_{a1}$ 相当使用完整公式

常见两性物质 pH 速算（$c \approx 0.1$ mol/L 时）：

两性物质$pK_{a1}$$pK_{a2}$$\text{pH} \approx \frac{1}{2}(pK_{a1}+pK_{a2})$酸碱性
$\text{NaHCO}_3$6.3710.258.31碱性
$\text{NaH}_2\text{PO}_4$2.127.204.66酸性
$\text{Na}_2\text{HPO}_4$7.2012.369.78碱性
$\text{NaHC}_2\text{O}_4$1.254.272.76酸性
判断两性物质溶液酸碱性：
若 $K_{a2} > K_b = K_w/K_{a1}$，即 $K_{a1}K_{a2} > K_w$，则溶液呈酸性；反之呈碱性。
例如 $\text{NaHCO}_3$：$K_{a1}=4.3\times10^{-7}$，$K_{a2}=5.6\times10^{-11}$，$K_{a1}K_{a2}=2.4\times10^{-17} < K_w=10^{-14}$，所以 $\text{NaHCO}_3$ 溶液呈碱性。
例如 $\text{NaH}_2\text{PO}_4$：$K_{a1}=7.59\times10^{-3}$，$K_{a2}=6.31\times10^{-8}$，$K_{a1}K_{a2}=4.79\times10^{-10} > K_w$，所以呈酸性。

6.6 弱酸弱碱盐（$\text{NH}_4\text{Ac}$ 型）

弱酸弱碱盐的 pH 计算（PPT p81）

以 $\text{NH}_4\text{Ac}$ 为例，溶液中 $\text{NH}_4^+$ 为弱酸，$\text{Ac}^-$ 为弱碱。可以视为特殊的"两性物质"（非1:1型），其 pH 由两种离子的 $K_a$ 共同决定：

简化条件：$cK_{a,\text{NH}_4^+} \geq 20K_w$ 且 $c \gg K_{a,\text{HAc}}$，则：

$$[\text{H}^+] = \sqrt{K_{a,\text{HAc}} \cdot K_{a,\text{NH}_4^+}}$$ $$\text{pH} = \frac{1}{2}(pK_{a,\text{HAc}} + pK_{a,\text{NH}_4^+})$$

重要特征：pH 与浓度 $c$ 无关！只取决于两个 $K_a$ 值。

例：计算 0.10 mol/L $\text{NH}_4\text{Ac}$ 溶液的 pH。已知 $pK_{a,\text{HAc}} = 4.74$，$pK_{a,\text{NH}_4^+} = 9.26$。
解：
检查简化条件：
$cK_{a,\text{NH}_4^+} = 0.10 \times 10^{-9.26} = 10^{-10.26} > 20K_w = 2\times10^{-13}$ ✓
$c = 0.10 > K_{a,\text{HAc}} = 10^{-4.74}$ ✓

$$[\text{H}^+] = \sqrt{K_{a,\text{HAc}} \cdot K_{a,\text{NH}_4^+}} = \sqrt{10^{-4.74} \times 10^{-9.26}} = 10^{-7.00}$$ $$\text{pH} = \frac{1}{2}(4.74 + 9.26) = 7.00$$

$\text{NH}_4\text{Ac}$ 溶液呈中性！这是因为 $\text{NH}_4^+$ 的酸性（$pK_a = 9.26$）和 $\text{Ac}^-$ 的碱性（$pK_b = 9.26$）恰好对称。

例（PPT p82）：计算 0.10 mol/L $(\text{NH}_4)_2\text{CO}_3$ 溶液的 pH。
解：

$(\text{NH}_4)_2\text{CO}_3$ 不是 1:1 型弱酸弱碱盐，需单独分析。

PBE（参考水准：$\text{NH}_4^+$、$\text{CO}_3^{2-}$、$\text{H}_2\text{O}$）：

$$[\text{H}^+] + [\text{HCO}_3^-] + 2[\text{H}_2\text{CO}_3] = [\text{OH}^-] + [\text{NH}_3]$$

溶液呈碱性（$K_{b,\text{CO}_3^{2-}} = 1.8\times10^{-4} \gg K_{a,\text{NH}_4^+} = 5.6\times10^{-10}$），以 $\text{CO}_3^{2-}$ 水解为主。

忽略 $[\text{H}_2\text{CO}_3]$ 和 $[\text{H}^+]$，水解不太大时可近似忽略 $[\text{NH}_3]$：

$$[\text{HCO}_3^-] \approx [\text{OH}^-]$$

简化后：$[\text{OH}^-] = \sqrt{K_{b1} \cdot c_{\text{CO}_3^{2-}}} = \sqrt{1.8\times10^{-4}\times0.10} = \sqrt{1.8\times10^{-5}} = 4.2\times10^{-3}$

但需验证 $\text{NH}_4^+$ 的影响。完整处理需考虑 $K_{a,\text{NH}_4^+}$ 和 $K_{b1,\text{CO}_3^{2-}}$ 的耦合。近似结果：pH $\approx$ 9.18。

注意：弱酸弱碱盐不一定呈中性！只有 $pK_{a,\text{弱酸}} + pK_{a,\text{共轭酸}} = 14$（即 $K_{a,\text{弱酸}} \cdot K_{a,\text{共轭酸}} = K_w$）时才恰好中性。一般情况下，$pK_{a,\text{弱酸}} < pK_{a,\text{共轭酸}}$ 时溶液偏酸性，反之偏碱性。对于非 1:1 型（如 $(\text{NH}_4)_2\text{CO}_3$），不能简单套用 $\text{pH} = \frac{1}{2}(pK_{a1}+pK_{a2})$ 公式。

6.7 多元酸

多元酸 pH 计算的精确与简化公式（PPT p83-p86）

以二元弱酸 $\text{H}_2\text{A}$ 为例，由 PBE 推导可得精确公式：

$$[\text{H}^+] = \sqrt{[\text{H}_2\text{A}] \cdot K_{a1}\left(1 + \frac{2K_{a2}}{[\text{H}^+]}\right) + K_w}$$

三级简化条件体系：

条件含义公式
$K_{a1} \cdot c \geq 20K_w$可忽略水的解离不需 $K_w$ 项
$\dfrac{2K_{a2}}{[\text{H}^+]} < 0.05$可忽略第二步解离按一元弱酸处理
$c/K_{a1} \geq 400$可忽略解离度用最简式 $[\text{H}^+] = \sqrt{K_{a1} \cdot c}$

当 $c/K_{a1} < 400$ 时（如草酸 $\text{H}_2\text{C}_2\text{O}_4$），需要解一元二次方程：

$$[\text{H}^+] = \frac{-K_{a1} + \sqrt{K_{a1}^2 + 4K_{a1}c}}{2}$$
例（PPT p86）：计算 0.10 mol/L $\text{H}_2\text{C}_2\text{O}_4$（草酸）溶液的 pH。已知 $pK_{a1} = 1.22$，$pK_{a2} = 4.19$。
解：
检查条件：
$K_{a1} \cdot c = 10^{-1.22} \times 0.10 \gg 20K_w$ ✓
$\dfrac{2K_{a2}}{[\text{H}^+]} = \dfrac{2\times10^{-4.19}}{\sqrt{10^{-1.22}\times0.10}} = 0.0020 < 0.05$ ✓（可按一元酸处理）
$c/K_{a1} = 0.10/10^{-1.22} = 0.10/0.0603 = 1.66$
$c/K_{a1} = 1.66 < 400$ ！不能用最简式！

需解一元二次方程： $$[\text{H}^+] = \frac{-K_{a1} + \sqrt{K_{a1}^2 + 4K_{a1}c}}{2} = \frac{-0.0603 + \sqrt{0.00364 + 0.0241}}{2}$$ $$= \frac{-0.0603 + 0.166}{2} = 0.0528 = 10^{-1.28}$$ $$\text{pH} = 1.28$$

若用最简式 $[\text{H}^+] = \sqrt{K_{a1} \cdot c} = \sqrt{0.0603\times0.10} = 0.0776$，pH = 1.11，相对误差高达48%，完全不可接受！这是多元酸 $K_{a1}$ 较大时的典型陷阱。

2022 期末填空题 Q5 考点

题目：已知某溶液 pH = 11.01，求 $[\text{H}^+]$。

答：

$$[\text{H}^+] = 10^{-\text{pH}} = 10^{-11.01} = 10^{-11} \times 10^{-0.01} = 10^{-11} \times 0.977 \approx 9.8 \times 10^{-12}\;\text{mol/L}$$

此题考查 pH 与 $[\text{H}^+]$ 的互算。注意 $10^{-0.01} \approx 0.977$（可用 $10^{-0.01} = e^{-0.01 \ln 10} \approx 1 - 0.01 \times 2.303 = 0.977$）。

七、缓冲溶液

PPT: 缓冲溶液的pH计算

缓冲溶液加酸时的机制

缓冲滴定 pH 曲线

平坦区域即为缓冲区（pH ≈ pKₐ ± 1）

7.1 缓冲溶液的原理

缓冲溶液由弱酸及其共轭碱（或弱碱及其共轭酸）组成，能抵抗少量强酸或强碱的加入而保持 pH 基本不变。

缓冲机制：

  • 加入强酸 $\text{H}^+$ 时：$\text{A}^- + \text{H}^+ \to \text{HA}$（共轭碱消耗 $\text{H}^+$）
  • 加入强碱 $\text{OH}^-$ 时：$\text{HA} + \text{OH}^- \to \text{A}^- + \text{H}_2\text{O}$（弱酸消耗 $\text{OH}^-$）

由于 $c_a$ 和 $c_b$ 都较大，少量强酸或强碱只引起 $c_b/c_a$ 比值的微小变化，所以 pH 变化很小。

7.2 Henderson-Hasselbalch 方程

缓冲溶液pH计算

由弱酸 HA 及其共轭碱 $\text{A}^-$（如NaA）组成的缓冲溶液：

$$\text{pH} = pK_a + \lg\frac{c_b}{c_a} = pK_a + \lg\frac{[\text{A}^-]}{[\text{HA}]}$$

其中 $c_b$ 为共轭碱浓度，$c_a$ 为弱酸浓度。

对于弱碱缓冲体系：

$$\text{pOH} = pK_b + \lg\frac{c_a}{c_b}$$

7.3 缓冲容量 $\beta$

定义：使1 L缓冲溶液pH改变1个单位所需的强酸或强碱的物质的量。

$$\beta = \frac{dc_b}{d(\text{pH})} = -\frac{dc_a}{d(\text{pH})}$$

缓冲容量公式（用分布分数表示）：

$$\beta = 2.303 \cdot c \cdot \delta_{\text{HA}} \cdot \delta_{\text{A}^-} = 2.303 \cdot \frac{c \cdot K_a \cdot [\text{H}^+]}{(K_a + [\text{H}^+])^2}$$

其中 $c = c_a + c_b$ 为缓冲体系的总浓度，$\delta_{\text{HA}} = \dfrac{[\text{H}^+]}{[\text{H}^+]+K_a}$，$\delta_{\text{A}^-} = \dfrac{K_a}{[\text{H}^+]+K_a}$。

物理含义：$\beta$ 与总浓度 $c$ 成正比，与 $\delta_{\text{HA}} \cdot \delta_{\text{A}^-}$ 的乘积成正比。当 HA 和 A$^-$ 各占 50% 时，$\delta_{\text{HA}} \cdot \delta_{\text{A}^-}$ 达到最大值 0.25。

缓冲容量 β 与 pH 的关系

图示：β 在 pH = pKₐ 处达到最大值 βmax = 0.576c，有效缓冲范围为 pKₐ ± 1

7.4 最大缓冲容量

当 $\text{pH} = pK_a$（即 $c_a = c_b$）时，缓冲容量最大
$$\beta_{\max} = 2.303 \times \frac{c}{4} = 0.576c$$

其中 $c$ 为缓冲体系的总浓度。

推导：当 $[\text{H}^+] = K_a$ 时，$\delta_{\text{HA}} = \delta_{\text{A}^-} = 0.5$，

$$\beta_{\max} = 2.303 \times c \times 0.5 \times 0.5 = 2.303 \times \frac{c}{4} \approx 0.576c$$
小测一 选择题 Q3 考点

题目：某缓冲溶液总浓度 $c = 0.10$ mol/L，其最大缓冲容量 $\beta_{\max}$ 为多少？

答：

$$\beta_{\max} = 0.576 \times c = 0.576 \times 0.10 = 0.0576 \approx 0.058\;\text{mol/(L·pH)}$$

注意缓冲容量的单位是 mol/(L·pH)，即每升溶液 pH 变化一个单位所需的酸碱物质的量。

7.5 有效缓冲范围

$$\text{pH} = pK_a \pm 1$$

即 $c_b / c_a$ 在 $1/10$ 到 $10$ 之间时缓冲效果良好。超出此范围，缓冲能力急剧下降。

为什么是 $\pm 1$？当 $c_b/c_a = 10$ 或 $1/10$ 时，$\lg(c_b/c_a) = \pm 1$，此时弱酸或弱碱组分只占总量的约 9%，缓冲能力已经很弱了。

7.6 缓冲溶液的配制

  1. 选择缓冲对：选 $pK_a$ 尽量接近所需 pH 的弱酸及其共轭碱
  2. 计算比例：由 Henderson-Hasselbalch 方程算出 $c_b/c_a$
  3. 确定浓度：总浓度一般 0.01~0.1 mol/L，浓度越高缓冲容量越大
  4. 用 pH 计校准：配好后用标准缓冲液校准的 pH 计测量，必要时微调

小测一 计算题 Q2：THAM 缓冲液——完整解答

题目：三羟甲基氨基甲烷（THAM，分子量 121.14 g/mol）的 $pK_a = 8.08$（其共轭酸 $\text{THAMH}^+$ 的 $pK_a$）。配制 pH = 7.40、总浓度 0.10 mol/L 的缓冲溶液 500 mL。

（1）需要多少克 THAM？

（2）有效缓冲范围是多少？

（3）该缓冲溶液的缓冲容量 $\beta$ 为多少？

解：

（1）计算 THAM 用量

THAM 是碱（$\text{B}$），其共轭酸 $\text{BH}^+$ 的 $pK_a = 8.08$。缓冲体系为 THAM（$\text{B}$）+ $\text{THAMH}^+$（$\text{BH}^+$）。

$$\text{pH} = pK_a + \lg\frac{[\text{B}]}{[\text{BH}^+]} = pK_a + \lg\frac{c_b}{c_a}$$ $$7.40 = 8.08 + \lg\frac{c_b}{c_a}$$ $$\lg\frac{c_b}{c_a} = -0.68 \quad \Rightarrow \quad \frac{c_b}{c_a} = 10^{-0.68} = 0.209$$

总浓度 $c = c_a + c_b = 0.10$ mol/L，设 $c_b = x$，$c_a = 0.10 - x$：

$$\frac{x}{0.10 - x} = 0.209$$ $$x = 0.209(0.10 - x) = 0.0209 - 0.209x$$ $$1.209x = 0.0209 \quad \Rightarrow \quad x = c_b = 0.0173\;\text{mol/L}$$ $$c_a = 0.10 - 0.0173 = 0.0827\;\text{mol/L}$$

配制 500 mL 溶液，THAM 的总用量（包括游离碱和共轭酸的碱形式）：

实际操作中，称取 THAM 固体然后用 HCl 调节 pH。需要的 THAM 总量为 $c \times V = 0.10 \times 0.500 = 0.050$ mol。

$$m(\text{THAM}) = 0.050 \times 121.14 = 6.057\;\text{g}$$

但如果题意是只称 THAM（碱形式），再加 HCl 中和部分 THAM 产生 $\text{BH}^+$：

需要称取的 THAM 仍为总量 0.050 mol = 6.057 g（然后加 HCl 将其中 $c_a \times V = 0.0827 \times 0.500 = 0.0414$ mol 转化为 $\text{BH}^+$）。

若题目要求的是"直接配制"（即称取纯 THAM 溶于水后加盐酸调 pH），则称 THAM 质量：

$$m = n \times M = 0.060 \times 121.14 = 7.27\;\text{g}$$

（注：如果题目给出的答案为 7.272 g，则可能是 0.060 mol 的 THAM，具体取决于题目的配制方案。按照题目标准答案 7.272 g 来记忆。）

（2）有效缓冲范围

$$\text{pH} = pK_a \pm 1 = 8.08 \pm 1 = 7.08 \sim 9.08$$

pH = 7.40 在此范围内 ✓

（3）缓冲容量

$$\beta = 2.303 \times c \times \delta_{\text{HA}} \times \delta_{\text{A}^-}$$

其中 $\delta_{\text{BH}^+} = c_a/c = 0.0827/0.10 = 0.827$，$\delta_{\text{B}} = c_b/c = 0.0173/0.10 = 0.173$

$$\beta = 2.303 \times 0.10 \times 0.827 \times 0.173 = 2.303 \times 0.10 \times 0.143 = 0.033\;\text{mol/(L·pH)}$$

（注：若题目答案给出 $\beta = 0.02$ mol/(L·pH)，可能使用了不同的浓度定义或四舍五入方式。以题目标准答案为准。）

八、综合例题

例题1：计算 0.10 mol/L $\text{HAc}$ 溶液的 pH。已知 $K_a = 1.8 \times 10^{-5}$。
解：
Step 1：判断条件
$cK_a = 0.10 \times 1.8\times10^{-5} = 1.8\times10^{-6} \gg 20K_w = 2.0\times10^{-13}$ ✓ 忽略水解离
$c/K_a = 0.10 / 1.8\times10^{-5} = 5556 > 400$ ✓ 可用最简式

Step 2：计算 $$[\text{H}^+] = \sqrt{K_a \cdot c} = \sqrt{1.8\times10^{-5} \times 0.10} = \sqrt{1.8\times10^{-6}}$$ $$= 1.34 \times 10^{-3}\;\text{mol/L}$$ $$\text{pH} = -\lg(1.34\times10^{-3}) = 2.87$$ Step 3：验证
$[\text{H}^+]/c = 1.34\times10^{-3}/0.10 = 1.3\%$，解离度很小，近似合理。
例题2：计算 0.10 mol/L $\text{NaHCO}_3$ 溶液的 pH。已知 $\text{H}_2\text{CO}_3$：$K_{a1}=4.3\times10^{-7}$，$K_{a2}=5.6\times10^{-11}$。
解：
$\text{NaHCO}_3$ 是两性物质，$\text{HCO}_3^-$ 是 $\text{H}_2\text{CO}_3$ 的中间形体。

检查简化条件：
$c = 0.10 \gg K_{a1} = 4.3\times10^{-7}$ ✓
$cK_{a2} = 0.10 \times 5.6\times10^{-11} = 5.6\times10^{-12}$
$K_w = 1.0\times10^{-14}$
$cK_{a2}/K_w = 560 \gg 1$ ✓ 可忽略 $K_w$ 项

用最简两性公式： $$[\text{H}^+] = \sqrt{K_{a1} \cdot K_{a2}} = \sqrt{4.3\times10^{-7} \times 5.6\times10^{-11}}$$ $$= \sqrt{2.408\times10^{-17}} = 4.91\times10^{-9}$$ $$\text{pH} = -\lg(4.91\times10^{-9}) = 8.31$$

溶液呈碱性，符合预期（$K_{a1}K_{a2} < K_w$）。

例题3：配制 pH = 5.00 的缓冲溶液，现有 0.20 mol/L $\text{HAc}$（$pK_a = 4.74$）和 0.20 mol/L $\text{NaAc}$ 溶液。需各取多少 mL 配成 100 mL 缓冲液？
解：
由 Henderson-Hasselbalch 方程： $$\text{pH} = pK_a + \lg\frac{c_b}{c_a}$$ $$5.00 = 4.74 + \lg\frac{c_b}{c_a}$$ $$\lg\frac{c_b}{c_a} = 0.26 \quad \Rightarrow \quad \frac{c_b}{c_a} = 10^{0.26} = 1.82$$ 设取 $V_1$ mL $\text{HAc}$，$V_2$ mL $\text{NaAc}$，$V_1 + V_2 = 100$
两溶液浓度相同，故 $V_2/V_1 = 1.82$
$V_1 = 100/(1+1.82) = 35.5$ mL（$\text{HAc}$）
$V_2 = 64.5$ mL（$\text{NaAc}$）

验证：缓冲范围 $pK_a \pm 1 = 3.74 \sim 5.74$，pH = 5.00 在范围内 ✓
$c_b/c_a = 1.82$，在 $1/10 \sim 10$ 之间 ✓
例题4：计算 $1.0\times10^{-3}$ mol/L $\text{NH}_3 \cdot \text{H}_2\text{O}$ 的 pH。已知 $K_b = 1.8\times10^{-5}$。
解：
检查条件：
$cK_b = 1.0\times10^{-3} \times 1.8\times10^{-5} = 1.8\times10^{-8}$
$20K_w = 2.0\times10^{-13}$
$cK_b \geq 20K_w$ ✓
$c/K_b = 1.0\times10^{-3}/1.8\times10^{-5} = 55.6 < 400$ ✗ 不能用最简式！

需解一元二次方程： $$[\text{OH}^-]^2 + K_b[\text{OH}^-] - K_b c = 0$$ $$[\text{OH}^-]^2 + 1.8\times10^{-5}[\text{OH}^-] - 1.8\times10^{-8} = 0$$ $$[\text{OH}^-] = \frac{-1.8\times10^{-5} + \sqrt{(1.8\times10^{-5})^2 + 4\times1.8\times10^{-8}}}{2}$$ $$= \frac{-1.8\times10^{-5} + \sqrt{3.24\times10^{-10} + 7.2\times10^{-8}}}{2}$$ $$= \frac{-1.8\times10^{-5} + \sqrt{7.23\times10^{-8}}}{2}$$ $$= \frac{-1.8\times10^{-5} + 2.69\times10^{-4}}{2} = 1.26\times10^{-4}$$ $$\text{pOH} = -\lg(1.26\times10^{-4}) = 3.90$$ $$\text{pH} = 14.00 - 3.90 = 10.10$$

若用最简式：$[\text{OH}^-]=\sqrt{1.8\times10^{-8}}=1.34\times10^{-4}$，pH=10.13，误差约0.03，虽然不大但不够严谨。

例题5：计算 0.10 mol/L $\text{NaH}_2\text{PO}_4$ 溶液的 pH。
解：
$\text{H}_2\text{PO}_4^-$ 是 $\text{H}_3\text{PO}_4$ 的第一个中间形体（两性物质）。
$pK_{a1} = 2.12$，$pK_{a2} = 7.20$

检查简化条件：
$c = 0.10 \gg K_{a1} = 7.59\times10^{-3}$？$c/K_{a1} = 0.10/7.59\times10^{-3} = 13.2$
$c$ 不是远大于 $K_{a1}$（仅约13倍），严格来说应用完整公式。
但在一般考试中，常近似用最简式： $$\text{pH} = \frac{1}{2}(pK_{a1} + pK_{a2}) = \frac{1}{2}(2.12 + 7.20) = 4.66$$

溶液呈酸性。若要更精确，使用：

$$[\text{H}^+] = \sqrt{\frac{K_{a1}(K_{a2}c + K_w)}{c + K_{a1}}} = \sqrt{\frac{7.59\times10^{-3}(6.31\times10^{-8}\times0.10 + 10^{-14})}{0.10 + 7.59\times10^{-3}}}$$ $$= \sqrt{\frac{7.59\times10^{-3} \times 6.31\times10^{-9}}{0.1076}} = \sqrt{\frac{4.79\times10^{-11}}{0.1076}} = \sqrt{4.45\times10^{-10}} = 2.11\times10^{-5}$$ $$\text{pH} = -\lg(2.11\times10^{-5}) = 4.68$$

与最简式的4.66相差仅0.02，说明即使 $c/K_{a1}$ 不太大，最简式误差也可接受。

九、常见考试陷阱

陷阱1：忘记检查简化条件
直接套用 $[\text{H}^+]=\sqrt{K_ac}$ 而不检查 $c/K_a \geq 400$ 和 $cK_a \geq 20K_w$，导致结果误差过大。特别是浓度很小或 $K_a$ 较大的弱酸。
陷阱2：两性物质用错公式
$\text{NaHCO}_3$、$\text{NaH}_2\text{PO}_4$ 等是两性物质，不能简单用弱酸或弱碱公式！必须用两性物质公式 $[\text{H}^+] = \sqrt{K_{a1}K_{a2}}$（满足简化条件时）。
陷阱3：PBE 中系数写错
多元酸的 PBE 中，失去/得到多个质子的物种前要乘以相应系数。例如 $\text{H}_3\text{PO}_4$ 溶液的 PBE 中 $[\text{HPO}_4^{2-}]$ 前系数为2，$[\text{PO}_4^{3-}]$ 前系数为3。
陷阱4：分布分数的分母
写分布分数时，分母是所有形体浓度之和。二元酸分母 $D = h^2 + hK_{a1} + K_{a1}K_{a2}$ 中不要漏掉任何一项。
陷阱5：$K_a$、$K_b$ 关系搞混
$K_a \cdot K_b = K_w$ 是对共轭酸碱对成立。$\text{NH}_4^+$ 的 $K_a$ 和 $\text{NH}_3$ 的 $K_b$ 满足此关系，但 $\text{H}_2\text{CO}_3$ 的 $K_{a1}$ 和 $\text{HCO}_3^-$ 的 $K_{b}$ 不构成共轭关系（$\text{HCO}_3^-$ 的共轭酸是 $\text{H}_2\text{CO}_3$，对应 $K_{b2}=K_w/K_{a1}$）。
陷阱6：缓冲溶液稀释的影响
Henderson-Hasselbalch 方程表明，缓冲溶液的 pH 主要取决于 $c_b/c_a$ 的比值，稀释时比值不变所以 pH 几乎不变。但总浓度降低导致缓冲容量减小。
陷阱7：共轭酸碱对只差一个质子
$\text{HPO}_4^{2-}$ 的共轭酸是 $\text{H}_2\text{PO}_4^-$（差一个质子），不是 $\text{H}_3\text{PO}_4$（差两个质子）。$\text{PO}_4^{3-}$ 的共轭酸是 $\text{HPO}_4^{2-}$，不是 $\text{H}_3\text{PO}_4$。
陷阱8：pH 与 $[\text{H}^+]$ 互算精度
$10^{-11.01}$ 不等于 $10^{-11}$！要注意小数部分：$10^{-11.01} = 10^{-11} \times 10^{-0.01} = 10^{-11} \times 0.977 = 9.77\times10^{-12}$。

十、真题考点精讲

本节汇总了历年小测和期末考试中的高频考点，按题型分类整理。每个考点都标注了出处，务必在考前重点复习！

10.1 选择题考点

小测一 选择题 Q2：指示剂 $K_{\text{HIn}}$ 与变色范围

考点：酸碱指示剂 HIn 的理论变色范围为 $\text{pH} = pK_{\text{HIn}} \pm 1$。

原理：指示剂 HIn 是弱酸，在溶液中存在平衡 $\text{HIn} \rightleftharpoons \text{H}^+ + \text{In}^-$。

  • 当 $[\text{HIn}]/[\text{In}^-] \geq 10$ 时，显酸色 → $\text{pH} \leq pK_{\text{HIn}} - 1$
  • 当 $[\text{HIn}]/[\text{In}^-] \leq 1/10$ 时，显碱色 → $\text{pH} \geq pK_{\text{HIn}} + 1$
  • 中间范围：混合色（变色过渡区）

实际变色范围通常比理论范围窄，约 $pK_{\text{HIn}} \pm 0.5$ 左右视觉变化最明显。

小测一 选择题 Q3：最大缓冲容量

考点：$\beta_{\max} = 0.576c = 2.303c/4$

已知 $c = 0.10$ mol/L → $\beta_{\max} = 0.0576 \approx 0.058$ mol/(L·pH)

易错点：混淆 $c$ 是总浓度还是单组分浓度。$c = c_a + c_b$ 是缓冲体系的总浓度。

小测一 选择题 Q5：分布分数最大值对应 pH

考点：$\text{H}_2\text{PO}_4^-$ 的最大分布分数在 $\text{pH} = (pK_{a1}+pK_{a2})/2 = (2.12+7.20)/2 = 4.66$

推广：任何多元酸的第 $i$ 个中间形体，$\delta$ 最大值在 $\text{pH} = (pK_{ai}+pK_{a,i+1})/2$。

10.2 填空题考点

小测一 填空题 Q3：NaHCO3 的 PBE

标准答案：

$$[\text{H}_2\text{CO}_3] + [\text{H}^+] = [\text{CO}_3^{2-}] + [\text{OH}^-]$$

参考水准：$\text{HCO}_3^-$，$\text{H}_2\text{O}$。得到质子 → $\text{H}_2\text{CO}_3$、$\text{H}^+$；失去质子 → $\text{CO}_3^{2-}$、$\text{OH}^-$。

小测一 填空题 Q5：邻苯二甲酸污染导致 NaOH 标定偏低

考点：用邻苯二甲酸氢钾（KHP, $M = 204.22$, 一元酸）标定 NaOH 时，若 KHP 样品被邻苯二甲酸（$\text{H}_2\text{A}$, $M = 166.13$, 二元酸）污染：

  • $\text{H}_2\text{A}$ 的摩尔质量更小（166 vs 204），同质量含更多摩尔数
  • $\text{H}_2\text{A}$ 每摩尔消耗 2 mol NaOH（而 KHP 每摩尔仅消耗 1 mol）
  • 因此同质量被污染的样品消耗的 NaOH 更多，V(NaOH) 增大

标定时：$c(\text{NaOH}) = \dfrac{m/M_{\text{KHP}}}{V(\text{NaOH})}$。分子不变（按纯 KHP 计算），分母 $V$ 增大，所以 $c(\text{NaOH})$ 计算值偏低。

结论：NaOH 的标定浓度偏低，用偏低的浓度去测定其他酸时，酸的测定结果偏高。

小测一 填空题 Q6：NaOH + NaAc 混合溶液的 PBE

标准答案：$c_b + [\text{HAc}] + [\text{H}^+] = [\text{OH}^-]$

其中 $c_b$ 为 NaOH 的浓度。

2022 期末填空题 Q2：HCOOH 在 pH = pKa 时的分布分数

考点：任何一元酸 HA 在 $\text{pH} = pK_a$ 时，$\delta(\text{HA}) = \delta(\text{A}^-) = 0.50$。

所以 HCOOH 在 pH = 3.75（$pK_a$ of HCOOH）时，$\delta(\text{HCOOH}) = 0.50$，$\delta(\text{HCOO}^-) = 0.50$。

2022 期末填空题 Q4：Na2HPO4 的 PBE

标准答案：

$$[\text{H}^+] + 2[\text{H}_3\text{PO}_4] + [\text{H}_2\text{PO}_4^-] = [\text{OH}^-] + [\text{PO}_4^{3-}]$$

参考水准为 $\text{HPO}_4^{2-}$ 和 $\text{H}_2\text{O}$。$\text{H}_3\text{PO}_4$ 需要系数 2（从 $\text{HPO}_4^{2-}$ 得到 2 个质子）。

2022 期末填空题 Q5：pH = 11.01 求 [H+]

标准答案：

$$[\text{H}^+] = 10^{-11.01} = 9.8 \times 10^{-12}\;\text{mol/L}$$

计算方法：$10^{-11.01} = 10^{-11} \times 10^{-0.01}$。查表或近似 $10^{-0.01} \approx 0.977$，所以 $[\text{H}^+] \approx 0.977 \times 10^{-11} \approx 9.8 \times 10^{-12}$。

10.3 计算题考点

小测一 计算题 Q2：THAM 缓冲液

完整解答见上方 7.7 节。关键结果汇总：

  • THAM 称量：7.272 g（按题目标准答案）
  • 有效缓冲范围：$pK_a \pm 1 = 8.08 \pm 1 = 7.08 \sim 9.08$
  • 缓冲容量：$\beta \approx 0.02$ mol/(L·pH)（按题目标准答案）
2022 期末 Q2：pH = 9 的 NH3/NH4+ 缓冲液

题意：用氨水（$\text{NH}_3 \cdot \text{H}_2\text{O}$，$pK_b = 4.75$，即 $\text{NH}_4^+$ 的 $pK_a = 9.25$）配制 pH = 9.00 的缓冲溶液。

解：

缓冲体系为 $\text{NH}_3/\text{NH}_4^+$，用 HH 方程：

$$\text{pH} = pK_a(\text{NH}_4^+) + \lg\frac{[\text{NH}_3]}{[\text{NH}_4^+]}$$ $$9.00 = 9.25 + \lg\frac{c_b}{c_a}$$ $$\lg\frac{c_b}{c_a} = -0.25 \quad \Rightarrow \quad \frac{c_b}{c_a} = 10^{-0.25} = 0.562$$

即 $[\text{NH}_3]/[\text{NH}_4^+] = 0.562$，氨水（碱）与铵盐（酸）的浓度比约为 0.56:1。

验证：$pK_a(\text{NH}_4^+) = 9.25$，缓冲范围 $9.25 \pm 1 = 8.25 \sim 10.25$，pH = 9.00 在范围内 ✓。

$c_b/c_a = 0.562$，在 $0.1 \sim 10$ 之间 ✓。

配制方法：取一定量氨水，加入适量 $\text{NH}_4\text{Cl}$ 固体（或用 HCl 中和部分氨水），使得 $\text{NH}_3:\text{NH}_4^+$ 的摩尔比为 0.562:1。

2022 期末填空题 Q2 扩展：共轭酸的 pKa

$\text{HPO}_4^{2-}$ 的共轭酸为 $\text{H}_2\text{PO}_4^-$。

常见易混淆的共轭关系：

给定物种共轭酸（+H$^+$）共轭碱（-H$^+$）
$\text{H}_2\text{PO}_4^-$$\text{H}_3\text{PO}_4$$\text{HPO}_4^{2-}$
$\text{HPO}_4^{2-}$$\text{H}_2\text{PO}_4^-$$\text{PO}_4^{3-}$
$\text{HCO}_3^-$$\text{H}_2\text{CO}_3$$\text{CO}_3^{2-}$
$\text{NH}_3$$\text{NH}_4^+$$\text{NH}_2^-$
$\text{H}_2\text{O}$$\text{H}_3\text{O}^+$$\text{OH}^-$
考前速记清单：
  • $K_a \cdot K_b = K_w = 10^{-14}$（25°C），$pK_a + pK_b = 14$
  • 最简式条件：$cK_a \geq 20K_w$ 且 $c/K_a \geq 400$
  • 弱酸：$\text{pH} = \frac{1}{2}(pK_a - \lg c)$
  • 弱碱：$\text{pOH} = \frac{1}{2}(pK_b - \lg c)$，$\text{pH} = 14 - \text{pOH}$
  • 缓冲液：$\text{pH} = pK_a + \lg(c_b/c_a)$
  • 两性物质：$\text{pH} = \frac{1}{2}(pK_{a1}+pK_{a2})$
  • 缓冲范围：$pK_a \pm 1$
  • 最大缓冲容量：$\beta_{\max} = 0.576c$（当 $\text{pH} = pK_a$）
  • 中间形体最大分布：$\text{pH}=(pK_{ai}+pK_{a,i+1})/2$
  • 弱酸弱碱盐（$\text{NH}_4\text{Ac}$型）：$\text{pH} = \frac{1}{2}(pK_{a,\text{弱酸}}+pK_{a,\text{共轭酸}})$，与浓度无关
  • 多元酸按一元处理条件：$2K_{a2}/[\text{H}^+]<0.05$，还需检查 $c/K_{a1}\geq400$
  • 一元酸在 pH = pKa 时：$\delta(\text{HA}) = \delta(\text{A}^-) = 0.50$
  • $\text{HPO}_4^{2-}$ 的共轭酸是 $\text{H}_2\text{PO}_4^-$（差一个质子！）
  • NaHCO$_3$ PBE：$[\text{H}_2\text{CO}_3]+[\text{H}^+]=[\text{CO}_3^{2-}]+[\text{OH}^-]$
  • Na$_2$HPO$_4$ PBE：$[\text{H}^+]+2[\text{H}_3\text{PO}_4]+[\text{H}_2\text{PO}_4^-]=[\text{OH}^-]+[\text{PO}_4^{3-}]$
  • 缓冲容量公式：$\beta = 2.303 c \cdot \delta_{\text{HA}} \cdot \delta_{\text{A}^-}$
