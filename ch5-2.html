第五章(下) 酸碱滴定法

一、酸碱指示剂

PPT: 酸碱指示剂原理

1.1 指示剂变色原理

酸碱指示剂本身是有机弱酸或弱碱，其共轭酸碱对具有不同的颜色：

$$\text{HIn} \rightleftharpoons \text{H}^+ + \text{In}^-$$ $$K_{\text{HIn}} = \frac{[\text{H}^+][\text{In}^-]}{[\text{HIn}]}$$

当 $[\text{HIn}]/[\text{In}^-] \geq 10$ 时，显酸色；当 $[\text{In}^-]/[\text{HIn}] \geq 10$ 时，显碱色。

1.2 变色范围

理论变色范围
$$\text{pH} = pK_{\text{HIn}} \pm 1$$

即变色范围约为2个pH单位。在 $\text{pH} = pK_{\text{HIn}}$ 时显混合色（变色点/理论变色点）。

Quiz 1 选择题 Q2

已知某指示剂 $K_{\text{HIn}} = 10^{-5}$，则其变色范围为？

解：$pK_{\text{HIn}} = 5$，变色范围 = $pK_{\text{HIn}} \pm 1$ = pH 4 ~ 6。

1.4 指示剂选择规则

指示剂选择三步法
  1. 先算化学计量点 pH（$\text{pH}_{\text{sp}}$）
  2. 确定突跃范围，指示剂变色范围须全部或部分落在突跃范围内
  3. 选变色点最接近 $\text{pH}_{\text{sp}}$ 的指示剂，使 $|E_t|$ 最小

1.5 影响变色范围的因素

  • 指示剂用量：用量越多，变色越不灵敏，应尽量少用（1~3滴）
  • 温度：影响 $K_{\text{HIn}}$，从而影响变色范围
  • 溶剂：非水溶剂中变色范围不同
  • 离子强度：高离子强度影响活度系数
  • 滴定顺序：从酸色变碱色 vs 从碱色变酸色，观察灵敏度不同

对于 0.1 mol/L 强酸-强碱滴定（突跃 4.3~9.7）：

  • 甲基橙（3.1~4.4）：变色范围部分落入，可用但不最优
  • 甲基红（4.4~6.2）：可用
  • 酚酞（8.0~9.8）：可用
  • 溴百里酚蓝（6.0~7.6）：最佳（变色点最接近化学计量点）

对于 0.01 mol/L 强酸-强碱滴定（突跃 5.3~8.7）：

  • 甲基橙（3.1~4.4）：不能用（完全落在突跃外）
  • 甲基红（4.4~6.2）：勉强可用（部分落入）
  • 溴百里酚蓝（6.0~7.6）：最佳
  • 酚酞（8.0~9.8）：部分落入，可用但误差稍大

3.1 以 NaOH 滴定 HAc 为例

设用 0.1000 mol/L NaOH 滴定 20.00 mL 0.1000 mol/L HAc（$K_a = 1.8\times10^{-5}$，$pK_a = 4.74$）：

3.2 四个阶段的 pH 计算

阶段 1：滴定前（纯弱酸）
$$[\text{H}^+] = \sqrt{K_a \cdot c} = \sqrt{1.8\times10^{-5} \times 0.10} = 1.34\times10^{-3}$$ $$\text{pH} = 2.87$$
阶段 2：滴定中（缓冲区，0% < 滴定分数 < 100%）

溶液中同时存在 HAc 和 NaAc，用 Henderson-Hasselbalch 方程：

$$\text{pH} = pK_a + \lg\frac{c_{\text{NaAc}}}{c_{\text{HAc}}} = pK_a + \lg\frac{n_{\text{NaAc}}}{n_{\text{HAc}}}$$

当滴定分数 = 50% 时，$n_{\text{NaAc}} = n_{\text{HAc}}$，$\text{pH} = pK_a = 4.74$（半当量点）

阶段 4：过量（NaOH 过量）

溶液中有过量 NaOH，共轭碱 $\text{Ac}^-$ 的水解被抑制，pH 主要由过量 NaOH 决定：

$$\text{pH} = 14 + \lg c_{\text{NaOH,过量}}$$

三A、强酸滴定弱碱

3A.1 HCl 滴定 CH$_3$NH$_2$（Quiz 1 计算题 Q3 原题）

Quiz 1 计算题 Q3：HCl 滴定甲胺

用 0.1000 mol/L HCl 滴定 20.00 mL 0.1000 mol/L $\text{CH}_3\text{NH}_2$（$pK_b = 3.38$）。

基本参数

$\text{CH}_3\text{NH}_2$ 是弱碱，$pK_b = 3.38$，$K_b = 10^{-3.38} = 4.17\times10^{-4}$

其共轭酸 $\text{CH}_3\text{NH}_3^+$：$pK_a = 14 - 3.38 = 10.62$，$K_a = 10^{-10.62} = 2.40\times10^{-11}$

这是强酸滴定弱碱的情形，化学计量点产物为弱酸 $\text{CH}_3\text{NH}_3^+$，pH < 7。

各阶段 pH 计算：

滴定前（0%）：纯弱碱

$$[\text{OH}^-] = \sqrt{K_b \cdot c} = \sqrt{10^{-3.38} \times 0.1} = \sqrt{4.17\times10^{-5}} = 6.46\times10^{-3}$$ $$\text{pOH} = 2.19, \quad \text{pH} = 11.81$$

半当量点（50%）：缓冲体系

$\text{CH}_3\text{NH}_2$ 和 $\text{CH}_3\text{NH}_3^+$ 等量共存：

$$\text{pOH} = pK_b = 3.38, \quad \text{pH} = 14 - 3.38 = 10.62$$

等价地：$\text{pH} = pK_a = 10.62$（半当量点 pH = 共轭酸的 $pK_a$）

化学计量点（100%）：纯共轭酸 $\text{CH}_3\text{NH}_3^+$

$c_{\text{sp}} = 0.1 \times 20/40 = 0.05\;\text{mol/L}$

$$[\text{H}^+] = \sqrt{K_a \cdot c_{\text{sp}}} = \sqrt{10^{-10.62} \times 0.05}$$ $$= \sqrt{2.40\times10^{-11} \times 0.05} = \sqrt{1.20\times10^{-12}} = 1.095\times10^{-6}$$ $$\text{pH} = -\lg(1.095\times10^{-6}) = 5.96$$

化学计量点 pH = 5.96，呈酸性。选甲基红（4.4~6.2）为指示剂。

过量后（200%）：HCl 大量过量

$$c_{\text{H}^+} = \frac{0.1\times20}{40} = 0.05, \quad \text{pH} = 1.30$$

3A.2 强酸滴弱碱的通用特征

与强碱滴弱酸的对比：
  • 强碱滴弱酸：化学计量点 pH > 7，选酚酞
  • 强酸滴弱碱：化学计量点 pH < 7，选甲基橙或甲基红
  • 两者是"镜像"关系，公式中 $K_a \leftrightarrow K_b$

四、直接滴定判据

弱酸能否被强碱直接滴定的判据
$$c \cdot K_a \geq 10^{-8}$$

其中 $c$ 为弱酸的初始浓度，$K_a$ 为弱酸的解离常数。

若 $cK_a < 10^{-8}$，则突跃范围太小（不到 0.3 个 pH 单位），无法选择合适指示剂，不能直接准确滴定。

同理，弱碱能否被强酸直接滴定：

$$c \cdot K_b \geq 10^{-8}$$
判断：能否用 0.1 mol/L NaOH 直接滴定 0.1 mol/L $\text{H}_3\text{BO}_3$（$K_a = 5.8\times10^{-10}$）？
解：
$cK_a = 0.1 \times 5.8\times10^{-10} = 5.8\times10^{-11} < 10^{-8}$
不能直接滴定。需要加入甘油（多元醇）使硼酸变为强酸后再滴定（见第七节）。

五、多元酸的滴定

PPT: 多元酸分步滴定的判据

5.1 分步滴定判据

分步滴定条件
$$\frac{K_{a1}}{K_{a2}} > 10^5 \quad (\text{有的教材取} > 10^4)$$

当相邻两级 $K_a$ 的比值大于 $10^5$ 时，可以分步滴定，在滴定曲线上出现两个明显突跃。

同时每一步还需满足 $cK_{ai} \geq 10^{-8}$ 的可滴定条件。

严格 vs 宽松判据：
  • $K_{a1}/K_{a2} \geq 10^5$：严格判据，分步滴定精度高，两个突跃完全分开
  • $K_{a1}/K_{a2} \geq 10^4$：宽松判据，可以勉强分步，但第一终点不够锐利
  • $K_{a1}/K_{a2} < 10^4$：不能分步滴定，只能一步到位

考试中若未特别说明，一般取 $10^4$ 为界。

5.2 $\text{H}_3\text{PO}_4$ 的滴定

$\text{H}_3\text{PO}_4$：$K_{a1}=7.6\times10^{-3}$，$K_{a2}=6.3\times10^{-8}$，$K_{a3}=4.4\times10^{-13}$

步骤$K_{ai}/K_{a,i+1}$能否分步滴定？$cK_a \geq 10^{-8}$？
第一步 $\to \text{H}_2\text{PO}_4^-$$K_{a1}/K_{a2} = 1.2\times10^5 > 10^5$可以$0.1\times7.6\times10^{-3} \gg 10^{-8}$ ✓
第二步 $\to \text{HPO}_4^{2-}$$K_{a2}/K_{a3} = 1.4\times10^5 > 10^5$可以$0.1\times6.3\times10^{-8} = 6.3\times10^{-9} < 10^{-8}$（勉强可用，突跃较小）
第三步 $\to \text{PO}_4^{3-}$----$0.1\times4.4\times10^{-13} \ll 10^{-8}$ ✗

结论：$\text{H}_3\text{PO}_4$ 可以滴定到两个终点：

  • 第一终点（SP1）：$\text{H}_3\text{PO}_4 \to \text{NaH}_2\text{PO}_4$，$\text{pH} \approx \frac{pK_{a1}+pK_{a2}}{2} = \frac{2.12+7.20}{2} = 4.66 \approx 4.7$，选甲基橙或溴甲酚绿
  • 第二终点（SP2）：$\text{NaH}_2\text{PO}_4 \to \text{Na}_2\text{HPO}_4$，$\text{pH} \approx \frac{pK_{a2}+pK_{a3}}{2} = \frac{7.20+12.36}{2} = 9.78 \approx 9.8$，选酚酞或百里酚酞
  • 第三步不可滴定（$K_{a3}$ 太小）

5.3 $\text{H}_2\text{C}_2\text{O}_4$ (草酸) 的滴定

$\text{H}_2\text{C}_2\text{O}_4$：$K_{a1}=5.9\times10^{-2}$，$K_{a2}=6.4\times10^{-5}$

$$K_{a1}/K_{a2} = 5.9\times10^{-2}/6.4\times10^{-5} \approx 920 \ll 10^4$$

结论：$K_{a1}/K_{a2}$ 太小，不能分步滴定。只能一次性滴定到 $\text{Na}_2\text{C}_2\text{O}_4$，即只有一个终点。消耗 NaOH 的物质的量为草酸的2倍。

Quiz 1 选择题 Q4 / 2022 期末填空 Q1

草酸只有一个突跃（一个终点），因为 $K_{a1}/K_{a2} \approx 920 < 10^4$。

这是高频考点。务必记住：草酸不能分步滴定！

5.4 $\text{Na}_2\text{CO}_3$ 的滴定（强酸滴弱碱的盐）

用 HCl 滴定 $\text{Na}_2\text{CO}_3$：

  • $K_{b1} = K_w/K_{a2} = 10^{-14}/5.6\times10^{-11} = 1.8\times10^{-4}$
  • $K_{b2} = K_w/K_{a1} = 10^{-14}/4.3\times10^{-7} = 2.3\times10^{-8}$
  • $K_{b1}/K_{b2} = 7800 < 10^5$，但 $> 10^4$，实际上可以勉强分步

第一终点：$\text{Na}_2\text{CO}_3 \to \text{NaHCO}_3$（pH $\approx$ 8.3，酚酞变色）

第二终点：$\text{NaHCO}_3 \to \text{H}_2\text{CO}_3$（pH $\approx$ 3.9，甲基橙变色）

5.5 Na$_3$PO$_4$ + Na$_2$HPO$_4$ 混合物的滴定

Quiz 1 实验设计题

用 HCl 标准溶液滴定 $\text{Na}_3\text{PO}_4$ + $\text{Na}_2\text{HPO}_4$ 混合物。

分析：

$\text{Na}_3\text{PO}_4$ 是三元碱，$\text{Na}_2\text{HPO}_4$ 是二元碱的第一步产物。用 HCl 滴定：

第一步（酚酞指示剂，滴至 pH $\approx$ 9.7）：

  • $\text{Na}_3\text{PO}_4 + \text{HCl} \to \text{Na}_2\text{HPO}_4 + \text{NaCl}$（$\text{Na}_3\text{PO}_4$ 全部反应）
  • $\text{Na}_2\text{HPO}_4$ 在此 pH 不反应
  • 消耗 HCl 体积 $V_1$，$V_1$ 对应 $\text{Na}_3\text{PO}_4$ 的量

第二步（甲基红指示剂，继续滴至 pH $\approx$ 4.7）：

  • 此时所有 $\text{Na}_2\text{HPO}_4$（原有的 + 第一步生成的）都被滴定到 $\text{NaH}_2\text{PO}_4$
  • 消耗 HCl 体积 $V_2$

计算：

$$n(\text{Na}_3\text{PO}_4) = c_{\text{HCl}} \cdot V_1$$ $$n(\text{Na}_2\text{HPO}_4)_{\text{原有}} = c_{\text{HCl}} \cdot V_2 - c_{\text{HCl}} \cdot V_1 = c_{\text{HCl}}(V_2 - V_1)$$

六、终点误差

PPT: 滴定终点误差计算

终点误差是考试重点！务必掌握强酸-强碱和弱酸-强碱两种情况的公式。

6.1 基本概念

终点误差（$E_t$，也称滴定误差）：由于滴定终点（指示剂变色点）与化学计量点不一致而引起的误差。

$$E_t = \frac{V_{\text{ep}} - V_{\text{sp}}}{V_{\text{sp}}} \times 100\%$$

其中 $V_{\text{ep}}$ 为终点时消耗的滴定剂体积，$V_{\text{sp}}$ 为化学计量点消耗的体积。

6.2 强酸-强碱滴定的终点误差

NaOH 滴定 HCl
$$\boxed{E_t = \frac{[\text{OH}^-]_{\text{ep}} - [\text{H}^+]_{\text{ep}}}{c_{\text{sp}}} \times 100\%}$$

其中 $[\text{H}^+]_{\text{ep}}$ 和 $[\text{OH}^-]_{\text{ep}}$ 为终点时的 $\text{H}^+$ 和 $\text{OH}^-$ 浓度，$c_{\text{sp}}$ 为化学计量点时被测物的浓度。

等价写法（利用 $\Delta\text{pH}$）：

$$E_t = \frac{10^{\text{pH}_{\text{ep}}-14} - 10^{-\text{pH}_{\text{ep}}}}{c_{\text{sp}}} \times 100\%$$

直接代入终点 $\text{pH}_{\text{ep}}$ 即可，分子 = $[\text{OH}^-]_{\text{ep}} - [\text{H}^+]_{\text{ep}}$。

当终点偏酸（$\text{pH}_{\text{ep}} < 7$）：$E_t < 0$（未滴到/欠滴定）
当终点偏碱（$\text{pH}_{\text{ep}} > 7$）：$E_t > 0$（过量滴定）

6.3 强碱滴定弱酸的终点误差（林邦误差公式）

林邦 (Lindberg) 公式 -- NaOH 滴定弱酸 HA
$$\boxed{E_t = \frac{10^{\Delta\text{pH}} - 10^{-\Delta\text{pH}}}{\sqrt{K_t \cdot c_{\text{sp}}}} \times 100\%}$$

其中 $K_t = K_a/K_w = 1/K_b$，$\Delta\text{pH} = \text{pH}_{\text{ep}} - \text{pH}_{\text{sp}}$。

当 $\Delta\text{pH} > 0$（终点偏碱）：$E_t > 0$（过滴定）
当 $\Delta\text{pH} < 0$（终点偏酸）：$E_t < 0$（欠滴定）

强酸滴定弱碱的终点误差

对于 HCl 滴定弱碱 B 的情况，林邦公式变为：

$$\boxed{E_t = \frac{10^{-\Delta\text{pH}} - 10^{\Delta\text{pH}}}{\sqrt{K_b/K_w \cdot c_{\text{sp}}}} \times 100\% = \frac{10^{-\Delta\text{pH}} - 10^{\Delta\text{pH}}}{\sqrt{K_t' \cdot c_{\text{sp}}}} \times 100\%}$$

其中 $K_t' = K_b/K_w = 1/K_a$（此处 $K_a$ 是共轭酸的），$\Delta\text{pH} = \text{pH}_{\text{ep}} - \text{pH}_{\text{sp}}$。

注意：滴定弱碱时，分子中 $10^{-\Delta\text{pH}}$ 和 $10^{\Delta\text{pH}}$ 的位置与滴定弱酸相反！也可以理解为：分母中使用的是 $K_b$（被滴物的 $K_b$）。

符号约定：

  • $E_t > 0$：过滴定（终点在化学计量点之后）
  • $E_t < 0$：欠滴定（终点在化学计量点之前）
  • $|E_t| \leq 0.1\%$ 一般认为可以接受

6.4 $\lg(c_{\text{sp}} \cdot K_t)$ 与终点误差的关系（PPT p64）

当 $|\Delta\text{pH}| = 0.3$ 时，不同 $\lg(c_{\text{sp}} \cdot K_t)$ 对应的终点误差：

$\lg(c_{\text{sp}} \cdot K_t)$$|E_t|$能否准确滴定？
10.00.0015%极佳
8.00.015%很好
6.00.15%勉强可接受
4.01.5%不可接受

规律：$\lg(c_{\text{sp}} \cdot K_t)$ 每减小2，$|E_t|$ 增大约10倍。要使 $|E_t| \leq 0.1\%$，需要 $\lg(c_{\text{sp}} \cdot K_t) \geq 6$，即 $c_{\text{sp}} \cdot K_t \geq 10^6$。

对于弱酸被强碱滴定：$K_t = K_a/K_w$，所以条件等价于 $c_{\text{sp}} \cdot K_a/K_w \geq 10^6$，即 $c \cdot K_a \geq 10^{-8}$。这正是直接滴定判据的来源！

终点误差的等价形式（直接用浓度）：对于 NaOH 滴定弱酸 HA 的情形，林邦公式也可以写成： $$E_t = \left(\frac{[\text{H}^+]_{\text{ep}}}{c_{\text{sp}}} - \frac{[\text{A}^-]_{\text{ep}} \cdot \delta_0}{K_a}\right) \times 100\%$$ 其中 $\delta_0 = [\text{H}^+]_{\text{ep}}/([\text{H}^+]_{\text{ep}}+K_a)$。但在考试中，使用 $\Delta\text{pH}$ 形式更方便。
例题：NaOH 滴定 HAc 的终点误差
用 0.1000 mol/L NaOH 滴定 20.00 mL 0.1000 mol/L HAc（$K_a=1.8\times10^{-5}$），以酚酞为指示剂（变色点 pH = 9.0）。计算终点误差。
解：
Step 1：计算化学计量点 pH
$c_{\text{sp}} = 0.1000 \times 20.00 / 40.00 = 0.0500\;\text{mol/L}$
$K_b = K_w/K_a = 10^{-14}/1.8\times10^{-5} = 5.56\times10^{-10}$
$[\text{OH}^-] = \sqrt{K_b \cdot c_{\text{sp}}} = \sqrt{5.56\times10^{-10} \times 0.050} = 5.27\times10^{-6}$
$\text{pOH} = 5.28$，$\text{pH}_{\text{sp}} = 8.72$

Step 2：计算终点误差
$\text{pH}_{\text{ep}} = 9.0$（酚酞变色点）
$\Delta\text{pH} = 9.0 - 8.72 = +0.28$

$K_t = K_a/K_w = 1.8\times10^{-5}/10^{-14} = 1.8\times10^{9}$
$\sqrt{K_t \cdot c_{\text{sp}}} = \sqrt{1.8\times10^{9} \times 0.050} = \sqrt{9.0\times10^{7}} = 9.49\times10^{3}$

$$E_t = \frac{10^{0.28} - 10^{-0.28}}{9.49\times10^{3}} \times 100\% = \frac{1.905 - 0.525}{9.49\times10^{3}} \times 100\%$$ $$= \frac{1.38}{9490} \times 100\% = +0.015\%$$ $|E_t| = 0.015\% < 0.1\%$，终点误差可以接受。正号表示略有过滴定。
Quiz 1 计算题 Q3 第4问：CH$_3$NH$_2$ 滴定的终点误差

用 0.1 mol/L HCl 滴定 0.1 mol/L $\text{CH}_3\text{NH}_2$（$pK_b=3.38$），以甲基橙为指示剂（终点 pH = 4.0），求 $E_t$。

解：

已知 $\text{pH}_{\text{sp}} = 5.96$（前面已算），$\text{pH}_{\text{ep}} = 4.0$

$\Delta\text{pH} = 4.0 - 5.96 = -1.96$

这是强酸滴弱碱的情形，使用林邦公式（弱碱版本）：

$$E_t = \frac{10^{-\Delta\text{pH}} - 10^{\Delta\text{pH}}}{\sqrt{K_b/K_w \cdot c_{\text{sp}}}} \times 100\%$$

其中 $K_b = 10^{-3.38}$，$K_b/K_w = 10^{-3.38}/10^{-14} = 10^{10.62}$

$c_{\text{sp}} = 0.05\;\text{mol/L}$

$\sqrt{K_b/K_w \cdot c_{\text{sp}}} = \sqrt{10^{10.62}\times 0.05} = \sqrt{10^{10.62}\times 5\times10^{-2}}$

$= \sqrt{5\times10^{8.62}} = \sqrt{5\times4.17\times10^{8}} = \sqrt{2.09\times10^{9}} = 4.57\times10^{4}$

分子：$10^{-(-1.96)} - 10^{-1.96} = 10^{1.96} - 10^{-1.96} = 91.2 - 0.0110 = 91.19$

$$E_t = \frac{91.19}{4.57\times10^4} \times 100\% = 0.20\%$$

$|E_t| = 0.20\% > 0.1\%$，终点误差偏大。这说明用甲基橙（pH=4.0）做终点不是最佳选择。如果用甲基红（pH=5.0），$\Delta\text{pH} = 5.0 - 5.96 = -0.96$，误差会更小。

2022 期末 Q9：终点误差大小排序

用 NaOH 滴定 HCl，分别用甲基红、酚酞、甲基橙作指示剂，$|E_t|$ 从小到大排序？

解：

化学计量点 pH = 7.00（强酸强碱）。

  • 甲基红变色点 $\approx$ pH 5.0，$|\Delta\text{pH}| = |5.0 - 7| = 2.0$
  • 酚酞变色点 $\approx$ pH 9.0，$|\Delta\text{pH}| = |9.0 - 7| = 2.0$
  • 甲基橙变色点 $\approx$ pH 4.0，$|\Delta\text{pH}| = |4.0 - 7| = 3.0$

但注意强酸-强碱 $E_t$ 公式中 $E_t \propto ([\text{OH}^-]_{\text{ep}} - [\text{H}^+]_{\text{ep}})$，不对称！

  • 甲基红（pH=5.0）：$|E_t| = |10^{-9} - 10^{-5}|/c_{\text{sp}} \times 100\%$，以 $10^{-5}$ 为主
  • 酚酞（pH=9.0）：$|E_t| = |10^{-5} - 10^{-9}|/c_{\text{sp}} \times 100\%$，也以 $10^{-5}$ 为主
  • 甲基橙（pH=4.0）：$|E_t| = |10^{-10} - 10^{-4}|/c_{\text{sp}} \times 100\%$，以 $10^{-4}$ 为主

答：$|E_t|$ 大小排序：甲基红 $\approx$ 酚酞 < 甲基橙

即甲基红和酚酞误差接近且较小，甲基橙误差最大。

七、双指示剂法

PPT: NaOH + Na2CO3 双指示剂法滴定流程图

7.1 NaOH + Na$_2$CO$_3$ 混合物

双指示剂法（经典考题）

用 HCl 标准溶液滴定 NaOH 和 $\text{Na}_2\text{CO}_3$ 的混合溶液：

第一步：加酚酞指示剂

滴定至酚酞褪色（pH $\approx$ 8.0），消耗 HCl 体积 $V_1$

此时：

  • NaOH 被完全中和：$\text{NaOH} + \text{HCl} \to \text{NaCl} + \text{H}_2\text{O}$
  • $\text{Na}_2\text{CO}_3$ 被滴定一半（至 $\text{NaHCO}_3$）：$\text{Na}_2\text{CO}_3 + \text{HCl} \to \text{NaHCO}_3 + \text{NaCl}$

第二步：加甲基橙指示剂

继续滴定至甲基橙变色（pH $\approx$ 4.0），消耗 HCl 体积 $V_2$

此时 $\text{NaHCO}_3$ 被完全中和：$\text{NaHCO}_3 + \text{HCl} \to \text{NaCl} + \text{H}_2\text{O} + \text{CO}_2$

计算：

$$n(\text{Na}_2\text{CO}_3) = c_{\text{HCl}} \cdot V_2 \quad (\text{因为 }V_2\text{ 对应 }\text{Na}_2\text{CO}_3\text{ 的另一半})$$ $$n(\text{NaOH}) = c_{\text{HCl}} \cdot (V_1 - V_2)$$
关键关系：$V_2$ 对应的恰好是 $\text{Na}_2\text{CO}_3$ 的另一半（从 $\text{NaHCO}_3 \to \text{H}_2\text{CO}_3$），所以：
$\text{Na}_2\text{CO}_3$ 消耗的总 HCl = $2V_2$
NaOH 消耗的 HCl = $V_1 - V_2$
必须有 $V_1 \geq V_2$，否则说明没有 NaOH（可能是 $\text{Na}_2\text{CO}_3$ + $\text{NaHCO}_3$ 的混合物）。

7.2 Na$_2$CO$_3$ + NaHCO$_3$ 混合物

用 HCl 标准溶液滴定：

第一步：加酚酞指示剂，滴至褪色，消耗 $V_1$

  • $\text{Na}_2\text{CO}_3 + \text{HCl} \to \text{NaHCO}_3 + \text{NaCl}$（$\text{Na}_2\text{CO}_3$ 变为 $\text{NaHCO}_3$）
  • $\text{NaHCO}_3$ 本身不反应（酚酞区间内）

第二步：加甲基橙指示剂，继续滴至变色，消耗 $V_2$

  • 所有 $\text{NaHCO}_3$（原有的 + 第一步生成的）被完全滴定

计算：

$$n(\text{Na}_2\text{CO}_3) = c_{\text{HCl}} \cdot V_1$$ $$n(\text{NaHCO}_3)_{\text{原有}} = c_{\text{HCl}} \cdot (V_2 - V_1)$$

（因为 $V_2$ 对应所有 $\text{NaHCO}_3$，其中 $V_1$ 份是 $\text{Na}_2\text{CO}_3$ 生成的，$V_2 - V_1$ 份是原有的。）

特征：$V_1 < V_2$（第二步消耗更多 HCl）。

7.3 各种碱性混合物的完整判断

$V_1$ 与 $V_2$ 的关系混合物组成各组分含量计算
$V_2 = 0$纯 NaOH$n(\text{NaOH}) = cV_1$
$V_1 > V_2 > 0$NaOH + $\text{Na}_2\text{CO}_3$$n(\text{Na}_2\text{CO}_3) = cV_2$; $n(\text{NaOH}) = c(V_1 - V_2)$
$V_1 = V_2 > 0$纯 $\text{Na}_2\text{CO}_3$$n(\text{Na}_2\text{CO}_3) = cV_1 = cV_2$
$0 < V_1 < V_2$$\text{Na}_2\text{CO}_3$ + $\text{NaHCO}_3$$n(\text{Na}_2\text{CO}_3) = cV_1$; $n(\text{NaHCO}_3) = c(V_2 - V_1)$
$V_1 = 0$纯 $\text{NaHCO}_3$$n(\text{NaHCO}_3) = cV_2$
为什么 NaOH + NaHCO$_3$ 不会同时出现？
因为 NaOH 是强碱，会立即与 $\text{NaHCO}_3$ 反应：$\text{NaOH} + \text{NaHCO}_3 \to \text{Na}_2\text{CO}_3 + \text{H}_2\text{O}$。所以混合物中最多只能是三种组合之一：NaOH+$\text{Na}_2\text{CO}_3$、纯$\text{Na}_2\text{CO}_3$、$\text{Na}_2\text{CO}_3$+$\text{NaHCO}_3$。

7.4 BaCl$_2$ 排除法测 NaOH + Na$_2$CO$_3$（PPT p76-p78）

双指示剂法的替代方案：先用 $\text{BaCl}_2$ 沉淀 $\text{Na}_2\text{CO}_3$，再滴定 NaOH。

方法步骤：

  1. 取 NaOH + $\text{Na}_2\text{CO}_3$ 混合试样，加入过量 $\text{BaCl}_2$：$\text{Na}_2\text{CO}_3 + \text{BaCl}_2 \to \text{BaCO}_3 \downarrow + 2\text{NaCl}$
  2. $\text{Na}_2\text{CO}_3$ 被完全沉淀，溶液中只剩 NaOH
  3. 以酚酞为指示剂，用 HCl 标准溶液滴定溶液中的 NaOH，消耗 $V_1$
  4. $n(\text{NaOH}) = c_{\text{HCl}} \cdot V_1$

未加 $\text{BaCl}_2$ 的另一份试样，直接用甲基橙为指示剂滴定（总碱量），消耗 $V_2$：

$$n(\text{NaOH}) + 2n(\text{Na}_2\text{CO}_3) = c_{\text{HCl}} \cdot V_2$$ $$n(\text{Na}_2\text{CO}_3) = \frac{c_{\text{HCl}}(V_2 - V_1)}{2}$$

优点：两次滴定相互独立，误差不累积。缺点：需要两份试样。

7.5 Na$_3$PO$_4$ + NaOH 混合物（2022 期末实验设计 Option 1）

2022 期末实验设计题

用 HCl 标准溶液滴定 $\text{Na}_3\text{PO}_4$ + NaOH 混合物。

分析思路（类比 $\text{Na}_2\text{CO}_3$ + NaOH）：

第一步：酚酞指示剂，滴至褪色（pH $\approx$ 9.7），消耗 $V_1$

  • $\text{NaOH} + \text{HCl} \to \text{NaCl} + \text{H}_2\text{O}$（NaOH 全部反应）
  • $\text{Na}_3\text{PO}_4 + \text{HCl} \to \text{Na}_2\text{HPO}_4 + \text{NaCl}$（$\text{Na}_3\text{PO}_4$ 被滴定到 $\text{Na}_2\text{HPO}_4$）

第二步：甲基红指示剂，继续滴至变色（pH $\approx$ 4.7），消耗 $V_2$

  • $\text{Na}_2\text{HPO}_4 + \text{HCl} \to \text{NaH}_2\text{PO}_4 + \text{NaCl}$

计算：

$$n(\text{Na}_3\text{PO}_4) = c_{\text{HCl}} \cdot V_2 \quad (\text{第二步的}V_2\text{对应全部}\text{Na}_2\text{HPO}_4\text{，即全部来自}\text{Na}_3\text{PO}_4)$$ $$n(\text{NaOH}) = c_{\text{HCl}} \cdot (V_1 - V_2)$$

八、NaOH 吸收 CO$_2$ 问题

8.1 CO$_2$ 的干扰机理

空气中的 $\text{CO}_2$ 溶于 NaOH 标准溶液中：

$$2\text{NaOH} + \text{CO}_2 \to \text{Na}_2\text{CO}_3 + \text{H}_2\text{O}$$

原来的纯 NaOH 溶液变成了 NaOH + $\text{Na}_2\text{CO}_3$ 混合液。

后果取决于使用的指示剂和被滴定的物质。

8.2 CO$_2$ 在水溶液中的分布形态（PPT p74-p75）

$\text{CO}_2$ 溶于水后以不同形态存在，取决于溶液 pH：

pH 范围主要形态说明
pH < 6.4$\text{CO}_2$（或 $\text{H}_2\text{CO}_3$）分子态为主
6.4 < pH < 10.3$\text{HCO}_3^-$碳酸氢根为主
pH > 10.3$\text{CO}_3^{2-}$碳酸根为主

对滴定的影响规则：

  • 当以甲基橙（MO）为指示剂，终点 pH $\approx$ 4.0 → $\text{CO}_2$ 以分子态存在 → 不影响滴定结果
  • 当以酚酞（PP）为指示剂，终点 pH $\approx$ 9.0 → $\text{CO}_2$ 被 NaOH 转变为 $\text{HCO}_3^-$ → 由于 $\text{CO}_2$ 的影响产生误差，故CO$_2$ 对酚酞终点有影响

8.3 不同指示剂下的影响

用酚酞指示剂：结果偏高！

酚酞变色 pH $\approx$ 8~9，在此 pH 下：

  • NaOH 与 HCl（被测物）完全反应 ✓
  • $\text{Na}_2\text{CO}_3$ 只被滴到 $\text{NaHCO}_3$（消耗1个 $\text{H}^+$），而不是 $\text{H}_2\text{CO}_3$

问题在于：2 mol NaOH 变成了 1 mol $\text{Na}_2\text{CO}_3$，而这 1 mol $\text{Na}_2\text{CO}_3$ 只消耗 1 mol HCl（到 $\text{NaHCO}_3$），等于 2 mol NaOH 的功能只发挥了 1 mol。

因此，要达到终点需要更多的被测 HCl，即测定值偏高。

2022 期末 Q8

用吸收了 $\text{CO}_2$ 的 NaOH 溶液滴定 HCl，以酚酞为指示剂，结果偏高。

用甲基橙指示剂：无影响！

甲基橙变色 pH $\approx$ 3.5~4.4，在此 pH 下：

  • $\text{Na}_2\text{CO}_3$ 被完全滴到 $\text{H}_2\text{CO}_3$（消耗2个 $\text{H}^+$）
  • 2 mol NaOH 变成 1 mol $\text{Na}_2\text{CO}_3$，但 1 mol $\text{Na}_2\text{CO}_3$ 消耗 2 mol HCl
  • 总消耗量不变，因此无影响
Quiz 1 选择题 Q7

用吸收了 $\text{CO}_2$ 的 NaOH 溶液滴定 HCl，以甲基橙为指示剂，对结果无影响。

解决方法：

  • NaOH 标准溶液应避免接触空气，用碱石灰管保护
  • 煮沸蒸馏水除 $\text{CO}_2$
  • 配制时用饱和 NaOH（$\text{Na}_2\text{CO}_3$ 不溶于浓 NaOH）上清液稀释
  • 如已吸收 $\text{CO}_2$，选用甲基橙作指示剂可消除影响

Quiz 1 填空 Q5

用邻苯二甲酸氢钾（KHP）标定 NaOH 溶液，若 KHP 中混有邻苯二甲酸（苯二甲酸）杂质，则标定结果使 NaOH 浓度偏低。

十、酸碱滴定的应用

10.1 凯氏定氮法 (Kjeldahl Method)

测定有机物中氮含量的经典方法：

  1. 消化：样品 + 浓 $\text{H}_2\text{SO}_4$ + 催化剂（$\text{CuSO}_4$、$\text{K}_2\text{SO}_4$），加热分解有机物，$\text{N} \to \text{NH}_4^+$
  2. 蒸馏：加过量 NaOH 使 $\text{NH}_4^+ \to \text{NH}_3 \uparrow$，蒸馏收集 $\text{NH}_3$ 于已知量的 $\text{H}_2\text{SO}_4$（或硼酸）吸收液中
  3. 滴定：用 NaOH 标准溶液回滴过量的 $\text{H}_2\text{SO}_4$；或直接用 HCl 标准溶液滴定硼酸铵
$$\text{N\%} = \frac{(c_1V_1 - c_2V_2) \times 14.01}{m_{\text{sample}} \times 1000} \times 100\%$$

其中 $c_1V_1$ 为吸收液中 $\text{H}_2\text{SO}_4$ 的当量，$c_2V_2$ 为回滴的 NaOH 的当量。

10.2 甲醛法测定铵盐中的氮（PPT p90）

甲醛法原理

$\text{NH}_4^+$ 是极弱酸（$K_a = 5.56\times10^{-10}$），$cK_a \ll 10^{-8}$，不能直接用 NaOH 滴定。

解决方案：加入过量中性甲醛溶液，甲醛与 $\text{NH}_4^+$ 反应生成六亚甲基四胺和强酸 $\text{H}^+$：

$$4\text{NH}_4^+ + 6\text{HCHO} \to (\text{CH}_2)_6\text{N}_4\text{H}^+ + 3\text{H}^+ + 6\text{H}_2\text{O}$$

反应释放的 $\text{H}^+$ 即可用 NaOH 标准溶液滴定。每个 $\text{NH}_4^+$ 释放一个可滴定的 $\text{H}^+$，因此：

$$n(\text{N}) = n(\text{NaOH})$$ $$w(\text{N}) = \frac{c_{\text{NaOH}} \times V_{\text{NaOH}} \times 14.01}{m_s \times 1000} \times 100\%$$

10.3 酸碱滴定法测定 SiO$_2$（PPT p86）

硅酸盐中 $\text{SiO}_2$ 的测定方法：

  1. 用 KOH 熔融：$\text{SiO}_2 \xrightarrow{\text{KOH 熔融}} \text{K}_2\text{SiO}_3$
  2. 加 HF 处理，使硅转化为 $\text{K}_2\text{SiF}_6$ 沉淀
  3. 过滤，用 KCl-乙醇溶液洗涤
  4. 中和游离酸（酚酞指示剂）
  5. 加入沸水使 $\text{K}_2\text{SiF}_6$ 水解：$\text{K}_2\text{SiF}_6 + 3\text{H}_2\text{O} \to \text{H}_2\text{SiO}_3 + 2\text{KF} + 4\text{HF}$
  6. 用 NaOH 标准溶液滴定释放的 HF（酚酞指示剂），产物为 NaF

10.4 不能直接滴定的弱酸碱的间接测定（PPT p91）

两种间接方法

方法一：化学反应增强酸碱性

  • 硼酸加甘油/甘露醇 → 增强酸性（见10.5节）
  • $\text{NH}_4^+$ 加甲醛 → 释放可滴定的 $\text{H}^+$（见10.2节）

方法二：离子交换法

将弱酸的盐通过强酸型阳离子交换树脂，将弱酸的阳离子换为 $\text{H}^+$，生成对应的强酸：

$$\text{R-SO}_3\text{H} + \text{NH}_4\text{Cl} \to \text{R-SO}_3\text{NH}_4 + \text{HCl}$$

流出液中的 HCl 即可用 NaOH 标准溶液直接滴定，间接求出 $\text{NH}_4\text{Cl}$ 的量。

类似地，弱碱的盐可通过强碱型阴离子交换树脂转化：

$$\text{R-NR'}_3\text{-OH} + \text{KNO}_3 \to \text{R-NR'}_3\text{-NO}_3 + \text{KOH}$$

流出液中的 KOH 用 HCl 滴定，即得 $\text{KNO}_3$ 的量。

10.5 硼酸 + 甘油增强法

硼酸不能直接滴定的解决方案

$\text{H}_3\text{BO}_3$ 是极弱酸（$K_a = 5.8\times10^{-10}$），$cK_a \ll 10^{-8}$，不能直接滴定。

加入甘油（丙三醇）或甘露醇等多元醇后，硼酸与多元醇生成络合酸，酸性大大增强：

$$\text{H}_3\text{BO}_3 + 2\text{C}_3\text{H}_5(\text{OH})_3 \to \text{配合酸} \quad K_a' \approx 10^{-4}$$

此时 $cK_a' = 0.1 \times 10^{-4} = 10^{-5} \gg 10^{-8}$，满足滴定条件。用 NaOH 标准溶液滴定，酚酞为指示剂。

十A、非水滴定法简介（PPT p95-p98）

PPT: 非水滴定法简介 -- 拉平效应与区分效应

非水滴定是本章的拓展内容，考试中可能以概念题或选择题形式出现。需掌握拉平效应和区分效应的概念。

10A.1 为什么需要非水滴定

在水溶液中，很多弱酸（$K_a < 10^{-8}$）和弱碱（$K_b < 10^{-8}$）无法直接准确滴定。通过更换溶剂，可以改变酸碱的相对强度，使原本不能滴定的物质变得可以滴定。

10A.2 拉平效应 (Leveling Effect)

拉平效应

在两性溶剂中，最强的酸是溶剂的共轭酸，最强的碱是溶剂的共轭碱。比溶剂共轭酸更强的酸都被"拉平"到同一强度。

例如在水中：$\text{HCl}$、$\text{HClO}_4$、$\text{HNO}_3$ 都是强酸，它们的酸性被"拉平"到 $\text{H}_3\text{O}^+$ 的水平，无法区分强弱。

惰性溶剂（如苯、CCl$_4$）没有拉平效应，但具有很好的区分效应。

10A.3 区分效应 (Differentiating Effect)

区分效应

能分辨酸（碱）强弱的能力。选择合适的溶剂可以使水中无法区分的酸（碱）表现出不同的强度。

例如在冰醋酸（HAc）中：

酸（水中均为强酸）在 HAc 溶剂中的 $pK_a$
$\text{HClO}_4$4.87
$\text{HCl}$8.55
$\text{H}_2\text{SO}_4$7.24
$\text{HNO}_3$9.40

在冰醋酸中，这些"强酸"的 $pK_a$ 差别很大，可以分别滴定！

10A.4 溶剂的选择原则

溶剂自身的离子积 $K_s$ 越小，分辨能力越强，滴定准确度越高。

  • 滴定弱碱（如药物生物碱）：选用酸性溶剂（冰醋酸），增强碱性；标准酸溶液用 $\text{HClO}_4$ 的冰醋酸溶液
  • 滴定弱酸：选用碱性溶剂（乙二胺、丁胺等），增强酸性；标准碱溶液用甲醇钠 $\text{CH}_3\text{ONa}$ 或氢氧化四丁基铵
  • 介电常数大的溶剂有利于离子对的离解，酸的强度增加

10A.5 非水滴定的标准溶液和终点确定

标准酸溶液：0.10 mol/L $\text{HClO}_4$ 的冰醋酸溶液

标准碱溶液：甲醇钠、氢氧化四丁基铵的苯甲醇溶液

终点确定方法：

  • 指示剂法：结晶紫（碱性溶剂中滴定弱碱常用）
  • 电位法：用玻璃电极-甘汞电极组成测量体系，记录电位突跃

11.3 2022 期末计算题 Q4：弱酸 HA 的完整滴定分析

2022 期末计算题 Q4

某一元弱酸 HA（$pK_a = 5.0$），浓度 0.1 mol/L，体积 20.00 mL，用 0.1 mol/L NaOH 滴定。

(1) 计算化学计量点 pH
(2) 选择合适指示剂
(3) 画出滴定曲线的大致趋势
(4) 若以酚酞（pH=9.0）为终点，计算 $E_t$

完整解答：

(1) 化学计量点 pH

$c_{\text{sp}} = 0.1 \times 20/40 = 0.05$ mol/L

$K_b = K_w/K_a = 10^{-14}/10^{-5} = 10^{-9}$

$[\text{OH}^-] = \sqrt{K_b \cdot c_{\text{sp}}} = \sqrt{10^{-9} \times 0.05} = \sqrt{5\times10^{-11}} = 7.07\times10^{-6}$

$\text{pOH} = 5.15$，$\text{pH}_{\text{sp}} = 8.85$

(2) 指示剂选择

化学计量点 pH = 8.85，在碱性区域。选酚酞（8.0~9.8），变色范围包含 8.85。

不能选甲基橙（3.1~4.4）或甲基红（4.4~6.2），远离化学计量点。

(3) 滴定曲线特征

  • 起点：$\text{pH} = \frac{1}{2}(pK_a - \lg c) = \frac{1}{2}(5.0 + 1) = 3.0$
  • 半当量点：$\text{pH} = pK_a = 5.0$（缓冲区平坦段中心）
  • 化学计量点：$\text{pH} = 8.85$（不在 pH=7！）
  • 突跃范围：约 pH 7.7 ~ 9.7（比强酸-强碱窄，且偏碱性）
  • 曲线不对称，化学计量点偏上方

(4) 终点误差

$\text{pH}_{\text{ep}} = 9.0$，$\Delta\text{pH} = 9.0 - 8.85 = +0.15$

$K_t = K_a/K_w = 10^{-5}/10^{-14} = 10^9$

$\sqrt{K_t \cdot c_{\text{sp}}} = \sqrt{10^9 \times 0.05} = \sqrt{5\times10^7} = 7071$

$$E_t = \frac{10^{0.15} - 10^{-0.15}}{7071} \times 100\% = \frac{1.413 - 0.708}{7071}\times100\% = \frac{0.705}{7071}\times100\% = 0.010\%$$

$|E_t| = 0.010\% \ll 0.1\%$，误差很小，酚酞是极佳选择。

例题1：某工厂废液含 NaOH 和 $\text{Na}_2\text{CO}_3$，取 25.00 mL 废液，用 0.2000 mol/L HCl 滴定。加酚酞指示剂，至褪色用去 HCl 26.14 mL（$V_1$）；再加甲基橙指示剂，继续滴定至变色用去 HCl 5.00 mL（$V_2$）。求废液中 NaOH 和 $\text{Na}_2\text{CO}_3$ 的浓度。
解：
Step 1：计算 $\text{Na}_2\text{CO}_3$ 的量
$V_2$ 对应 $\text{Na}_2\text{CO}_3$ 的第二步滴定（$\text{NaHCO}_3 \to \text{H}_2\text{CO}_3$）
$n(\text{Na}_2\text{CO}_3) = c_{\text{HCl}} \times V_2 = 0.2000 \times 5.00 = 1.000\;\text{mmol}$
$c(\text{Na}_2\text{CO}_3) = 1.000/25.00 = 0.04000\;\text{mol/L}$

Step 2：计算 NaOH 的量
$V_1$ 中包含：NaOH 全部 + $\text{Na}_2\text{CO}_3$ 的第一步（与 $V_2$ 等量）
NaOH 消耗的 HCl：$V_1 - V_2 = 26.14 - 5.00 = 21.14\;\text{mL}$
$n(\text{NaOH}) = 0.2000 \times 21.14 = 4.228\;\text{mmol}$
$c(\text{NaOH}) = 4.228/25.00 = 0.1691\;\text{mol/L}$

验证：$V_1 = 21.14 + 5.00 = 26.14$ ✓
总 HCl = $V_1 + V_2 = 26.14 + 5.00 = 31.14$ mL
$n(\text{HCl}) = 0.2000 \times 31.14 = 6.228\;\text{mmol}$
$n(\text{NaOH}) + 2n(\text{Na}_2\text{CO}_3) = 4.228 + 2\times1.000 = 6.228$ ✓
例题2：判断下列酸能否用 0.1 mol/L NaOH 直接滴定，并说明理由。
(a) $\text{HCOOH}$（$K_a = 1.8\times10^{-4}$）
(b) $\text{HCN}$（$K_a = 6.2\times10^{-10}$）
(c) $\text{H}_3\text{BO}_3$（$K_a = 5.8\times10^{-10}$）
解：

(a) $cK_a = 0.1 \times 1.8\times10^{-4} = 1.8\times10^{-5} \gg 10^{-8}$ ✓ 可以直接滴定

(b) $cK_a = 0.1 \times 6.2\times10^{-10} = 6.2\times10^{-11} < 10^{-8}$ ✗ 不能直接滴定（$K_a$ 太小，突跃不明显）

(c) $cK_a = 0.1 \times 5.8\times10^{-10} = 5.8\times10^{-11} < 10^{-8}$ ✗ 不能直接滴定。但加入甘油后 $K_a'$ 增大到约 $10^{-4}$，此时 $cK_a' = 10^{-5} \gg 10^{-8}$，可以滴定。

例题3：用 0.1000 mol/L NaOH 滴定 0.1000 mol/L $\text{H}_3\text{PO}_4$ 20.00 mL，计算第一化学计量点和第二化学计量点的 pH。
解：
$\text{H}_3\text{PO}_4$：$pK_{a1}=2.12$，$pK_{a2}=7.20$，$pK_{a3}=12.36$

第一化学计量点：溶液为 $\text{NaH}_2\text{PO}_4$（两性物质）
$c_{\text{sp}} = 0.1000 \times 20.00/40.00 = 0.0500\;\text{mol/L}$
$$\text{pH} \approx \frac{pK_{a1} + pK_{a2}}{2} = \frac{2.12 + 7.20}{2} = 4.66$$ 第二化学计量点：溶液为 $\text{Na}_2\text{HPO}_4$（两性物质）
$c_{\text{sp}} = 0.1000 \times 20.00/60.00 = 0.0333\;\text{mol/L}$
$$\text{pH} \approx \frac{pK_{a2} + pK_{a3}}{2} = \frac{7.20 + 12.36}{2} = 9.78$$

第一终点选甲基橙或溴甲酚绿（pH 4.66 附近），第二终点选酚酞或百里酚酞（pH 9.78 附近）。

例题4：0.1 mol/L NaOH 滴定 0.1 mol/L $\text{NH}_4\text{Cl}$，能否直接滴定？若能，求化学计量点 pH 和选择指示剂。
解：

$\text{NH}_4^+$ 是弱酸，$K_a = K_w/K_b = 10^{-14}/1.8\times10^{-5} = 5.56\times10^{-10}$

$cK_a = 0.1\times5.56\times10^{-10} = 5.56\times10^{-11} < 10^{-8}$

不能直接滴定！$\text{NH}_4\text{Cl}$ 的酸性太弱。

解决方法：加入过量 NaOH 后蒸馏出 $\text{NH}_3$，用硼酸吸收后再用 HCl 滴定（凯氏定氮法思路）。

例题5（PPT p59-p60 综合例题）：用 0.1000 mol/L HCl 滴定 20.00 mL 0.1100 mol/L 的某一元弱碱 B 溶液，当加入 HCl 18.00 mL 时，溶液的 pH = 8.60。求：
(1) 该一元弱碱的 $pK_b$ 值
(2) 计量点时溶液的 pH 值
(3) 该强酸滴定弱碱的滴定突跃范围
(4) 若以甲基红为指示剂，滴定到 pH = 5.0 为终点，求终点误差
解：

(1) 求 $pK_b$

加入 HCl 18.00 mL 时，处于缓冲区域：

剩余碱 $c_b = \dfrac{0.1100\times20.00 - 0.1000\times18.00}{38.00} = \dfrac{0.4\;\text{mmol}}{38.00\;\text{mL}}$

已生成共轭酸 $c_a = \dfrac{0.1000\times18.00}{38.00} = \dfrac{1.8\;\text{mmol}}{38.00\;\text{mL}}$

pH = 8.60，$\text{pOH} = 5.40$。由 $[\text{OH}^-] = K_b \cdot \dfrac{c_b}{c_a}$：

$$10^{-5.40} = K_b \times \frac{0.4}{1.8} \quad \Rightarrow \quad K_b = 1.79\times10^{-5}, \quad pK_b = 4.75$$

(2) 计量点 pH

$pK_a = 14 - 4.75 = 9.25$，$c_{\text{sp}} = \dfrac{0.1100\times20.00}{42.00} = 0.05238\;\text{mol/L}$

$$[\text{H}^+] = \sqrt{K_a \cdot c_{\text{sp}}} = \sqrt{10^{-9.25}\times0.05238} = \sqrt{2.95\times10^{-11}} = 5.43\times10^{-6}$$ $$\text{pH}_{\text{sp}} = 5.27$$

(3) 突跃范围

突跃下限（滴定99.9%，HCl近似到位）：$\text{pH} \approx pK_a - \lg(c_{\text{sp}}/K_b) \approx 4.3$

突跃上限（过量0.1%）：$c_{\text{H}^+} = \dfrac{0.1\times0.022}{42.02} \approx 5\times10^{-5}$，pH $\approx$ 4.30

完整突跃范围约 pH 4.3 ~ 6.25。

(4) 终点误差

$\Delta\text{pH} = 5.0 - 5.27 = -0.27$

$K_t' = K_b/K_w = 10^{-4.75}/10^{-14} = 10^{9.25}$

$\sqrt{K_t' \cdot c_{\text{sp}}} = \sqrt{10^{9.25}\times0.05238} = \sqrt{9.31\times10^{7}} = 9.65\times10^{3}$

$$E_t = \frac{10^{0.27} - 10^{-0.27}}{9.65\times10^3}\times100\% = \frac{1.862 - 0.537}{9650}\times100\% = +0.014\%$$

$|E_t| = 0.014\% < 0.1\%$，误差可接受。

十三、常见考试陷阱与易错点

陷阱1：弱酸滴定用错指示剂
强碱滴定弱酸，化学计量点在碱性区域（pH > 7），绝不能用甲基橙！很多同学不加思考地选甲基橙，导致严重欠滴定。
陷阱2：多元酸分步滴定条件记错
分步滴定要求 $K_{a1}/K_{a2} > 10^4$（严格为 $10^5$）。$\text{H}_2\text{C}_2\text{O}_4$ 的 $K_{a1}/K_{a2} \approx 920$，不能分步！
陷阱3：双指示剂法中 $V_1$ 和 $V_2$ 的含义混淆
$V_1$ = 酚酞褪色时消耗的 HCl（包括所有 NaOH + $\text{Na}_2\text{CO}_3$ 的一半）
$V_2$ = 再加甲基橙后继续消耗的 HCl（= $\text{Na}_2\text{CO}_3$ 的另一半）
NaOH 对应的 HCl = $V_1 - V_2$（不是 $V_1$！）
陷阱4：终点误差公式中 $c_{\text{sp}}$ 用错
$c_{\text{sp}}$ 是化学计量点时被测物在溶液中的浓度（已经被稀释！），不是初始浓度。如 20.00 mL 0.1000 mol/L HAc 被 20.00 mL NaOH 滴定，$c_{\text{sp}} = 0.1000 \times 20/40 = 0.0500\;\text{mol/L}$。
陷阱5：忘记 $\text{H}_3\text{PO}_4$ 第三步不可滴定
$K_{a3} = 4.4\times10^{-13}$ 极小，$cK_{a3} \ll 10^{-8}$，第三步无法滴定。所以 $\text{H}_3\text{PO}_4$ 最多滴两个终点，不能滴到 $\text{Na}_3\text{PO}_4$。
陷阱6：硼酸滴定必须加甘油
直接滴定 $\text{H}_3\text{BO}_3$ 不可行（$K_a$ 太小）。这是考试中常出的"陷阱选择题"。
陷阱7：NaOH 吸收 CO$_2$ 的影响与指示剂有关
酚酞 → 偏高（$\text{Na}_2\text{CO}_3$ 只滴到 $\text{NaHCO}_3$，碱量"缩水"）
甲基橙 → 无影响（$\text{Na}_2\text{CO}_3$ 完全滴到 $\text{H}_2\text{CO}_3$，碱量不变）
绝不能一概而论说"偏高"或"无影响"，必须看指示剂！
陷阱8：强酸滴弱碱的终点误差公式分子顺序
滴定弱酸时分子为 $10^{\Delta\text{pH}} - 10^{-\Delta\text{pH}}$；滴定弱碱时分子为 $10^{-\Delta\text{pH}} - 10^{\Delta\text{pH}}$。
混淆顺序会导致符号相反！
陷阱9：浓度效应的不对称性
强酸-强碱浓度增大10倍：突跃两端各扩1个 pH，总共增大 2个 pH 单位。
弱酸-强碱浓度增大10倍：突跃仅下限扩1个 pH，总共增大 1个 pH 单位。
原因：弱酸体系的突跃上限由过量 NaOH 决定（与弱酸浓度无关），下限由 $cK_a$ 决定。
考前速记清单：
  • 直接滴定判据：$cK_a \geq 10^{-8}$（弱碱同理 $cK_b \geq 10^{-8}$）
  • 分步滴定判据：$K_{a1}/K_{a2} > 10^4$（严格 $10^5$）
  • 强碱滴弱酸 → 酚酞；强酸滴弱碱 → 甲基橙/甲基红
  • 化学计量点 pH：强-强 = 7；强碱-弱酸 > 7（用 $\sqrt{K_bc_{\text{sp}}}$）；强酸-弱碱 < 7（用 $\sqrt{K_ac_{\text{sp}}}$）
  • 半当量点：滴弱酸 pH = $pK_a$；滴弱碱 pH = $pK_a$（共轭酸的）
  • 双指示剂法：NaOH+$\text{Na}_2\text{CO}_3$ 中 NaOH = $c(V_1 - V_2)$，$\text{Na}_2\text{CO}_3 = cV_2$
  • $\text{Na}_2\text{CO}_3$+$\text{NaHCO}_3$ 中 $\text{Na}_2\text{CO}_3 = cV_1$，$\text{NaHCO}_3 = c(V_2 - V_1)$
  • 硼酸 + 甘油 → 可滴定
  • $\text{H}_3\text{PO}_4$：两个终点（SP1 pH$\approx$4.7，SP2 pH$\approx$9.8）
  • $\text{H}_2\text{C}_2\text{O}_4$：一个终点（$K_{a1}/K_{a2}$ 太小）
  • NaOH 吸 $\text{CO}_2$：酚酞偏高，甲基橙无影响
  • 强酸强碱浓度 $\times 10$ → 突跃 $+2$ pH；弱酸强碱浓度 $\times 10$ → 突跃 $+1$ pH
  • CH$_3$NH$_2$（$pK_b=3.38$）被 HCl 滴定：SP pH = 5.96，半当量点 pH = 10.62
  • 终点误差：$|E_t| \leq 0.1\%$ 可接受
  • 甲醛法：$\text{NH}_4^+$ + HCHO → 释放 $\text{H}^+$，用 NaOH 滴定间接测氮
  • 离子交换法：弱酸/碱盐过阳/阴离子树脂 → 转化为可滴定的强酸/碱
  • $\text{CO}_2$ 分布：pH < 6.4 为 $\text{H}_2\text{CO}_3$；6.4~10.3 为 $\text{HCO}_3^-$；>10.3 为 $\text{CO}_3^{2-}$
  • 非水滴定：拉平效应（溶剂"拉平"强酸碱至同一水平）；区分效应（分辨酸碱强弱）
  • 滴定弱碱用酸性溶剂（冰醋酸 + $\text{HClO}_4$）；滴定弱酸用碱性溶剂
